# Clinical Coding Standards

These guidelines establish rigorous protocols for clinical coding in obstetric, perinatal, and neoplasm contexts, prioritizing accurate diagnosis sequencing, site-specific treatment logic, and the correct application of modifiers and unspecified codes.

**Key Features:**
- Pregnancy Complication Coding distinguishes pre-existing conditions from those developing during pregnancy to assign correct codes.
- Trimester Coding Guidelines assign the final character based on provider documentation of the trimester or number of weeks.
- Newborn Examination Coding requires codes for all clinically significant conditions requiring evaluation or treatment.
- Pregnancy Substance Use Coding mandates assigning subcategory O99.31 for alcohol or O99.33 for tobacco followed by secondary F10 or F17 codes.
- Perinatal Condition Coding defines the perinatal period as before birth through day 28, requiring continued use of perinatal codes for lifelong conditions.
- Pregnancy Complication Diagnosis Sequencing sequences the principal complication first in non-delivery episodes and the admission-prompting condition if delivery occurs.
- ICD-10 Maternal Fetal Care Coding assigns 7th character '0' for single gestations and uses O35/O36 only when fetal conditions modify maternal management.
- HIV Pregnancy Diagnosis Coding sequences Chapter 15 codes (O98.7-) before HIV-related illness codes (B20) as the principal diagnosis.
- Gestational Diabetes Code Subcategory O24.4 excludes other O24 category codes when coding for gestational diabetes mellitus.
- Fetal Surgery Coding Guidelines uses category O35 for fetal conditions and prohibits Chapter 16 perinatal codes on the mother's record for fetal issues.
- High-Risk Pregnancy Coding Guidelines restricts category O09 codes to the prenatal period and directs labor/delivery complications to Chapter 15.
- Principal Diagnosis Assignment for Malignancy designates the primary malignancy as the principal diagnosis when treatment is directed at the primary site.
- Secondary Neoplasm Principal Diagnosis Logic designates the secondary neoplasm as the principal diagnosis when treatment is directed solely toward the secondary site.
- Malignant Neoplasm Transplant Coding Protocol codes malignant neoplasm of a transplanted organ as a transplant complication using category T86.- followed by C80.2.

**Notable Details:**
- The perinatal period is defined as before birth through the 28th day following birth.
- If a condition originates in the perinatal period and continues for life, the perinatal code must be used regardless of age.
- Codes from category O09 are intended for use only during the prenatal period.
- For fetal surgery, assign a diagnosis code from category O35 to identify the fetal condition.
- Do not use Chapter 16 perinatal codes on the mother's record to identify fetal conditions.

### Neoplasm Coding Guidelines

These guidelines establish a comprehensive framework for neoplasm coding that prioritizes the site of treatment and the status of the malignancy to determine principal diagnosis assignment. They provide specific protocols for managing complex scenarios such as metastatic disease, unspecified primaries, and complications arising from neoplasm treatment.

**Key Features:**
- Principal Diagnosis Assignment for Malignancy: Designates the primary malignancy as the principal diagnosis when treatment is directed at the primary site.
- Primary Malignancy Coding Guidelines: Requires use of primary malignancy codes until treatment is completed or excised with no further therapy.
- Neoplasm and Anemia Encounter Coding Guidelines: Sequences malignancy code first followed by anemia code when admission is for anemia management associated with malignancy.
- Tumor Coding Clarification Protocol: Mandates provider queries when documentation is unclear regarding the status of multiple tumors in the same organ.
- C80.0 Unspecified Neoplasm Use Case: Restricts C80.0 to advanced metastatic disease with no known primary or secondary sites specified.
- C80 Unspecified Neoplasm Coding Guidelines: Assigns C80.1 only when the primary site cannot be determined and rarely in the inpatient setting.
- Extranodal Lymphoma Coding Rules: Assigns codes from categories C81-C85 with a final character for extranodal sites without coding the secondary solid organ.
- Secondary Neoplasm Principal Diagnosis Logic: Designates the secondary neoplasm as the principal diagnosis when treatment is directed solely toward the secondary site.
- Neoplasm Coding and Diagnosis Guidelines: Makes the complication the principal diagnosis when the encounter is for treatment of a complication from neoplasm surgery.
- Metastatic Site Principal Diagnosis Coding: Designates the metastatic site as the principal diagnosis when treatment is directed toward it only.
- Malignant Neoplasm Transplant Coding Protocol: Codes malignant neoplasm of a transplanted organ as a transplant complication using category T86.- followed by C80.2.

**Notable Details:**
- Code C80.0 is for use only when the patient has advanced metastatic disease with no known primary or secondary sites specified.
- Code C80.1 equates to Cancer, unspecified and should rarely be used in the inpatient setting.
- Multiple malignant tumors in the same organ may represent different primaries or metastatic disease depending on the site.
- Do not assign a code for the secondary neoplasm of the affected solid organ in cases of extranodal metastasis.
- Ensure the encounter is explicitly for a primary malignancy with metastasis to apply the rule for designating the metastatic site as principal.
- Documentation must identify the neoplasm as occurring in a transplanted organ to apply the transplant complication protocol.
- Apply secondary neoplasm principal diagnosis logic even if the primary malignancy remains present in the patient.

### ICD-10 Coding Guidelines

These guidelines establish standardized protocols for selecting and sequencing diagnosis codes in ICD-10-CM, specifically addressing outpatient first-listed diagnoses, Z code applications for circumstances like screenings and immunizations, and specific rules for transplant and HIV-related encounters.

**Key Features:**
- Outpatient First-Listed Diagnosis Coding: Uses the term 'first-listed diagnosis' instead of 'principal diagnosis' for outpatient encounters.
- Kidney Transplant Complication Coding: Assigns code T86.1 for documented complications like rejection or failure, excluding chronic kidney disease alone.
- HIV Medication Coding: Requires code B20 for documented HIV disease and Z79.899 for long-term antiretroviral medication use.
- Z Code Usage Guidelines: Permits Z codes as first-listed or principal diagnoses in any setting when identifying reasons unrelated to disease or injury.
- CKD Post-Transplant Coding: Mandates coding the specific CKD stage (N18) and transplant status (Z94.0) separately from transplant complications.
- Z28 Underimmunization Code Mapping: Requires mapping Z28 to Z29, Z40, or Z41 based on the specific nature of the encounter.
- Z13 Screening Encounter Classification: Defines screening as testing for disease in well individuals, allowing Z13 as a first-listed code for screening visits.
- Z05 Secondary Coding for Z38: Requires Z05 codes to be assigned as secondary codes following the primary Z38 code for liveborn infants.
- Z Code Procedure Code Requirements: Stipulates that Z codes are not procedure codes and must be accompanied by a corresponding procedure code if a procedure was performed.
- Z Code Guidelines for Specified Encounters: Directs the use of Z codes for encounters focused on health checkups, screenings, or other specified circumstances.
- Encounter Codes for Abnormal Findings: Assigns the code for general medical examination with abnormal findings as the first-listed diagnosis when new conditions or severity changes are identified.

**Notable Details:**
- The term 'principal diagnosis' is replaced by 'first-listed diagnosis' in the outpatient setting.

These guidelines establish strict sequencing and assignment rules for complex clinical scenarios involving malignancies, obstetrical conditions, and congenital anomalies. They prioritize the primary focus of treatment while defining specific timeframes and exclusion criteria for modifiers like POA and tPA status.

**Key Features:**
- Pathological Fracture Coding Sequencing: Prioritizes fracture or neoplasm code based on the primary treatment focus.
- Congenital Anomaly Exemption Logic: Assigns 'Y' modifier to congenital conditions except those in categories Q00-Q99.
- Intrauterine and Delivery Condition Adjudication: Requires 'Y' modifier for conditions present at birth or developed in utero.
- POA Assignment Logic for Obstetrical Conditions: Determines Present on Admission status based on condition presence at admission, independent of delivery timing.
- Postpartum Complication Definition: Classifies complications occurring within the six-week period following delivery.
- Peripartum and Postpartum Period Definitions: Defines postpartum as six weeks and peripartum as the last month of pregnancy to five months postpartum.
- tPA Administration Exclusion Criteria: Excludes claims with prior tPA administration in a different facility within 24 hours.
- Malignancy Principal Diagnosis and Prophylactic Organ Removal Rules: Prevents symptoms or signs from replacing malignancy as principal diagnosis.
- Chapter 16 Coding Guidelines: Requires Chapter 16 codes to be sequenced first for perinatal conditions.
- O9A Sequencing for Toxic Effects: Mandates subcategory O9A.2 codes to be sequenced first for toxic effects during pregnancy.
- Anemia Sequencing for Chemotherapy Adverse Effects: Sequences anemia code first when admission is solely for managing chemotherapy-induced anemia.
