# ICD-10-CM Coding Guidelines

These guidelines establish a standardized framework for healthcare data quality by defining strict rules for code assignment, diagnosis sequencing, and the handling of uncertain clinical conditions. They ensure consistency across care settings by prioritizing definitive documentation while providing specific logic for signs, symptoms, and resource utilization.

**Key Features:**
- Non-Pressure Ulcer Coding Guidelines: Assign unspecified stage codes when documentation is missing and unstageable codes only when eschar or grafts prevent clinical determination.
- Uncertain Diagnosis Coding Guidelines: Code diagnoses as established for inpatient admissions but exclude 'probable' or 'suspected' diagnoses for outpatient encounters.
- Principal Diagnosis Sequencing Rules: Sequence any equally qualifying diagnosis first when no specific direction exists and prioritize the condition occasioning admission even if treatment was not performed.
- Underdosing Code Assignment Guidelines: Assign T36-T50 codes with characters '6' for underdosing without requiring documentation of condition change and never as principal diagnoses.
- Code Assignment and Diagnosis Exception Logic: Base coding on provider diagnostic statements except for specific metrics like BMI or pressure ulcer stages where non-provider documentation is accepted.
- Sign and Symptom Coding Guidelines: Report signs or symptoms as diagnoses when a definitive diagnosis is not established and use unspecified codes when clinical information is insufficient.
- Diagnostic Statement Coding Guidelines: Code diagnoses included in the final diagnostic statement while excluding resolved conditions or status-post procedures from previous admissions.
- Diagnosis Sequencing for Contrasting Diagnoses: Code 'either/or' diagnoses as confirmed and sequence them based on admission circumstances when no principal diagnosis can be determined.
- Deep Tissue Pressure Injury Coding: Assign specific codes for pressure-induced deep tissue damage only when the diagnosis is explicitly documented as such.
- Syndrome Coding Index Guidance: Follow Alphabetic Index guidance for syndromes and assign codes for documented manifestations when the index lacks specific entries.
- Impending and Threatened Condition Coding: Code 'impending' or 'threatened' conditions as confirmed if they occurred and reference index subterms if they did not.
- Combination Code Assignment Logic: Use a single combination code when it fully identifies all diagnostic conditions or when directed by the Alphabetic Index.
- Clinical Coding for Resource Utilization: Assign codes only for conditions that require treatment, prolong length of stay, or necessitate resource utilization.
- G89 Pain Classification Guidelines: Exclude psychological-only pain (F45.41) and combine F45.42 with G89 when a psychological component exists.
- G89 Pain Code Sequencing Logic: Sequence G89 first only for pain control encounters; otherwise, sequence site-specific codes first.

**Notable Details:**
- Unstageable pressure ulcer codes (L89.--0) are restricted to cases where stage cannot be determined due to eschar or grafts.
- Outpatient encounters strictly prohibit coding diagnoses documented as 'probable', 'suspected', or 'working diagnosis'.

### Sepsis and Burn Coding Guidelines

These guidelines establish strict sequencing rules for coding sepsis, burns, and specific infections like MRSA and COVID-19, prioritizing the underlying cause or highest severity. They mandate the use of specific subcategories for severe conditions while prohibiting their use as principal diagnoses in certain contexts. - MRSA Infection and Carrier Coding: Infection codes are assigned with B95.62 as cause when no combination exists, while Z22.322 is used only for documented colonization alongside infection.
- Septic Shock Coding Sequencing: Systemic infection is sequenced first, followed by R65.21 or T81.12, with septic shock codes never serving as principal diagnoses.
- Sequela and Late Effect Coding Guidelines: Sequela codes describe residual effects after the acute phase and are sequenced after the condition code.
- Severe Sepsis Coding Guidelines: Underlying infection is sequenced first, requiring explicit documentation of organ dysfunction or severe sepsis before assigning R65.2.
- Burn Sequela Coding Guidelines: Late effects of burns require a burn code with the 7th character 'S' when the condition is explicitly residual.
- Sequela Code Sequencing Logic: The code describing the sequelae of a complication must be assigned before the sequela code itself.
- Sepsis and Localized Infection Coding Guidelines: Underlying systemic infection is sequenced first for sepsis admissions, with localized infection as secondary.
- Code Sequencing Logic for Encounter Notes: 'Code also' notes require two codes but do not dictate sequencing order, which depends on encounter circumstances.
- MRSA Combination Code Assignment: Combination codes including the organism are assigned instead of separate B95.62 codes for the infection.
- Infected Burn Coding Guidelines: An additional code for infection is required only when the burn site is explicitly documented as infected.
- Puerperal Sepsis Coding Guidelines: Code O85 is used with secondary organism codes, avoiding A40/A41 categories and excluding O85 for post-procedural sepsis.

### Chronic Disease Coding Guidelines

These guidelines establish a comprehensive framework for coding chronic diseases, specifically focusing on hypertension and its complex interactions with organ systems like the heart, kidneys, and brain. They provide strict rules for sequencing, distinguishing between primary conditions, complications, and device-related failures to ensure accurate clinical representation.

**Key Features:**
- Hypertensive Disease Coding: Assigns I10-I15 codes based on whether hypertension is controlled or uncontrolled.
- Diabetes Mellitus Coding: Requires combination codes that include type, affected body system, and complications.
- Insulin Pump Failure Coding: Sequences T85.6 for pump malfunction followed by T38.3X6- for drug underdosing or overdose.
- Resistant Hypertension Coding: Adds I10.0 or I1A.0 as additional codes when treatment resistance is documented.
- I13 Hypertensive Heart and Kidney Disease Coding: Uses I12 for kidney involvement and I13 for combined heart and kidney involvement.
- CKD Coding Sequencing: Prioritizes sequencing based on Tabular List conventions regarding contributing conditions like diabetes and hypertension.
- Hypertensive Retinopathy Sequencing: Requires pairing subcategory H35.0 with I10-I15 codes based on the reason for encounter.
- Hypertensive Crisis Coding Sequencing: Assigns I16 for urgency/emergency alongside specific hypertension codes based on encounter reason.
- Hypertensive Cerebrovascular Coding Protocol: Mandates I60-I69 codes before hypertension codes for cerebrovascular disease.
- Hypertension Comorbidity Coding Rules: Presumes a causal relationship between hypertension and heart/kidney conditions if linked by 'with'.
- I27 Pulmonary Hypertension Coding: Classifies secondary pulmonary hypertension in I27 alongside associated conditions.
- Postpancreatectomy Diabetes Coding: Assigns E89.1 as the principal diagnosis with Z90.41 as an additional code.
- Cerebrovascular Sequelae Coding: Specifies dominant or nondominant side for hemiplegia/paresis using I69 codes.

**Notable Details:**
- Diabetes codes must be sequenced based on the reason for the particular encounter.
- As many codes as necessary should be assigned to describe all complications of diabetes.
- T85.6 is the principal code for both underdose and overdose due to pump malfunction.
- Code I10.0 is used as an additional code for apparent, treatment-resistant, or true resistant hypertension.
- Use N18 as a secondary code with I12 or I13 to identify the stage of chronic kidney disease.
