# Glaucoma Coding Assignment
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Glaucoma Coding Assignment

**Key Behaviors:**
- Assign as many codes from category H40 as needed to identify the type of glaucoma, the affected eye, and the glaucoma stage.
- Report only the code for the type of glaucoma, bilateral, with the seventh character for the stage when both eyes are documented as the same type and stage and a bilateral code exists.
- Report only one code for the type of glaucoma with the appropriate seventh character for the stage when both eyes are documented as the same type and stage but no bilateral code exists.
- Assign the appropriate code for each eye rather than the code for bilateral glaucoma when each eye is documented as having a different type or stage and the classification distinguishes laterality.
- Assign one code for each type of glaucoma with the appropriate seventh character for the stage when each eye is documented as having a different type and the classification does not distinguish laterality.

**Requirements / Properties:**
- Documentation must identify the type of glaucoma, the affected eye, and the glaucoma stage.
- Documentation must specify if the condition is bilateral and whether the type and stage are the same or different for each eye.
- The classification must distinguish laterality to determine if separate codes are required for each eye.
- A code for bilateral glaucoma must exist in the classification to be used when both eyes are identical.
- The seventh character must correspond to the specific glaucoma stage documented for each eye.
