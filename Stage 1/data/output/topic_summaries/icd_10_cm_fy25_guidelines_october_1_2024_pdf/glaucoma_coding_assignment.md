# Glaucoma Coding Assignment
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Glaucoma Coding Assignment

**Key Behaviors:**
- Assign as many codes from category H40 as needed to identify the type of glaucoma, the affected eye, and the glaucoma stage.
- Report only the code for bilateral glaucoma with the seventh character for the stage when both eyes are the same type and stage and a bilateral code exists.
- Report only one code for the type of glaucoma with the appropriate seventh character when both eyes are the same type and stage but no bilateral code exists.
- Assign the appropriate code for each eye rather than the code for bilateral glaucoma when each eye has a different type or stage and laterality is distinguished.
- Assign one code for each type of glaucoma with the appropriate seventh character when each eye has a different type and laterality is not distinguished.

**Mandatory Coding Criteria:**
- Documentation must identify the type of glaucoma, the affected eye, and the glaucoma stage.
- If bilateral glaucoma is present, the specific relationship between eye types and stages determines whether a single bilateral code or multiple eye-specific codes are assigned.
- When laterality is not distinguished in the classification (e.g., H40.10, H40.20), codes must be assigned for each eye individually if types or stages differ.
- The seventh character must be used to indicate the glaucoma stage for each assigned code.
- A code for bilateral glaucoma is only reported if the classification explicitly provides one for the documented type and stage.
