# Z79 Code Assignment Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Z79 Code Assignment Guidelines

**Key Behaviors:**
- A code from category Z05 may be assigned as a principal or first-listed code for readmissions or encounters when the code from category Z38 no longer applies.
- Codes from category Z05 are restricted to use only for healthy newborns and infants for which no condition after study is found to be present.

**Requirements / Properties:**
- The patient must be a healthy newborn or infant.
- No condition after study must be found to be present.
- The code from category Z38 must no longer apply for the encounter.
