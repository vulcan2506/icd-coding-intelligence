# ICD-10-CM Myocardial Infarction Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Myocardial Infarction Coding Guidelines

**Key Behaviors:**
- Use category I22 for subsequent type 1 or unspecified AMI within 4 weeks of an initial AMI, sequenced with I21 based on encounter circumstances.
- Assign only code I21.A1 for subsequent type 2 AMI and only code I21.A9 for subsequent type 4 or 5 AMI, avoiding I22 codes.
- Assign appropriate I21 codes for subsequent myocardial infarctions of different types occurring within 4 weeks of each other, rather than using I22.
- Assign codes I21.0-I21.4 exclusively for type 1 AMIs and I21.A9 for types 3, 4a, 4b, 4c, and 5.
- Follow 'Code also' and 'Code first' notes for complications and postprocedural myocardial infarctions during or following cardiac surgery.

**Mandatory Coding Criteria:**
- Subsequent AMI must be type 1 or unspecified to use category I22.
- Subsequent type 2 AMI must be coded as I21.A1 with underlying cause coded first if applicable.
- Subsequent type 4 or 5 AMI must be coded as I21.A9.
- Initial and subsequent AMIs must both be type 1 or unspecified to justify the use of category I22.
- Codes I21.0-I21.4 should only be assigned for type 1 AMIs.
- Code I21.A1 must be sequenced with the underlying cause coded first if applicable.
- Code I24.8 must not be assigned for demand ischemia.
