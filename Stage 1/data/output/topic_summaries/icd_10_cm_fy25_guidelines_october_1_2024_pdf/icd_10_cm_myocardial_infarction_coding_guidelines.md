# ICD-10-CM Myocardial Infarction Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Myocardial Infarction Coding Guidelines

**Key Behaviors:**
- Use category I22 for subsequent type 1 or unspecified AMI within 4 weeks of an initial AMI, sequenced with I21 based on encounter circumstances.
- Assign code I21.A1 for subsequent type 2 AMI and I21.A9 for subsequent type 4 or 5 AMI, avoiding I22 codes entirely.
- If a subsequent AMI of one type occurs within 4 weeks of a different type, assign appropriate I21 codes for each type and do not assign I22.
- Assign codes I21.0-I21.4 exclusively for type 1 AMIs and I21.A9 for acute myocardial infarctions type 3, 4a, 4b, 4c, and 5.
- Follow 'Code also' and 'Code first' notes for complications and postprocedural myocardial infarctions during or following cardiac surgery.

**Requirements / Properties:**
- Documentation must confirm the subsequent AMI occurred within the 4-week time frame of the initial AMI to utilize category I22.
- The initial and subsequent myocardial infarctions must both be type 1 or unspecified to justify the use of category I22 codes.
- For type 2 AMI, the underlying cause must be coded first if applicable, and the code I21.A1 must be used regardless of NSTEMI or STEMI description.
- Codes I21.0-I21.4 must only be assigned when the clinical documentation confirms the event is a type 1 AMI.
- Coding of postprocedural myocardial infarctions must adhere to specific 'Code first' instructions related to complications of care.
- The underlying cause must be coded first when assigning code I21.A1 for Type 2 AMI, if applicable.
- Code I24.8 must not be assigned for demand ischemia even if the documentation mentions acute ischemic heart disease.

**Deprecated in this version:**
- Assigning code I22 for subsequent myocardial infarctions that are type 2, type 4, or type 5.
- Using code I22 when the initial and subsequent AMIs are of different types occurring within the 4-week window.
- Assigning code I24.8 for demand ischemia when a type 2 AMI is documented.
- Using codes I21.0-I21.4 for any AMI type other than type 1.
- Failing to sequence I22 and I21 codes based on the specific circumstances of the encounter.

**New in this version:**
- Explicit distinction between type 1, type 2, and type 4/5 AMIs requiring specific I21 subcategory codes (I21.A1, I21.A9) instead of generic I22.
- Mandatory use of I21.A1 for type 2 AMI even if described as NSTEMI or STEMI, replacing previous broader categorizations.
- Requirement to code the underlying cause first for type 2 AMI to ensure accurate etiology capture.
- Specific instruction to avoid I22 codes entirely when subsequent AMIs involve types other than 1 or unspecified.
