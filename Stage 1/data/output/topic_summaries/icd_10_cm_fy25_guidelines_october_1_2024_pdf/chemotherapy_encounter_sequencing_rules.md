# Chemotherapy Encounter Sequencing Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Chemotherapy Encounter Sequencing

**Key Behaviors:**
- Sequence the diagnosis chiefly responsible for outpatient therapeutic services first.
- Code other diagnoses (e.g., chronic conditions) as additional diagnoses.
- List the appropriate Z code for chemotherapy or radiation therapy first if it is the primary reason for admission/encounter.
- List the diagnosis or problem for which the service is being performed second if chemotherapy/radiation is the primary reason.

**Mandatory Coding Criteria:**
- The diagnosis must be shown in the medical record to be chiefly responsible for the outpatient services provided.
- The primary reason for the admission/encounter must be chemotherapy or radiation therapy to trigger the Z code sequencing exception.
