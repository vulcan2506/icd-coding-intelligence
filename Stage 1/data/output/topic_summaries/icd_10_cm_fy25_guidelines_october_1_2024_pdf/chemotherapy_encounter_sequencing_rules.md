# Chemotherapy Encounter Sequencing Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Chemotherapy Encounter Sequencing Rules

**Key Behaviors:**
- Sequence the diagnosis chiefly responsible for outpatient services first for therapeutic-only encounters.
- Sequence other diagnoses (e.g., chronic conditions) as additional diagnoses.
- List the appropriate Z code for chemotherapy or radiation therapy first if it is the primary reason for admission/encounter.
- List the diagnosis or problem for which the service is being performed second when chemotherapy/radiation is the primary reason.

**Requirements / Properties:**
- The diagnosis must be shown in the medical record as chiefly responsible for the outpatient services provided.
- The primary reason for the admission or encounter must be explicitly identified as chemotherapy or radiation therapy to trigger the Z code exception.
