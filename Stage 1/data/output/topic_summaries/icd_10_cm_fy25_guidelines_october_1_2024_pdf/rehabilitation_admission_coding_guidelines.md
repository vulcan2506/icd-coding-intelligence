# Rehabilitation Admission Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Rehabilitation Admission Coding Guidelines

**Key Behaviors:**
- Sequence first the code for the condition for which the service is being performed when the purpose is rehabilitation.
- Report the appropriate aftercare code as the first-listed diagnosis if the condition requiring rehabilitation is no longer present.
- Assign the injury code with the appropriate seventh character for subsequent encounter as the first-listed diagnosis for rehabilitation services following active treatment of an injury.

**Requirements / Properties:**
- Clinical documentation must clearly identify the condition for which rehabilitation services are being performed.
- Documentation must confirm whether the condition requiring rehabilitation is currently present or resolved.
- For injury-related rehabilitation, the seventh character must indicate a subsequent encounter with routine healing.

**New in this version:**
- Explicit instruction to use specific aftercare codes (e.g., Z47.1) as the principal diagnosis when the underlying condition is resolved.
- Mandatory use of injury codes with seventh character 'D' (subsequent encounter) for post-injury rehabilitation admissions.
- Reference to Section I.C.21.c.7 regarding factors influencing health states and contact with health services for aftercare.
- Reference to Section I.C.19.a for additional information on the use of seventh characters for injury codes.
