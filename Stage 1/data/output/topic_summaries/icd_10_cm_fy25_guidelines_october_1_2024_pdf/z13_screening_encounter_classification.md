# Z13 Screening Encounter Classification
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Z13 Screening Encounter Classification

**Key Behaviors:**
- Screening is defined as testing for disease in seemingly well individuals for early detection, distinct from diagnostic examinations for suspected conditions.
- A screening code may be a first-listed code if the reason for the visit is specifically the screening exam.
- A screening code may be used as an additional code if the screening is performed during an office visit for other health problems.
- A screening code is not necessary if the screening is inherent to a routine examination (e.g., pap smear during a routine pelvic exam).
- If a condition is discovered during screening, the code for the condition is assigned as an additional diagnosis.

**Mandatory Coding Criteria:**
- The Z code indicates that a screening exam is planned.
- A procedure code is required to confirm that the screening was performed.
- For diagnostic examinations (testing to rule out or confirm a suspected diagnosis), the sign or symptom must be used to explain the reason for the test.
