# Extranodal Lymphoma Coding Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Extranodal Lymphoma Coding Rules

**Key Behaviors:**
- Assign codes from categories C81-C85 with a final character identifying 'extranodal and solid organ sites' when malignant neoplasm metastasizes beyond lymph nodes.
- Do not assign a code for the secondary neoplasm of the affected solid organ in cases of extranodal metastasis.
- Use specific codes (e.g., C83.39) to identify the specific lymphoma type and extranodal location.

**Requirements / Properties:**
- Documentation must confirm the neoplasm is malignant and of lymphoid tissue.
- Documentation must confirm metastasis has occurred beyond the lymph nodes to extranodal or solid organ sites.
- The specific type of lymphoma must be identified to select the correct category code.

**Deprecated in this version:**
- Assigning a code for the secondary neoplasm of the affected solid organ when the primary diagnosis is extranodal lymphoma metastasis.

**New in this version:**
- Requirement to use the final character identifying 'extranodal and solid organ sites' for all malignant neoplasms of lymphoid tissue metastasizing beyond lymph nodes.
