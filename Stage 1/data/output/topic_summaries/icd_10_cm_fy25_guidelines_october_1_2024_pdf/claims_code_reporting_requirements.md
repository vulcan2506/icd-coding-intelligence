# Claims Code Reporting Requirements
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Claims Code Reporting Requirements

**Key Behaviors:**
- Only specific codes are permissible for reporting purposes
- Categories and subcategories are not permissible for reporting
- Any applicable 7th character extension is required for reporting

**Deprecated in this version:**
- Use of categories for reporting purposes
- Use of subcategories for reporting purposes
