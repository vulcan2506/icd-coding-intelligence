# Claims Code Reporting Requirements
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Claims Code Reporting Standards

**Key Behaviors:**
- Only specific codes are permissible for reporting purposes
- Categories and subcategories are not permissible for reporting
- Any applicable 7th character must be included in the code
- Reporting must adhere to strict code specificity requirements

**Mandatory Coding Criteria:**
- Reporting must use specific codes rather than categories or subcategories
- Applicable 7th characters must be appended to codes where required
