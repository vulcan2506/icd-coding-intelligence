# I13 Hypertensive Heart and Kidney Disease Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** I13 Hypertensive Heart and Kidney Disease Coding

**Key Behaviors:**
- Assign codes from category I13 when hypertension is present with both heart and kidney involvement.
- Use the appropriate code from category N18 as a secondary code with a code from category I13 to identify the stage of chronic kidney disease.
- If heart failure is present, assign an additional code from category I50 to identify the type of heart failure.
- Do not use individual codes for hypertension, heart disease, and chronic kidney disease if a code from I13 is available.
- Code acute renal failure in addition to chronic kidney disease and sequence according to the circumstances of the admission.

**Requirements / Properties:**
- Both hypertension and a condition classifiable to category N18 (Chronic kidney disease) must be present to assign I12.
- CKD should not be coded as hypertensive if the provider indicates the CKD is not related to the hypertension.
- A code from category I13 must be used instead of individual codes for hypertension, heart disease, and chronic kidney disease when all three are present.
- The Includes note at I13 specifies that conditions included at I11 and I12 are included together in I13.
- Acute renal failure must be coded in addition to chronic kidney disease when both are present.

**Deprecated in this version:**
- Using individual codes for hypertension, heart disease, and chronic kidney disease when a combination code from I13 is available.
- Coding CKD as hypertensive if the provider indicates the CKD is not related to the hypertension.
- Failing to assign an additional code from category I50 when heart failure is present with hypertensive heart and kidney disease.
- Not sequencing acute renal failure according to the circumstances of the admission when both acute and chronic renal failure are present.
- Omitting the stage of chronic kidney disease when assigning a code from category I13.

**New in this version:**
- The requirement to use the appropriate code from category N18 as a secondary code with a code from category I13 to identify the stage of chronic kidney disease.
- The specification that the Includes note at I13 specifies that the conditions included at I11 and I12 are included together in I13.
- The mandate to code acute renal failure in addition to chronic kidney disease when both are present.
- The instruction to sequence acute renal failure according to the circumstances of the admission/encounter.
- The clarification that CKD should not be coded as hypertensive if the provider indicates the CKD is not related to the hypertension.
