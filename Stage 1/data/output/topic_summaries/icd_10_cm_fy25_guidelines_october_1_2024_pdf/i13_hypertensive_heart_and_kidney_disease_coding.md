# I13 Hypertensive Heart and Kidney Disease Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** I13 Hypertensive Heart and Kidney Disease Coding

**Key Behaviors:**
- Assign codes from category I12 when both hypertension and chronic kidney disease (N18) are present.
- Assign codes from combination category I13 when there is hypertension with both heart and kidney involvement.
- Use the appropriate code from category N18 as a secondary code with I12 or I13 to identify the stage of chronic kidney disease.
- If heart failure is present with I13, assign an additional code from category I50 to identify the type of heart failure.
- Code acute renal failure in addition to hypertensive chronic kidney disease if present, sequencing according to admission circumstances.

**Mandatory Coding Criteria:**
- Both hypertension and a condition classifiable to category N18 must be present to assign I12.
- The provider must indicate that CKD is related to hypertension; if not related, do not code as hypertensive.
- For I13, there must be hypertension with both heart and kidney involvement.
- The Includes note at I13 specifies that conditions included at I11 and I12 are included together in I13.
- If a patient has hypertension, heart disease, and chronic kidney disease, use a code from I13 instead of individual codes for each condition.
