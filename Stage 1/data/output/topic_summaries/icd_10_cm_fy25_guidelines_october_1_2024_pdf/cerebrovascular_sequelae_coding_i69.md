# Cerebrovascular Sequelae Coding (I69)
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Cerebrovascular Sequelae Coding (I69)

**Key Behaviors:**
- Codes from category I69 indicate conditions classifiable to categories I60-I67 as causes of sequela (neurologic deficits) classified elsewhere.
- Codes specifying hemiplegia, hemiparesis, and monoplegia identify whether the dominant or nondominant side is affected.
- If the affected side is documented but not specified as dominant or nondominant, default to dominant for ambidextrous patients, non-dominant for left side, and dominant for right side.
- Codes from category I69 may be assigned with codes from I60-I67 if the patient has current cerebrovascular disease and deficits from an old cerebrovascular disease.
- Codes from category I69 should not be assigned if the patient does not have neurologic deficits.

**Requirements / Properties:**
- The patient must have neurologic deficits to qualify for a code from category I69.
- The neurologic deficits must be caused by conditions classifiable to categories I60-I67.
- Documentation must specify the affected side (dominant or nondominant) if possible to ensure accurate coding.
- Codes from I69 can be used alongside I60-I67 only when there is a mix of current and old cerebrovascular disease.
- Personal history codes (Z86.73) should not be combined with I69 codes for the same condition if active deficits exist.
- Codes from category I69 must not be assigned if the patient does not have neurologic deficits.

**Deprecated in this version:**
- Assigning I69 codes without documented neurologic deficits.
- Using unspecified laterality codes when side information is available.
- Combining I69 codes with Z86.73 for active deficits instead of using I69 with I60-I67.
- Failing to apply default rules for dominant/nondominant side when documentation is incomplete.
- Coding I69 for deficits that are an integral part of the initial disease process rather than late effects.

**New in this version:**
- Explicit default rules for assigning dominant or nondominant side when documentation is incomplete.
- Clarification on combining I69 with I60-I67 for patients with both current and old cerebrovascular disease.
- Mandatory exclusion of I69 codes if no neurologic deficits are present.
- Updated guidance on using personal history codes (Z86.73) in conjunction with I69.
- Enhanced emphasis on documenting the specific type of neurologic deficit for accurate I69 coding.
