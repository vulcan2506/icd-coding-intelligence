# External Cause Code Reporting Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** External Cause Code Reporting Guidelines

**Key Behaviors:**
- External cause codes should never be sequenced as the first-listed or principal diagnosis.
- Codes capture the cause, intent, place, activity, and status of the person at the time of the event.
- Reporting is not nationally mandatory unless a state mandate or specific payer requirement exists.
- Providers are encouraged to voluntarily report codes to support injury research and prevention strategies.

**Requirements / Properties:**
- State-based external cause code reporting mandates must be met for compliance in those jurisdictions.
- Specific payer requirements must be satisfied if the payer mandates these codes.
