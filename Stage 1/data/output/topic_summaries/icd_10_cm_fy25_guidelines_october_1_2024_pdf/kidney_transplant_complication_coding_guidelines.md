# Kidney Transplant Complication Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Kidney Transplant Complication Coding

**Key Behaviors:**
- Code T86.1 for documented complications of a kidney transplant such as transplant failure or rejection.
- Do not assign T86.1- for post-transplant patients with chronic kidney disease (CKD) unless a specific transplant complication is documented.
- Query the provider if documentation is unclear regarding whether the patient has a transplant complication.
- Assign a code from subcategory T86.1 for conditions affecting transplanted kidney function other than CKD, paired with a secondary complication code.
- Use categories Z89-Z90 and Z93-Z99 only if there are no complications or malfunctions of the organ or equipment.

**Requirements / Properties:**
- Documentation must explicitly state a transplant complication (e.g., failure, rejection) to justify the use of code T86.1.
- Chronic kidney disease (CKD) alone following a transplant does not require code T86.1 without a documented complication.
- A secondary code identifying the specific complication must be assigned alongside the T86.1 code for non-CKD conditions.
- Provider clarification is mandatory when the documentation is ambiguous regarding the presence of a transplant complication.
- Categories Z89-Z90 and Z93-Z99 are conditional and require the absence of organ or equipment complications.

**Deprecated in this version:**
- Assigning code T86.1- solely for post-transplant chronic kidney disease without documented failure or rejection.
- Coding transplant complications without verifying that they are not integral to the disease process or expected outcomes.
- Using Z-dependence categories when complications or malfunctions of the transplanted organ are present.
- Failing to query the provider when the distinction between CKD and transplant complication is unclear in the record.
- Omitting the secondary complication code when assigning a T86.1 code for specific functional conditions.

**New in this version:**
- Explicit instruction to query providers for unclear documentation regarding transplant complications.
- Clarification that CKD following transplant requires separate coding unless linked to a documented complication like rejection.
- Specific reference to subcategory T86.1 for non-CKD conditions affecting transplanted kidney function.
- Guidance on sequencing sepsis due to infection in transplanted organs per section I.C.1.d.4.
- Updated note on the restricted use of Z89-Z90 and Z93-Z99 categories in the presence of organ complications.
- Sequencing of sepsis due to infection in a transplanted organ is governed by section I.C.1.d.4.
- Coding of organ transplant complications generally follows the information provided in section I.C.19.
