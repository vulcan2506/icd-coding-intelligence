# Kidney Transplant Complication Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Kidney Transplant Complication Coding

**Key Behaviors:**
- Assign code T86.1 for documented complications of a kidney transplant such as transplant failure or rejection.
- Do not assign T86.1- for post-transplant patients with chronic kidney disease (CKD) unless a specific transplant complication like failure or rejection is documented.
- Query the provider if documentation is unclear regarding whether the patient has a transplant complication.
- Assign codes from subcategory T86.1 for conditions affecting transplanted kidney function other than CKD, paired with a secondary code identifying the complication.
- Use categories Z89-Z90 and Z93-Z99 only if there are no complications or malfunctions of the organ, amputation site, or dependent equipment.
- Assign a secondary code to identify the specific complication when using subcategory T86.1 for conditions affecting transplanted kidney function other than CKD.
- Refer to section I.C.1.d.4 for specific sequencing instructions regarding sepsis due to infection in a transplanted organ.

**Mandatory Coding Criteria:**
- Documentation must explicitly state a transplant complication (e.g., failure, rejection) to assign T86.1.
- A secondary code must identify the specific complication when assigning codes from subcategory T86.1.
- Provider query is required if the documentation is unclear about the presence of a transplant complication.
- Categories Z89-Z90 and Z93-Z99 require confirmation that no complications or malfunctions exist.
- Refer to section I.C.14 for coding CKD following a kidney transplant without complications.
- Documentation must explicitly state a transplant complication (e.g., failure, rejection) to assign code T86.1.
- Categories Z89-Z90 and Z93-Z99 require confirmation that there are no complications or malfunctions of the organ, amputation site, or dependent equipment.
