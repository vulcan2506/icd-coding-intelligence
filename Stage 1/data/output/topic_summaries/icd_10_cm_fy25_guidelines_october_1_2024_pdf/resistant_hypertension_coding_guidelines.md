# Resistant Hypertension Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Resistant Hypertension Coding Guidelines

**Key Behaviors:**
- Assign code I1A.0 as an additional code when apparent, treatment resistant, or true resistant hypertension is documented by the provider.
- Sequence a code for the specific type of existing hypertension first if known, followed by the resistant hypertension code.
- Require two codes for secondary hypertension: one for the underlying etiology and one from category I15.

**Requirements / Properties:**
- Provider must explicitly document 'apparent treatment resistant hypertension', 'treatment resistant hypertension', or 'true resistant hypertension'.
- Documentation must confirm blood pressure remains above goal despite the use of antihypertensive medications.
- For secondary hypertension, both the underlying etiology code and the category I15 code must be present.
- Sequencing of codes is determined by the reason for admission or encounter.

**New in this version:**
- Code I1A.0 is designated for resistant hypertension scenarios.
