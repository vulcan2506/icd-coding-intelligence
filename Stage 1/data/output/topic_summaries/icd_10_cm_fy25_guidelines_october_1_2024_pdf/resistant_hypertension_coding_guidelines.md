# Resistant Hypertension Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Resistant Hypertension Coding Guidelines

**Key Behaviors:**
- Assign code I10.0 as an additional code when apparent treatment resistant hypertension, treatment resistant hypertension, or true resistant hypertension is documented by the provider.
- Sequence a code for the specific type of existing hypertension first if known, followed by the resistant hypertension code.

**Mandatory Coding Criteria:**
- Provider must explicitly document apparent treatment resistant hypertension, treatment resistant hypertension, or true resistant hypertension.
- A code for the specific type of existing hypertension must be sequenced first if known.
