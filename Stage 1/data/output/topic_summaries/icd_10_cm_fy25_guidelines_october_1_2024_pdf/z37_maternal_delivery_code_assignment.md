# Z37 Maternal Delivery Code Assignment
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Z37 Maternal Delivery Code Assignment

**Key Behaviors:**
- A code from category Z37, Outcome of delivery, should be included on every maternal record when a delivery has occurred.
- These codes are not to be used on subsequent records or on the newborn record.

**Requirements / Properties:**
- A delivery event must have occurred for the Z37 code to be assigned.
- The code must be assigned to the maternal record, not the newborn record.

**Deprecated in this version:**
- Using Z37 codes on subsequent maternal records.
- Using Z37 codes on the newborn record.
