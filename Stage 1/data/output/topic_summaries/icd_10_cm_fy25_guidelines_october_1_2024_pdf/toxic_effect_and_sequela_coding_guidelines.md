# Toxic Effect and Sequela Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Toxic Effect and Sequela Coding Guidelines

**Key Behaviors:**
- Classify ingestion or contact with harmful substances as toxic effect codes in categories T51-T65.
- Assign the toxic effect code first, followed by codes for all associated manifestations.
- Include an associated intent code (accidental, intentional self-harm, assault, or undetermined) for toxic effects.

**Requirements / Properties:**
- Documentation must confirm the substance was ingested or came in contact with the person.
- All associated manifestations of the toxic effect must be coded.
- Intent must be documented to select the correct category (accidental, intentional self-harm, assault, undetermined).
