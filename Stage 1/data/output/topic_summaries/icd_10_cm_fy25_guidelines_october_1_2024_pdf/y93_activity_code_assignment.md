# Y93 Activity Code Assignment
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Y93 Activity Code Assignment

**Key Behaviors:**
- Assign a code from category Y93 to describe the patient's activity at the time the injury or health condition occurred.
- An activity code is used only once, at the initial encounter for treatment.
- Only one code from Y93 should be recorded on a medical record.
- Do not assign Y93.9, Unspecified activity, if the activity is not stated.
- Activity codes are not applicable to poisonings, adverse effects, misadventures, or sequela.

**Mandatory Coding Criteria:**
- The activity must be explicitly stated in the documentation.
- The code must be assigned at the initial encounter for treatment.
- The code must be used in conjunction with external cause and intent codes if the activity provides additional information about the event.
