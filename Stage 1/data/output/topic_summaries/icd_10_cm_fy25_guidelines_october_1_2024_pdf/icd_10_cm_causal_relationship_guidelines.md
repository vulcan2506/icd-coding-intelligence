# ICD-10-CM Causal Relationship Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Causal Relationship Guidelines

**Key Behaviors:**
- Interpret 'with' or 'in' in code titles or instructional notes to mean 'associated with' or 'due to', presuming a causal relationship between linked conditions.
- Code conditions as related even without explicit provider documentation if linked by relational terms, unless documentation clearly states they are unrelated or another guideline requires documented linkage.
- Require explicit provider documentation linking conditions to code them as related when conditions are not specifically linked by relational terms in the classification.
- Recognize that 'with' in the Alphabetic Index is sequenced immediately following the main term or subterm, not in alphabetical order.

**Requirements / Properties:**
- Provider documentation must explicitly link conditions to code them as related when the classification does not specifically link them via relational terms.
- Documentation must clearly state conditions are unrelated to prevent coding them as related when an explicit causal link is absent and no specific guideline overrides this.

**New in this version:**
- The presumption of a causal relationship for conditions linked by 'with' or 'in' in the Alphabetic Index or Tabular List, even in the absence of explicit provider documentation.
