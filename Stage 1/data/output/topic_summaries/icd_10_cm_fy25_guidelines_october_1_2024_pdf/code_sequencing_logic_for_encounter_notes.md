# Code Sequencing Logic for Encounter Notes
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Code Sequencing Logic for Encounter Notes

**Key Behaviors:**
- A 'code also' note instructs that two codes may be required to fully describe a condition.
- The 'code also' note does not provide sequencing direction.
- Sequencing depends on the circumstances of the encounter.

**Requirements / Properties:**
- Two codes must be assigned to fully describe the condition when a 'code also' note is present.
- The coder must determine the correct sequence based on the specific circumstances of the encounter.
