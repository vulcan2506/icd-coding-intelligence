# Sequela and Late Effect Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Sequela and Late Effect Coding Guidelines

**Key Behaviors:**
- A sequela is the residual effect after the acute phase of an illness or injury has terminated with no time limit on usage.
- Coding generally requires two codes sequenced: the condition or nature of the sequela first, followed by the sequela code second.
- Sequela external cause codes must use the 7th character 'S' and are used with any report of a late effect or sequela resulting from a previous injury.
- For subsequent visits treating a late effect, use a late effect external cause code; do not use for follow-up care without documented late effects.
- A sequela external cause code should never be used with a related current nature of injury code.
- An exception exists where the sequela code is followed by a manifestation code identified in the Tabular List title.
- Sequela codes may be expanded at the fourth, fifth, or sixth character levels to include specific manifestations.

**Requirements / Properties:**
- Documentation must clearly indicate the residual effect is a result of a terminated acute phase of illness or injury.
- The condition or nature of the sequela must be documented to be sequenced first in the claim.
- External cause codes must include the 7th character 'S' to denote sequela status.
- Late effect external cause codes are mandatory for subsequent visits where a late effect is being treated.
- Current nature of injury codes are prohibited when paired with sequela external cause codes.

**Deprecated in this version:**
- Using the code for the acute phase of an illness or injury alongside a code for the late effect.
- Using a sequela external cause code with a related current nature of injury code.
- Using a late effect external cause code for subsequent visits when no late effect of the injury has been documented.
- Assigning unspecified laterality codes when documentation does not specify the side.
- Using sequela codes without sequencing the condition or nature of the sequela first.

**New in this version:**
- Expanded sequela codes at the fourth, fifth, or sixth character levels that include specific manifestations.
- Mandatory use of 7th character 'S' for external cause codes related to sequela.
- Specific guidelines for sequencing cerebrovascular disease sequelae and pregnancy complication sequelae.
- Requirement to query providers if laterality is unspecified before assigning codes.
- Distinction between treating a late effect versus follow-up care for injury assessment.
- Specific guidelines exist for sequencing cerebrovascular disease sequelae.
- Specific guidelines exist for sequencing sequelae of pregnancy complications.
