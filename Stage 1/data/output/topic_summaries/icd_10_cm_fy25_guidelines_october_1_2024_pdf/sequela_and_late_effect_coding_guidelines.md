# Sequela and Late Effect Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Sequela and Late Effect Coding Guidelines

**Key Behaviors:**
- Sequela codes are used for residual effects after the acute phase of an illness or injury has terminated, with no time limit on usage.
- Coding generally requires two codes sequenced in the following order: the condition or nature of the sequela is sequenced first, followed by the sequela code.
- Sequela external cause codes must use the 7th character 'S' and should be used with any report of a late effect or sequela resulting from a previous injury.
- A sequela external cause code should never be used with a related current nature of injury code.
- Late effect external cause codes should be used for subsequent visits when a late effect of the initial injury is being treated.
- Sequela codes may be expanded at the fourth, fifth, or sixth character levels to include specific manifestations.
- If the sequela code is followed by a manifestation code identified in the Tabular List and title, the manifestation code takes precedence.

**Mandatory Coding Criteria:**
- The acute phase of an illness or injury that led to the sequela is never used with a code for the late effect.
- If the sequela code is followed by a manifestation code identified in the Tabular List, the manifestation code takes precedence.
- Late effect external cause codes must not be used for subsequent visits for follow-up care (e.g., to assess healing, to receive rehabilitative therapy) when no late effect of the injury has been documented.
- The condition or nature of the sequela must be sequenced first before the sequela code.
- Sequela codes may be expanded at the fourth, fifth, or sixth character levels to include specific manifestations.
- Sequela external cause codes must use the 7th character 'S'.
- A sequela external cause code should never be used with a related current nature of injury code.
