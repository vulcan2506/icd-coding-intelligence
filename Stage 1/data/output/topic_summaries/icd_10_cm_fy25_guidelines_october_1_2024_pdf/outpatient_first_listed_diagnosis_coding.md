# Outpatient First-Listed Diagnosis Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Outpatient First-Listed Diagnosis Coding

**Key Behaviors:**
- The term first-listed diagnosis is used in lieu of principal diagnosis in the outpatient setting.
- ICD-10-CM coding conventions and general/disease-specific guidelines take precedence over outpatient guidelines.
- The search for correct code assignment must begin through the Alphabetic Index, never the Tabular List initially.
- For diagnostic services only, sequence the chiefly responsible diagnosis first, with chronic conditions as additional diagnoses.
- For interpreted diagnostic tests with available final reports, code confirmed or definitive diagnoses without coding related signs and symptoms.
- Diagnoses may not be established at the time of the initial encounter/visit.
- It may take two or more visits before the diagnosis is confirmed.

**Mandatory Coding Criteria:**
- Diagnosis must be chiefly responsible for the outpatient services provided during the encounter/visit.
- Final report must be available at the time of coding for interpreted diagnostic tests.
- Related signs and symptoms must not be coded as additional diagnoses when a confirmed diagnosis exists.
- Routine laboratory/radiology testing without associated signs/symptoms requires assignment of Z01.89.
- If routine testing occurs with a non-routine test, both the Z code and the code for the non-routine test reason must be assigned.
