# COVID-19 Vaccination Status Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** COVID-19 Vaccination Status Coding

**Key Behaviors:**
- Code Z28.310 is assigned when the patient has not received a COVID-19 vaccine of any type.
- Code Z28.311 is assigned when the patient has been partially vaccinated for COVID-19 as per CDC recommendations.
- Vaccination status coding relies on CDC guidelines in place at the time of the encounter.

**Requirements / Properties:**
- Documentation must confirm the patient has not received any COVID-19 vaccine for Z28.310.
- Documentation must confirm partial vaccination status aligns with current CDC recommendations for Z28.311.
- Clinicians other than the patient's provider must follow Section I.B.14 for underimmunization documentation.

**New in this version:**
- Introduction of specific ICD-10-CM codes Z28.310 and Z28.311 for tracking vaccination status.
- Mandatory alignment of vaccination status definitions with CDC recommendations at the time of service.
