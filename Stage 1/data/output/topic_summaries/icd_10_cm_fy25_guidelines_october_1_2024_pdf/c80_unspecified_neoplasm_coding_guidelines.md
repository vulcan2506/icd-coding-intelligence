# C80 Unspecified Neoplasm Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** C80 Unspecified Neoplasm Coding

**Key Behaviors:**
- Code C80.1 equates to Cancer, unspecified.
- This code should only be used when no determination can be made as to the primary site of a malignancy.
- This code should rarely be used in the inpatient setting.

**Mandatory Coding Criteria:**
- No determination can be made as to the primary site of a malignancy.
