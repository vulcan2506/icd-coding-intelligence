# CKD Post-Transplant Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** CKD Post-Transplant Coding Guidelines

**Key Behaviors:**
- Assign appropriate N18 code for the patient's stage of CKD and code Z94.0 for kidney transplant status.
- Do not code CKD alone as a transplant complication.
- If a transplant complication such as failure or rejection is documented, follow section I.C.19.g for coding complications.
- Query the provider if documentation is unclear regarding the presence of a transplant complication.

**Requirements / Properties:**
- Documentation must explicitly state the stage of CKD to assign the correct N18 code.
- Code Z94.0 must be assigned to indicate kidney transplant status.
- Transplant complications must be clearly documented as failure, rejection, or other specific complications to be coded separately from CKD.

**Deprecated in this version:**
- Coding CKD alone as a transplant complication without additional documentation of failure or rejection.

**New in this version:**
- Mandatory assignment of Z94.0 alongside N18 codes for patients with CKD post-transplant.
