# ICD-10-CM Coding for Radiation and Chemotherapy Admissions
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Coding for Radiation and Chemotherapy Admissions

**Key Behaviors:**
- Assign Z51.0, Z51.11, or Z51.12 as the principal diagnosis for external beam radiotherapy, immunotherapy, or chemotherapy admissions
- Sequence complication codes (e.g., uncontrolled nausea, vomiting, dehydration) after the principal therapy code
- Assign the appropriate malignancy code as the principal diagnosis for brachytherapy admissions
- Sequence complication codes after the malignancy code for brachytherapy admissions
- Apply these rules when patients develop specific complications during the admission

**Mandatory Coding Criteria:**
- Documentation must confirm the admission purpose is for external beam radiotherapy, immunotherapy, or chemotherapy
- Documentation must confirm the admission purpose is for insertion or implantation of radioactive elements
- Complications must be documented as occurring during the admission
