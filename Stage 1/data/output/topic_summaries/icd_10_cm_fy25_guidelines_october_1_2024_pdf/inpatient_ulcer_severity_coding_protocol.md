# Inpatient Ulcer Severity Coding Protocol
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Inpatient Ulcer Severity Coding Protocol

**Key Behaviors:**
- Assign two separate codes for ulcers that progress in severity during an inpatient stay.
- The first code represents the site and severity level of the ulcer on admission.
- The second code represents the same ulcer site and the highest severity level reported during the stay.

**Mandatory Coding Criteria:**
- Patient must be admitted to an inpatient hospital.
- The ulcer must be a non-pressure ulcer.
- The ulcer must progress to a higher severity level during the hospital stay.
- Documentation must specify the site and severity level on admission.
- Documentation must specify the highest severity level reported during the stay.
- See Section I.B.14. for pressure ulcer stage documentation by clinicians other than patient's provider
