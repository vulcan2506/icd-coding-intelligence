# Multiple Code Sequencing Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Multiple Code Sequencing Rules

**Key Behaviors:**
- 'Use additional code' notes indicate that a secondary code should be added if known to fully describe a condition.
- When a 'code first' note is present and an underlying condition exists, the underlying condition must be sequenced first.
- 'Code, if applicable, any causal condition first' notes allow the current code to be the principal diagnosis only if the causal condition is unknown or not applicable.
- If a causal condition is known for a 'code first' scenario, the code for that condition must be sequenced as the principal diagnosis.
- Multiple codes are required for sequela, complication codes, and obstetric codes to fully describe the condition.
- For bacterial infections not included in Chapter 1, a secondary code from category B95 or B96 must be added to identify the specific bacterial organism.
- 'Use additional code' notes appear at codes not part of an etiology/manifestation pair where a secondary code is useful to fully describe a condition.

**Requirements / Properties:**
- The underlying condition must be known to sequence it first when a 'code first' note is present.
- The causal condition must be known to sequence it as the principal diagnosis when applicable.
- The secondary code must be known to be added when a 'use additional code' note is present.
- The condition must affect multiple body systems or require more than one code to be fully described.
- The specific guidelines for sequela, complication, and obstetric codes must be consulted for further instruction.
- The specific guidelines for sequela, complication, and obstetric codes must be consulted for further instruction regarding multiple code requirements.
