# Superficial Injury Coding Exclusion
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Superficial Injury Coding Exclusion

**Key Behaviors:**
- Superficial injuries such as abrasions or contusions are not coded when associated with more severe injuries of the same site.
- Coding focus shifts exclusively to the more severe injury present at the same anatomical location.
- Superficial injury codes are suppressed to prevent redundancy and ensure accurate severity representation.

**Requirements / Properties:**
- Documentation must explicitly link the superficial injury to a more severe injury at the same site.
- The more severe injury must be confirmed as the primary diagnosis for the encounter.

**Deprecated in this version:**
- Coding both the superficial injury and the more severe injury as separate primary diagnoses for the same site.
- Assigning an abrasion code when a laceration or fracture exists at the identical location.

**New in this version:**
- Mandatory exclusion of superficial injury codes when a more severe injury is documented at the same site.
