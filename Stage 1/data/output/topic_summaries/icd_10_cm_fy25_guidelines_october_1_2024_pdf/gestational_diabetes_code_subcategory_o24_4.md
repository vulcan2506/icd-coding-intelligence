# Gestational Diabetes Code Subcategory O24.4
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Gestational Diabetes Coding and Exclusions

**Key Behaviors:**
- Codes for gestational diabetes are restricted to subcategory O24.4.
- No other code from category O24 should be used with a code from O24.4.
- When a patient is treated with both diet and insulin, only the code for insulin-controlled is required.
- When a patient is treated with both diet and oral hypoglycemic medications, only the code for controlled by oral hypoglycemic drugs is required.
- Codes Z79.4, Z79.84, and Z79.85 should not be assigned with codes from subcategory O24.4.

**Mandatory Coding Criteria:**
- Patient must not have been diabetic prior to pregnancy.
- Condition must occur during the second or third trimester of pregnancy.
- Treatment method must be documented to determine the specific O24.4 code (diet, insulin, or oral hypoglycemic).
- If multiple treatments are used, the most intensive treatment (insulin or oral hypoglycemic) dictates the code assignment.
- Abnormal glucose tolerance in pregnancy requires assignment from subcategory O99.81, not O24.4.
