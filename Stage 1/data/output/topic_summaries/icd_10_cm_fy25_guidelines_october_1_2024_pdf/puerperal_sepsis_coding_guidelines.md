# Puerperal Sepsis Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Puerperal Sepsis Coding Guidelines

**Key Behaviors:**
- Assign code O85 with a secondary code to identify the causal organism (e.g., B95-B96).
- Do not use codes from category A40 or A41 for puerperal sepsis.
- Use additional codes to identify severe sepsis (R65.2-) and associated acute organ dysfunction if applicable.
- Do not assign code O85 for sepsis following an obstetrical procedure.

**Mandatory Coding Criteria:**
- Secondary code must identify the causal organism.
- Causal organism codes must be from category B95-B96.
- Codes from category A40 or A41 must not be used for puerperal sepsis.
- Severe sepsis and acute organ dysfunction must be coded separately if applicable.
