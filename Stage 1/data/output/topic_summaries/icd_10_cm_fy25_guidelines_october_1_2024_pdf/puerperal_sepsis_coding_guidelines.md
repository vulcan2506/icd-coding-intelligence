# Puerperal Sepsis Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Puerperal Sepsis Coding Guidelines

**Key Behaviors:**
- Assign code O85 with a secondary code to identify the causal organism from categories B95-B96
- Do not use category A40 or A41 codes for puerperal sepsis
- Use additional codes to identify severe sepsis (R65.2-) and associated acute organ dysfunction if applicable
- Do not assign code O85 for sepsis following an obstetrical procedure

**Requirements / Properties:**
- Documentation must identify the causal organism to support the secondary B95-B96 code assignment
- Clinical documentation must confirm the sepsis is puerperal in nature and not post-procedural
- Documentation of severe sepsis or acute organ dysfunction is required to justify additional R65.2- codes

**Deprecated in this version:**
- Using category A40 (Streptococcal sepsis) or A41 (Other sepsis) for puerperal sepsis cases
- Assigning code O85 for sepsis resulting from an obstetrical procedure

**New in this version:**
- Mandatory use of secondary codes from B95-B96 to specify the bacterial organism for O85
- Requirement to code severe sepsis (R65.2-) and acute organ dysfunction alongside O85 when clinically present
