# ICD-10-CM Principal Diagnosis Sequencing
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Etiology/Manifestation Sequencing and Interrelated Conditions

**Key Behaviors:**
- Underlying etiology conditions must be sequenced first, followed by manifestation codes.
- Manifestation codes titled 'in diseases classified elsewhere' are never permitted as first-listed or principal diagnosis codes.
- Alphabetic Index entries listing etiology followed by manifestation codes in brackets indicate the code in brackets must be sequenced second.
- For interrelated conditions potentially meeting principal diagnosis definition, either may be sequenced first unless specific circumstances indicate otherwise.
- 'Code first' and 'Use additional code' notes serve as sequencing rules for both etiology/manifestation combinations and other specific codes.
- Codes with 'in diseases classified elsewhere' in the title are never permitted to be used as first-listed or principal diagnosis codes.
- The code in brackets in the Alphabetic Index entry must always be sequenced second.

**Mandatory Coding Criteria:**
- Manifestation codes must be used in conjunction with an underlying condition code.
- Manifestation codes must be listed following the underlying condition code.
- The code in brackets in the Alphabetic Index must always be sequenced second.
- Sequencing must follow the Tabular List, Alphabetic Index, circumstances of admission, or therapy provided.
- Codes with 'in diseases classified elsewhere' in the title must not be used as principal diagnosis.
