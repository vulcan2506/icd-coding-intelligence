# Outpatient Surgery Coding Protocol
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Outpatient Surgery Coding Protocol

**Key Behaviors:**
- Code the reason for the surgery as the first-listed diagnosis (reason for the encounter) for outpatient surgery cases.
- Apply this coding rule even if the surgery is not performed due to a contraindication.

**Requirements / Properties:**
- The encounter must be classified as an outpatient surgery (same day surgery).
- The reason for the surgery must be documented as the primary reason for the encounter.
