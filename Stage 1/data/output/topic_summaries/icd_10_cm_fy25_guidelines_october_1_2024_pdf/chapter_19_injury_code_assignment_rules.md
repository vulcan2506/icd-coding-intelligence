# Chapter 19 Injury Code Assignment Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Injury Code Assignment Exclusion for Medical Interventions

**Key Behaviors:**
- Injury codes from Chapter 19 should not be assigned for injuries that occur during, or as a result of, a medical intervention.
- Assign the appropriate complication code(s) instead of injury codes for such events.

**Mandatory Coding Criteria:**
- The injury must occur during or as a result of a medical intervention to trigger this rule.
- An appropriate complication code must be available and assigned.
