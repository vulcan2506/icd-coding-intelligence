# Preoperative Evaluation Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Preoperative Evaluation Coding

**Key Behaviors:**
- Sequence first a code from subcategory Z01.81 for encounters involving preoperative evaluations only.
- Assign an additional diagnosis code to describe the reason for the surgery.
- Code any findings related to the preoperative evaluation.

**Mandatory Coding Criteria:**
- The encounter must be for preoperative evaluations only.
- A code from subcategory Z01.81 must be sequenced first.
- The reason for surgery must be documented as an additional diagnosis.
