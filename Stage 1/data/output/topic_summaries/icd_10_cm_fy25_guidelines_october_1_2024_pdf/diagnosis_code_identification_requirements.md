# Diagnosis Code Identification Requirements
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Diagnosis Code Identification Requirements

**Key Behaviors:**
- Codes from A00.0 through T88.9, Z00-Z99, and U00-U85 must be used to identify diagnoses, symptoms, conditions, problems, complaints, or other reasons for the encounter/visit.

**Mandatory Coding Criteria:**
- Diagnosis codes must be selected from the specific range A00.0 through T88.9.
- Diagnosis codes must be selected from the specific range Z00 through Z99.
- Diagnosis codes must be selected from the specific range U00 through U85.
