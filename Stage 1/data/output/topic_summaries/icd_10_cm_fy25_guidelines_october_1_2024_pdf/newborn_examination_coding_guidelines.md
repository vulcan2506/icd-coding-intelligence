# Newborn Examination Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Newborn Examination Coding Guidelines

**Key Behaviors:**
- Code all clinically significant conditions noted on routine newborn examination that require clinical evaluation, therapeutic treatment, diagnostic procedures, extended hospital stay, increased nursing care, or have implications for future health care needs.
- Do not assign a code for prematurity unless it is explicitly documented by the provider.
- Assign codes in categories P05 and P07 based on recorded birth weight and estimated gestational age.
- When both birth weight and gestational age are available, assign two codes from category P07 with the birth weight code sequenced before the gestational age code.
- Codes should be assigned for conditions specified by the provider as having implications for future health care needs.

**Requirements / Properties:**
- Conditions must require clinical evaluation, therapeutic treatment, diagnostic procedures, extended length of hospital stay, increased nursing care and/or monitoring, or have implications for future health care needs to be coded.
- Prematurity must be explicitly documented by the provider before a code is assigned.
- Codes for disorders related to slow fetal growth (P05) and short gestation/low birth weight (P07) must be based on recorded birth weight and estimated gestational age.
- If both birth weight and gestational age are available, two codes from category P07 must be assigned.
- The code for birth weight must be sequenced before the code for gestational age when both are assigned from category P07.
