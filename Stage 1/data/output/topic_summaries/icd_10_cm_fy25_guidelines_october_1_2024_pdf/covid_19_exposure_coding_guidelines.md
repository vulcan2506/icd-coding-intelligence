# COVID-19 Exposure Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** COVID-19 Exposure Coding Guidelines

**Key Behaviors:**
- Assign code Z20.822 for asymptomatic individuals with actual or suspected exposure to COVID-19.
- Assign code Z20.822 for symptomatic individuals where infection is ruled out or test results are inconclusive/unknown.
- Refer to guideline I.C.21.c.1 for additional guidance on category Z20 code usage.
- Direct to guideline I.C.1.g.1.a if COVID-19 infection is confirmed.
- Distinguish between exposure status and confirmed infection status for appropriate code selection.

**Requirements / Properties:**
- Documentation must confirm asymptomatic status or rule out infection for symptomatic individuals to use Z20.822.
- Test results must be inconclusive or unknown for symptomatic individuals to use Z20.822.
- Confirmed COVID-19 cases must follow guideline I.C.1.g.1.a instead of exposure codes.
- Adherence to guideline I.C.21.c.1 is required for proper use of category Z20 codes.
- Differentiation between actual/suspected exposure and confirmed infection is mandatory.

**New in this version:**
- Introduction of specific code Z20.822 for contact with and (suspected) exposure to COVID-19.
- Mandatory application of Z20.822 for asymptomatic individuals with exposure regardless of symptom status.
- Clarification that symptomatic individuals with ruled-out infection require Z20.822 rather than infection codes.
