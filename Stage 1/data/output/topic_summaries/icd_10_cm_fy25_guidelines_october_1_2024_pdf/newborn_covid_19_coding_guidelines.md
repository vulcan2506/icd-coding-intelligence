# Newborn COVID-19 Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Newborn COVID-19 Coding Guidelines

**Key Behaviors:**
- Assign code U07.1 for newborns testing positive for COVID-19 without documented transmission type.
- Assign codes P35.8 and U07.1 when the condition is documented as contracted in utero or during birth.
- Assign a code from category Z38 as the principal diagnosis for the birth episode in newborn records.

**Mandatory Coding Criteria:**
- Documentation must indicate the newborn tested positive for COVID-19.
- Documentation must specify transmission occurred in utero or during birth to assign P35.8.
- Principal diagnosis must be from category Z38 for the birth episode.
