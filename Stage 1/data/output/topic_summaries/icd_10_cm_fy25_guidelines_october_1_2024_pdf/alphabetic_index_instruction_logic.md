# Alphabetic Index Instruction Logic
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Alphabetic Index Instruction Logic

**Key Behaviors:**
- The 'see' instruction following a main term indicates that another term must be referenced to locate the correct code.
- The 'see also' instruction indicates an additional main term that may provide useful entries but is not mandatory if the original term provides the necessary code.
- It is necessary to follow the 'see' note to the referenced main term to find the valid code.
- Followers of 'see also' notes are optional when the primary main term already yields the required code.
- These instructions guide coders on whether to search alternative terms or rely on the initial search result.

**Requirements / Properties:**
- When a 'see' instruction is present, the coder must navigate to the referenced main term to assign a code.
- The 'see also' instruction does not mandate a code lookup if the primary term is sufficient.
