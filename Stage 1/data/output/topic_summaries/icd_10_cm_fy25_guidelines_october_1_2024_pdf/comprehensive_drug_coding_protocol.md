# Comprehensive Drug Coding Protocol
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Comprehensive Drug Coding Protocol

**Key Behaviors:**
- Use as many codes as necessary to describe completely all drugs, medicinal or biological substances.
- Assign appropriate code for the nature of the adverse effect followed by the appropriate code for the adverse effect of the drug (T36-T50) when coding adverse effects of correctly prescribed and properly administered drugs.

**Mandatory Coding Criteria:**
- The adverse effect must be of a drug that has been correctly prescribed and properly administered.
- The code for the drug must have a 5th or 6th character '5' (e.g., T36.0X5-).
- Examples of nature of adverse effect include tachycardia, delirium, gastrointestinal hemorrhaging, vomiting, hypokalemia, hepatitis, renal failure, or respiratory failure.
