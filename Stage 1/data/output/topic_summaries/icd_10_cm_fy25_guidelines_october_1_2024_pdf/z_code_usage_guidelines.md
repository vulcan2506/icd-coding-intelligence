# Z Code Usage Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Z Code Usage Guidelines

**Key Behaviors:**
- Z codes are applicable for use in any healthcare setting.
- Z codes may be assigned as a first-listed or principal diagnosis code in the inpatient setting.
- Z codes may be assigned as a secondary code depending on the circumstances of the encounter.
- Certain specific Z codes are restricted to use only as first-listed or principal diagnosis codes.
- Selection of Z code position depends on the clinical circumstances of the encounter.

**Requirements / Properties:**
- The code must be relevant to the circumstances of the healthcare encounter.
- Specific Z codes must be verified against restrictions regarding first-listed or principal diagnosis usage.
