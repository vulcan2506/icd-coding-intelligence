# Z Code Usage Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Z Code Usage Guidelines

**Key Behaviors:**
- Z codes are for use in any healthcare setting.
- Z codes may be used as either a first-listed or secondary code depending on the circumstances of the encounter.
- Certain Z codes may only be used as first-listed or principal diagnosis.

**Mandatory Coding Criteria:**
- Z codes must be selected based on the specific circumstances of the encounter.
- First-listed status is required for certain specific Z codes.
