# Neoplasm and Anemia Encounter Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Neoplasm and Anemia Encounter Sequencing

**Key Behaviors:**
- When admission is for management of anemia associated with malignancy and treatment is only for anemia, sequence malignancy code first followed by anemia code D63.0.
- When admission is for management of a complication associated with neoplasm (e.g., dehydration) and treatment is only for the complication, sequence complication code first followed by neoplasm code.

**Mandatory Coding Criteria:**
- Treatment must be exclusively for anemia to sequence malignancy as principal diagnosis.
- Treatment must be exclusively for the complication to sequence complication as principal diagnosis.
- Anemia must be documented as associated with the malignancy.
- Code D63.0 must be used for anemia in neoplastic disease.
- Provider must document the specific complication being managed.
