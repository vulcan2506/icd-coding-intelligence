# Perinatal Condition Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Perinatal Condition Coding Guidelines

**Key Behaviors:**
- The perinatal period is defined as before birth through the 28th day following birth for coding and reporting purposes.
- If a condition originates in the perinatal period and continues throughout the patient's life, the perinatal code must continue to be used regardless of the patient's age.

**Requirements / Properties:**
- Conditions must originate within the defined perinatal period (before birth to 28th day post-birth) to be coded using perinatal codes.
- Continuity of care documentation must confirm the condition started in the perinatal period to justify using the perinatal code for adult patients.
