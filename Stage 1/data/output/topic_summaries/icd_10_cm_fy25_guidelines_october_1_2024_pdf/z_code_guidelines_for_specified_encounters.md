# Z Code Guidelines for Specified Encounters
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Z Code Guidelines for Specified Encounters

**Key Behaviors:**
- Z codes are used to identify the reason for encounters that are not related to a disease or injury
- Z codes should be assigned when the encounter is for specified circumstances such as health checkups or screenings
- Z codes must be selected based on the specific purpose of the encounter as documented in the medical record

**Requirements / Properties:**
- The encounter must be explicitly for a specified circumstance rather than a disease or injury
- Documentation must clearly indicate the reason for the encounter to justify the use of a Z code
- The selected Z code must accurately reflect the primary purpose of the healthcare visit
