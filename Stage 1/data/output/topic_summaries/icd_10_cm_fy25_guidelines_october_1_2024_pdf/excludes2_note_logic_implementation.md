# Excludes2 Note Logic Implementation
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Excludes2 Note Logic Implementation

**Key Behaviors:**
- Represents 'Not included here' logic where the excluded condition is not part of the represented condition.
- Allows simultaneous coding of the code with the excluded code when both conditions are present in the patient.
- Indicates that the excluded condition is distinct from the condition represented by the code.

**Mandatory Coding Criteria:**
- Both conditions must be documented as present in the patient at the same time to use both codes together.
