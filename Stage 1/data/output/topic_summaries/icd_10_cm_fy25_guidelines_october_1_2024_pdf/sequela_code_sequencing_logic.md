# Sequela Code Sequencing Logic
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Sequela Code Sequencing Logic

**Key Behaviors:**
- Sequela codes must be sequenced following the code describing the sequelae of the complication.
- The sequela code is treated as a secondary diagnosis relative to the complication code.
- Coding order is determined by the causal relationship between the complication and its sequela.

**Requirements / Properties:**
- Documentation must explicitly link the current condition as a sequela of a prior complication.
- The complication code must be present and valid to support the sequela code assignment.

**Deprecated in this version:**
- Sequela codes being sequenced before the complication code.
- Assigning sequela codes without a documented antecedent complication.

**New in this version:**
- Mandatory sequencing rule requiring the complication code to precede the sequela code in the claim.
