# Sequela Code Sequencing Logic
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Sequela Code Sequencing Logic

**Key Behaviors:**
- Sequela codes must be sequenced following the code describing the sequelae of the complication.

**Mandatory Coding Criteria:**
- The code describing the sequelae of the complication must be assigned first.
