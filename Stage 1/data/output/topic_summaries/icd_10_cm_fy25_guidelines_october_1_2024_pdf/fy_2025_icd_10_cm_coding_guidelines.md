# FY 2025 ICD-10-CM Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Neoplasm Coding and Sequencing

**Key Behaviors:**
- Primary malignant neoplasms overlapping two or more contiguous sites must be classified to the subcategory/code .8 ('overlapping lesion') unless specifically indexed elsewhere.
- Malignant neoplasms of ectopic tissue must be coded to the site of origin mentioned in the documentation.
- The neoplasm table in the Alphabetic Index should be referenced first, but if the histological term is documented, that term must be referenced first to determine the appropriate column in the Neoplasm Table.
- Multiple neoplasms of the same site that are not contiguous (e.g., tumors in different quadrants of the same breast) require codes for each site to be assigned.
- The Tabular List must be referenced to verify the correct code and ensure a more specific site code does not exist.

**Mandatory Coding Criteria:**
- It is necessary to determine from the record if the neoplasm is benign, in-situ, malignant, or of uncertain histologic behavior.
- If malignant, any secondary (metastatic) sites should also be determined.
- The histological term must be referenced in the Alphabetic Index before the Neoplasm Table if documented.
- Verification in the Tabular List is required to confirm the correct code and rule out more specific site codes.
