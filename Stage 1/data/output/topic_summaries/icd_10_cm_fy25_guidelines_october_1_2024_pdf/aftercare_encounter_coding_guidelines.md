# Aftercare Encounter Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Aftercare Encounter Coding Guidelines

**Key Behaviors:**
- Aftercare Z codes should not be used if treatment is directed at a current, acute disease; instead, the diagnosis code must be used.
- Aftercare Z codes should not be used for aftercare of injuries; assign the acute injury code with the appropriate 7th character for subsequent encounter.
- Aftercare codes are generally first listed to explain the specific reason for the encounter, unless a diagnosis code is applicable.
- Certain aftercare Z code categories require a secondary diagnosis code to describe the resolving condition or sequelae.
- Status Z codes may be used with aftercare Z codes to indicate the nature of the aftercare, but not when the aftercare code already indicates the type of status.
- Aftercare codes should be used in conjunction with other aftercare codes or diagnosis codes to provide better detail on the specifics of an aftercare encounter visit, unless otherwise directed by the classification.
- Certain aftercare Z code categories need a secondary diagnosis code to describe the resolving condition or sequelae, while for others the condition is included in the code title.

**Requirements / Properties:**
- Exceptions exist for codes Z51.0 (Encounter for antineoplastic radiation therapy) and subcategory Z51.1 (Encounter for antineoplastic chemotherapy and immunotherapy), which must be first listed followed by the diagnosis code.
- If the reason for the encounter is more than one type of antineoplastic therapy, code Z51.0 and a code from subcategory Z51.1 may be assigned together, with one reported as a secondary diagnosis.
- An aftercare code may be used as an additional code when some type of aftercare is provided in addition to the reason for admission and no diagnosis code is applicable.
- Sequencing of multiple aftercare codes depends on the circumstances of the encounter.
- Additional Z code aftercare category terms include fitting and adjustment, and attention to artificial openings.

**Deprecated in this version:**
- Using aftercare Z codes for current, acute diseases.
- Using aftercare Z codes for aftercare of injuries.
- Using a status code when the aftercare code already indicates the type of status (e.g., using Z43.0 with Z93.0).
- Listing a diagnosis code before the aftercare code when the aftercare is the primary reason for the encounter.
- Failing to assign a secondary diagnosis code when required by specific aftercare Z code categories.

**New in this version:**
- Specific Z codes for plastic and reconstructive surgery following medical procedure or healed injury (Z42).
- Specific Z codes for attention to artificial openings (Z43).
- Specific Z codes for fitting and adjustment of external prosthetic device (Z44).
- Specific Z codes for surgical aftercare following surgery on the circulatory system (Z48.812).
- Mapping between Z49 (Encounter for care involving renal dialysis) and Z51 (Encounter for other aftercare and medical care).
- Z42 Encounter for plastic and reconstructive surgery following medical procedure or healed injury
- Z43 Encounter for attention to artificial openings
