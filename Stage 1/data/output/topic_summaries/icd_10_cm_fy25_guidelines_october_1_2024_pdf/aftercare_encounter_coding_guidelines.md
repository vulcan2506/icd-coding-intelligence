# Aftercare Encounter Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Aftercare Encounter Coding Guidelines

**Key Behaviors:**
- Aftercare Z codes are used when initial treatment is complete and the patient requires continued care during healing or recovery, or for long-term consequences.
- Aftercare Z codes should not be used if treatment is directed at a current, acute disease; instead, the diagnosis code is used.
- Aftercare Z codes should not be used for aftercare of injuries; assign the acute injury code with the appropriate 7th character for subsequent encounter.
- Aftercare codes are generally first listed to explain the specific reason for the encounter, unless a diagnosis code is applicable.
- Status Z codes may be used with aftercare Z codes to indicate the nature of the aftercare, except when the aftercare code already indicates the type of status.
- An aftercare code may be used as an additional code when some type of aftercare is provided in addition to the reason for admission and no diagnosis code is applicable.
- The sequencing of multiple aftercare codes depends on the circumstances of the encounter.

**Mandatory Coding Criteria:**
- For encounters chiefly to receive radiation therapy, chemotherapy, or immunotherapy for a neoplasm, code Z51.0 or Z51.1 must be listed first, followed by the diagnosis code.
- If the reason for the encounter is more than one type of antineoplastic therapy, code Z51.0 and a code from subcategory Z51.1 may be assigned together, with one reported as a secondary diagnosis.
- Certain aftercare Z code categories require a secondary diagnosis code to describe the resolving condition or sequelae.
- Status codes should not be used when the aftercare code indicates the type of status (e.g., do not use Z43.0 with Z93.0).
