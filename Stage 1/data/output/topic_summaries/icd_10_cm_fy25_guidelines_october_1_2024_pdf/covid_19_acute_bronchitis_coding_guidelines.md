# COVID-19 Acute Bronchitis Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** COVID-19 Acute Bronchitis Coding

**Key Behaviors:**
- Assign code U07.1 for confirmed COVID-19 cases
- Assign code J20.8 for acute bronchitis due to other specified organisms
- Assign code J40 for bronchitis NOS due to COVID-19

**Mandatory Coding Criteria:**
- Acute bronchitis must be confirmed as due to COVID-19
- Distinguish between acute bronchitis and bronchitis NOS
