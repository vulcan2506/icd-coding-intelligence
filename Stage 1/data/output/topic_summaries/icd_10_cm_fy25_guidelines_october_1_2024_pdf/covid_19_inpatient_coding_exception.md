# COVID-19 Inpatient Coding Exception
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** COVID-19 Inpatient Coding Exception

**Key Behaviors:**
- Assign code U07.1 for confirmed COVID-19 diagnoses based on provider documentation alone, without requiring a positive test result.
- Do not assign code U07.1 if the provider documents 'suspected', 'possible', 'probable', or 'inconclusive' COVID-19.
- Code signs and symptoms instead of U07.1 for uncertain COVID-19 diagnoses per guideline I.C.1.g.1.g.

**Mandatory Coding Criteria:**
- Provider must document a confirmed diagnosis of COVID-19.
- Provider documentation of a positive COVID-19 test result is sufficient but not strictly required for confirmation.
- Uncertain diagnoses (suspected, possible, probable, inconclusive) must be coded as signs and symptoms.
