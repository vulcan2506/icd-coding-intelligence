# COVID-19 Inpatient Coding Exception
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** COVID-19 Inpatient Coding Exception

**Key Behaviors:**
- Assign code U07.1 for confirmed COVID-19 diagnoses documented by the provider or via positive test results.
- Treat provider documentation of COVID-19 as sufficient confirmation without requiring explicit positive test result documentation.
- Do not assign code U07.1 if the provider documents 'suspected', 'possible', 'probable', or 'inconclusive' COVID-19.
- Code signs and symptoms instead of U07.1 when the diagnosis is not confirmed.
- Apply this exception to override standard hospital inpatient guideline Section II, H.

**Requirements / Properties:**
- Provider must document a confirmed diagnosis of COVID-19 or a positive test result.
- Diagnosis must not be qualified as 'suspected', 'possible', 'probable', or 'inconclusive'.

**Deprecated in this version:**
- Assigning code U07.1 for suspected, possible, probable, or inconclusive COVID-19 cases.
- Requiring explicit positive test result documentation when provider diagnosis is confirmed.

**New in this version:**
- Code U07.1 is valid based solely on provider documentation of confirmed COVID-19 without a positive test result.
- Guideline exception to hospital inpatient guideline Section II, H for confirmed COVID-19.
