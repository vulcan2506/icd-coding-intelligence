# MIS-C Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** MIS-C Coding Guidelines

**Key Behaviors:**
- Assign code U07.1 as the principal diagnosis for individuals with MIS and current COVID-19 infection.
- Assign code M35.81 as an additional diagnosis for Multisystem inflammatory syndrome in all MIS-C scenarios.
- Assign code U09.9 for individuals with a history of COVID-19 who develop MIS.
- Assign code Z20.822 for individuals with suspected exposure to COVID-19 who develop MIS without current infection or history.
- Assign additional codes for any associated complications of MIS.

**Requirements / Properties:**
- Current COVID-19 infection must be documented to assign U07.1 as the principal diagnosis.
- History of COVID-19 must be documented to assign U09.9 instead of U07.1.
- Known or suspected exposure to COVID-19 without current infection or history must be documented to assign Z20.822.
- Code M35.81 is mandatory as an additional diagnosis in all MIS-C cases.
- Associated complications must be documented to justify additional code assignment.

**New in this version:**
- Use of code U07.1 specifically for MIS-C with concurrent COVID-19 infection.
- Use of code U09.9 for MIS-C arising from post-COVID-19 condition.
- Use of code Z20.822 for MIS-C arising from exposure without infection.
- Mandatory pairing of M35.81 with specific COVID-19 related codes based on infection status.
- Requirement to code associated complications separately from the primary MIS diagnosis.
