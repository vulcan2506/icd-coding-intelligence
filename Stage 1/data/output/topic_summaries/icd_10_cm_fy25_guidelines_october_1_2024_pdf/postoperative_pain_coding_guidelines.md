# Postoperative Pain Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Postoperative Pain Coding Guidelines

**Key Behaviors:**
- Use provider documentation to guide coding of postoperative pain.
- Default to acute form code for unspecified post-thoracotomy or postoperative pain.
- Do not code routine or expected postoperative pain immediately after surgery.
- Assign non-complication postoperative pain to category G89.

**Mandatory Coding Criteria:**
- Pain must be documented by the provider.
- Pain must not be routine or expected immediately post-surgery.
- Pain must be associated with a specific complication to use complication codes instead of G89.
