# Malignant Neoplasm Transplant Coding Protocol
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Malignant Neoplasm Transplant Coding Protocol

**Key Behaviors:**
- Code malignant neoplasm of a transplanted organ as a transplant complication.
- Assign category T86.- code first, followed by code C80.2.
- Use an additional code for the specific malignancy.

**Mandatory Coding Criteria:**
- Condition must be a malignant neoplasm of a transplanted organ.
- Category T86.- code must be assigned before C80.2.
- Code C80.2 must be used for malignant neoplasm associated with transplanted organ.
- An additional code for the specific malignancy must be included.
