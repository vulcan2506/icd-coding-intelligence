# ICD-10 Code R75 Assignment for Inconclusive HIV
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM R75 Assignment for Inconclusive HIV

**Key Behaviors:**
- Assigns code R75 to patients with inconclusive HIV serology results
- Excludes assignment if a definitive HIV diagnosis is present
- Excludes assignment if clinical manifestations of HIV illness are present
- Applies specifically when laboratory evidence is the only finding
- Ensures code R75 is used only for inconclusive laboratory evidence
