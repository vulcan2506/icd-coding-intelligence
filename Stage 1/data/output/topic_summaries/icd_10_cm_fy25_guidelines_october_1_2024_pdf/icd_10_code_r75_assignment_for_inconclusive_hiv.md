# ICD-10 Code R75 Assignment for Inconclusive HIV
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Code R75 Assignment for Inconclusive HIV

**Key Behaviors:**
- Assign code R75 to patients with inconclusive HIV serology results
- Do not assign R75 if a definitive HIV diagnosis is established
- Do not assign R75 if manifestations of HIV illness are present
- Use R75 specifically when laboratory evidence is the only finding
- Exclude R75 assignment when clinical evidence confirms the infection

**Mandatory Coding Criteria:**
- Documentation must explicitly state inconclusive serology results
- Documentation must confirm absence of definitive diagnosis
- Documentation must confirm absence of HIV illness manifestations
