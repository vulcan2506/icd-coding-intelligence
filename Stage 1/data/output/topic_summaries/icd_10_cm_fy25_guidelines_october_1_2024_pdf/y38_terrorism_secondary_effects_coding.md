# Y38 Terrorism Secondary Effects Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Y38 Terrorism Secondary Effects Coding

**Key Behaviors:**
- Assign code Y38.9 for conditions occurring subsequent to the terrorist event
- Do not assign Y38.9 for conditions directly due to the initial terrorist act
- Combine Y38.9 with another Y38 code if an initial injury and a subsequent result both exist

**Requirements / Properties:**
- Clinical documentation must confirm the condition is a subsequent result of the event
- Documentation must distinguish between initial act injuries and secondary effects

**Deprecated in this version:**
- Assigning Y38.9 for conditions directly caused by the initial terrorist act

**New in this version:**
- Permitted combination of Y38.9 with other Y38 codes for dual injury scenarios
