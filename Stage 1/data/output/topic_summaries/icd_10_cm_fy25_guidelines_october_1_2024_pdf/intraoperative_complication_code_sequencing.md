# Intraoperative Complication Code Sequencing
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Intraoperative Complication Code Sequencing

**Key Behaviors:**
- Intraoperative and postprocedural complication codes found within body system chapters should be sequenced first.
- These codes are followed by a code for the specific complication, if applicable.
- Complication codes from body system chapters are assigned for intraoperative and postprocedural complications unless specifically indexed to a T code in chapter 19.

**Mandatory Coding Criteria:**
- The complication must be intraoperative or postprocedural.
- The complication must be specific to the organs and structures of the relevant body system chapter.
- The complication must not be specifically indexed to a T code in chapter 19.
