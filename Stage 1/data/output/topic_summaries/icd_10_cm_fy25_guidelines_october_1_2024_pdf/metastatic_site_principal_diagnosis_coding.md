# Metastatic Site Principal Diagnosis Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Metastatic Site Principal Diagnosis Coding

**Key Behaviors:**
- Designate metastatic site as principal diagnosis when treatment is directed toward it only
- Code primary malignancy as an additional code when metastatic site is principal

**Mandatory Coding Criteria:**
- Encounter must be for a primary malignancy with metastasis
- Treatment must be directed toward the metastatic site(s) only
