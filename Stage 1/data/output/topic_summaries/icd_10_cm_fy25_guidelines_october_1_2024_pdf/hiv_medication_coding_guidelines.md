# HIV Medication Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** HIV Medication Coding Guidelines

**Key Behaviors:**
- Assign code B20 for patients with documented HIV disease, HIV-related illness, or AIDS currently managed on antiretroviral medications.
- Assign code Z79.899 as an additional code to identify the long-term (current) use of antiretroviral medications.

**Requirements / Properties:**
- Patient must have documented HIV disease, HIV-related illness, or AIDS.
- Patient must be currently managed on antiretroviral medications.
