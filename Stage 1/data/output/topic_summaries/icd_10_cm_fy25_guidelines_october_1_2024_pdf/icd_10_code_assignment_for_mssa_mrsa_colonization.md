# ICD-10 Code Assignment for MSSA/MRSA Colonization
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Code Assignment for MSSA/MRSA Colonization

**Key Behaviors:**
- Assign code Z22.322 for patients documented as having MRSA colonization.
- Assign code Z22.321 for patients documented as having MSSA colonization.
- Recognize colonization as the presence of bacteria without necessarily causing illness or a specific disease process.

**Mandatory Coding Criteria:**
- Documentation must explicitly state the patient is a carrier or suspected carrier of Methicillin resistant Staphylococcus aureus.
- Documentation must explicitly state the patient is a carrier or suspected carrier of Methicillin susceptible Staphylococcus aureus.
- Provider must document colonization status (e.g., 'MRSA screen positive' or 'MRSA nasal swab positive') to support code assignment.
