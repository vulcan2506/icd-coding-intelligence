# ICD-10 Code Assignment for MSSA/MRSA Colonization
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Code Assignment for MSSA/MRSA Colonization

**Key Behaviors:**
- Assign code Z22.322 for patients documented as having MRSA colonization.
- Assign code Z22.321 for patients documented as having MSSA colonization.
- Recognize that colonization means the pathogen is present without necessarily causing illness.
- Accept documentation such as 'MRSA screen positive' or 'MRSA nasal swab positive' as evidence of colonization.
- Do not code colonization as a disease process or cause of a specific condition unless explicitly documented as such by the provider.

**Requirements / Properties:**
- Provider must explicitly document the presence of MSSA or MRSA colonization.
- Documentation must distinguish between colonization (carrier state) and active disease.
- If colonization is not documented as the cause of a specific condition, it should not be coded as such.

**New in this version:**
- Introduction of specific carrier codes Z22.321 and Z22.322 for Methicillin susceptible and resistant Staphylococcus aureus respectively.
