# Inpatient Z-Code Classification Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Inpatient Z-Code Classification Guidelines

**Key Behaviors:**
- Certain non-specific Z codes are discouraged in the inpatient setting due to redundancy with other codes.
- Use of Z codes in the outpatient setting is limited to instances where no further documentation permits more precise coding.
- Any sign, symptom, or reason for visit captured in another code should be used instead of a non-specific Z code.

**Mandatory Coding Criteria:**
- Documentation must permit more precise coding before a Z code can be used in the outpatient setting.
- The condition must not be better described by another specific code in the classification.
