# Inpatient Z-Code Classification Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Inpatient Z-Code Classification Guidelines

**Key Behaviors:**
- Certain Z codes are considered non-specific or redundant in the inpatient setting and should generally not be used.
- Z code use in the inpatient setting is restricted to instances where no further documentation permits more precise coding.
- Any sign, symptom, or reason for visit captured in another code should be used instead of a non-specific Z code.

**Requirements / Properties:**
- Documentation must permit more precise coding before a Z code can be justified in the inpatient setting.
- If a sign or symptom is captured in another code, that code must be used instead of a Z code.

**Deprecated in this version:**
- Using non-specific Z codes in the inpatient setting when more precise codes are available.
- Using Z codes for signs or symptoms that are already captured by other specific codes.
