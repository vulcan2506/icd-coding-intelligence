# Uncertain Diagnosis Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Uncertain Diagnosis Coding Guidelines

**Key Behaviors:**
- Code uncertain diagnoses (e.g., 'probable', 'suspected') as if they existed or were established for inpatient admissions.
- Basis for coding uncertain diagnoses includes diagnostic workup, arrangements for further workup, and initial therapeutic approach.
- Do not code diagnoses documented with uncertainty terms for outpatient encounters; instead, code to the highest degree of certainty (symptoms, signs, abnormal results).
- Apply uncertain diagnosis coding rules only to inpatient admissions in short-term, acute, long-term care, and psychiatric hospitals.
- Differentiate coding practices between inpatient settings (code as established) and outpatient settings (code as symptoms/signs).

**Requirements / Properties:**
- Diagnosis must be qualified by terms like 'probable', 'suspected', 'likely', 'questionable', 'possible', 'still to be ruled out', 'compatible with', or 'consistent with'.
- Setting must be an inpatient admission to a short-term, acute, long-term care, or psychiatric hospital.
- Coding must be supported by diagnostic workup, arrangements for further workup, or initial therapeutic approach corresponding to the established diagnosis.
- For non-inpatient encounters, documentation must support coding to the highest degree of certainty (symptoms, signs, abnormal test results).
- Uncertain diagnosis terms must not be used as the primary basis for coding in outpatient encounters.

**Deprecated in this version:**
- Coding uncertain diagnoses as if they were established for outpatient encounters.
- Using uncertain diagnosis terms (e.g., 'probable', 'suspected') as the primary basis for coding in outpatient settings.
- Applying inpatient uncertain diagnosis coding rules to outpatient encounters.
- Coding uncertain diagnoses without supporting diagnostic workup or therapeutic approach in inpatient settings.
- Failing to distinguish between inpatient and outpatient coding practices for uncertain diagnoses.

**New in this version:**
- Explicit clarification that uncertain diagnosis coding applies only to inpatient admissions.
- Mandatory requirement to code uncertain diagnoses as if established based on workup and therapeutic approach.
- Prohibition against coding uncertain diagnoses in outpatient settings; must code symptoms, signs, or abnormal results instead.
- Updated definitions for uncertain diagnosis terms including 'compatible with' and 'consistent with'.
- Emphasis on differentiating coding practices between inpatient and outpatient settings for uncertain diagnoses.
