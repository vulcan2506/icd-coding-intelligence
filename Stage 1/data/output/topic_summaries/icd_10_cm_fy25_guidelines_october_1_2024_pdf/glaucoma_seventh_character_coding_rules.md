# Glaucoma Seventh Character Coding Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Glaucoma Seventh Character Coding Rules

**Key Behaviors:**
- Assign seventh character '4' for indeterminate stage based on clinical documentation when stage cannot be determined
- Assign seventh character '0' for unspecified stage when there is no documentation regarding the stage of glaucoma
- Do not confuse seventh character '4' (indeterminate) with seventh character '0' (unspecified)

**Requirements / Properties:**
- Clinical documentation must explicitly indicate that the stage cannot be determined to justify character '4'
- Absence of any stage documentation requires the use of character '0' rather than '4'
