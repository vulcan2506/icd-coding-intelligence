# Glaucoma Seventh Character Coding Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Glaucoma Seventh Character Coding Rules

**Key Behaviors:**
- Seventh character '4' is assigned when the glaucoma stage cannot be clinically determined based on clinical documentation.
- Seventh character '0' is assigned when there is no documentation regarding the stage of the glaucoma.
- Seventh character '4' must not be confused with or substituted for seventh character '0'.

**Mandatory Coding Criteria:**
- Clinical documentation must explicitly indicate that the stage cannot be determined to justify character '4'.
- Absence of stage documentation requires the use of character '0' rather than '4'.
