# Cerebrovascular Accident Code Assignment
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Cerebrovascular Accident Code Assignment

**Key Behaviors:**
- Medical record documentation must clearly specify the cause- and-effect relationship between the medical intervention and the cerebrovascular accident.
- Code assignment depends on whether the event was an infarction or hemorrhage.
- Code assignment depends on whether the event occurred intraoperatively or postoperatively.
- For cerebral hemorrhage, code assignment depends on the specific type of procedure performed.

**Requirements / Properties:**
- Documentation must establish a clear causal link between the intervention and the stroke.
- The clinical record must distinguish between infarction and hemorrhage.
- The timing of the event (intraoperative vs. postoperative) must be explicitly documented.
- The specific type of procedure performed must be identified for hemorrhage cases.
