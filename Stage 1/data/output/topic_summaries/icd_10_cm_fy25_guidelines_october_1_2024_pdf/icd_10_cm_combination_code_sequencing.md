# ICD-10-CM Combination Code Sequencing
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Combination Code Sequencing

**Key Behaviors:**
- Combination codes for atherosclerotic heart disease with angina pectoris include both conditions, making an additional code for angina unnecessary.
- A causal relationship between atherosclerosis and angina pectoris is assumed unless documentation specifies a different cause for the angina.
- In cases of acute myocardial infarction (AMI) admission, the AMI must be sequenced before the underlying coronary artery disease.

**Mandatory Coding Criteria:**
- Documentation must indicate that angina is not due to a cause other than atherosclerosis to assume the causal relationship.
- Admission status must be due to acute myocardial infarction (AMI) to trigger the sequencing rule placing AMI first.
