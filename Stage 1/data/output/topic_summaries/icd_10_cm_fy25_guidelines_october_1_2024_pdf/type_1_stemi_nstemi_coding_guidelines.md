# Type 1 STEMI/NSTEMI Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Type 1 STEMI/NSTEMI Coding Guidelines

**Key Behaviors:**
- Subcategories I21.0-I21.2 and I21.3 are used for type 1 ST elevation myocardial infarction (STEMI).
- Code I21.4 is used for type 1 non-ST elevation myocardial infarction (NSTEMI) and nontransmural MIs.
- If a type 1 NSTEMI evolves to STEMI, assign the STEMI code.
- If a type 1 STEMI converts to NSTEMI due to thrombolytic therapy, it is still coded as STEMI.
- For encounters after the 4-week time frame where the patient is still receiving care, assign appropriate aftercare codes rather than category I21.

**Requirements / Properties:**
- ICD-10-CM codes must identify the site of the infarction, such as anterolateral wall or true posterior wall.
- The myocardial infarction must meet the definition for 'other diagnoses' to be reported during the acute phase.
- Codes from category I21 may continue to be reported for encounters occurring while the myocardial infarction is equal to, or less than, four weeks old.
- For old or healed myocardial infarctions not requiring further care, code I25.2 may be assigned.
- Aftercare codes must be assigned for encounters after the 4-week time frame if the patient is still receiving care related to the myocardial infarction.
