# Presymptomatic Type 1 Diabetes Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Presymptomatic Type 1 Diabetes Coding

**Key Behaviors:**
- Codes E10.A- are assigned for early-stage type 1 diabetes that predates the onset of symptoms.

**Requirements / Properties:**
- Clinical documentation must confirm the diagnosis is early-stage type 1 diabetes.
