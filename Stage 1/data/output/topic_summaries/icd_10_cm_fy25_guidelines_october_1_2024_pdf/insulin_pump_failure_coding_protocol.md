# Insulin Pump Failure Coding Protocol
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Insulin Pump Failure Coding Protocol

**Key Behaviors:**
- Assign T85.6 code specifying pump malfunction as principal/first-listed for underdose due to pump failure
- Follow T85.6 with T38.3X6- for underdosing of insulin and oral hypoglycemic drugs
- Assign T85.6 code as principal/first-listed for overdose due to pump malfunction
- Follow T85.6 with T38.3X1- for poisoning by insulin and oral hypoglycemic drugs, accidental (unintentional)
- Assign additional codes for type of diabetes mellitus and associated complications due to underdosing

**Mandatory Coding Criteria:**
- Documentation must specify the type of pump malfunction
- Documentation must indicate underdose or overdose resulting from the pump failure
- Documentation must specify the type of diabetes mellitus
- Documentation must identify associated complications due to underdosing
- Documentation must confirm accidental (unintentional) nature for overdose poisoning codes
