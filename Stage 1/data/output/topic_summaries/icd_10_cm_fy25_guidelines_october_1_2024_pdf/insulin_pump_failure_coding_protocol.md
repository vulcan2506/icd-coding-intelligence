# Insulin Pump Failure Coding Protocol
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Insulin Pump Failure Coding Protocol

**Key Behaviors:**
- Assign code from subcategory T85.6 as the principal or first-listed code for insulin pump failure underdosing, followed by code T38.3X6-.
- Assign code from subcategory T85.6 as the principal or first-listed code for insulin pump failure overdose, followed by code T38.3X1-.
- Additional codes must be assigned for the type of diabetes mellitus and any associated complications resulting from the underdosing or overdose.

**Requirements / Properties:**
- Documentation must specify the type of pump malfunction to select the correct code from subcategory T85.6.
- The encounter must be due to an insulin pump failure resulting in either an underdose or an overdose of insulin.
- Codes for diabetes mellitus and associated complications must be included if applicable to the underdosing or overdose event.
