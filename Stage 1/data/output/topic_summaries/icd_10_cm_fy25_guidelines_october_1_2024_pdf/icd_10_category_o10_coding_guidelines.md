# ICD-10 Category O10 Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Category O10 and P36 Coding Guidelines

**Key Behaviors:**
- Category O10 includes codes for hypertensive heart and hypertensive chronic kidney disease.
- When assigning an O10 code with heart or kidney disease, a secondary code from the hypertension category must be added to specify the type of failure or disease.
- Category P36 includes congenital sepsis.
- If sepsis is documented without specifying congenital or community-acquired, the default is congenital (Category P36).
- If the P36 code does not include the causal organism, an additional code from category B96 must be assigned.
- If the P36 code includes the causal organism, an additional code from category B95 or B96 should not be assigned.
- The default assignment for unspecified sepsis origin in perinates is congenital sepsis (Category P36).

**Mandatory Coding Criteria:**
- Secondary code from the appropriate hypertension category is required when O10 includes hypertensive heart disease or hypertensive chronic kidney disease.
- Additional code from category B96 is required if the P36 code does not include the causal organism.
- Additional codes for severe sepsis (R65.2-) and acute organ dysfunction should be used if applicable.
- A secondary code from the appropriate hypertension category is required when assigning an O10 code that includes hypertensive heart disease or hypertensive chronic kidney disease.
