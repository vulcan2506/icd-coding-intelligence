# ICD-10 Category O10 Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Category O10 and P36 Coding Guidelines

**Key Behaviors:**
- Category O10 includes codes for hypertensive heart and hypertensive chronic kidney disease complicating pregnancy, childbirth, and the puerperium.
- When assigning an O10 code that includes hypertensive heart disease or hypertensive chronic kidney disease, a secondary code from the appropriate hypertension category must be added to specify the type of heart failure or chronic kidney disease.
- Category P36 includes congenital sepsis; if a perinate is documented as having sepsis without documentation of congenital or community acquired status, the default is congenital and a code from category P36 should be assigned.
- If the P36 code includes the causal organism, an additional code from category B95 or B96 should not be assigned.
- If the P36 code does not include the causal organism, an additional code from category B96 should be assigned, and additional codes may be used to identify severe sepsis and associated acute organ dysfunction.

**Requirements / Properties:**
- Documentation must specify the type of heart failure or chronic kidney disease when using O10 codes that include these conditions.
- Documentation must clarify whether sepsis is congenital or community acquired; if unspecified, it defaults to congenital.
- If the P36 code includes the causal organism, no additional code from category B95 or B96 is permitted.
- If the P36 code does not include the causal organism, an additional code from category B96 is mandatory.
- Severe sepsis (R65.2-) and associated acute organ dysfunction must be identified with additional codes if applicable.

**Deprecated in this version:**
- Assigning an additional code from category B95 or B96 when the P36 code already includes the causal organism.
- Failing to assign a secondary code from the appropriate hypertension category when using an O10 code that includes hypertensive heart disease or hypertensive chronic kidney disease.

**New in this version:**
- The inclusion of hypertensive heart and hypertensive chronic kidney disease within Category O10.
- The default assignment of congenital sepsis for perinates with sepsis without laterality or acquisition site documentation.
- The requirement to use additional codes to identify severe sepsis (R65.2-) and any associated acute organ dysfunction.
