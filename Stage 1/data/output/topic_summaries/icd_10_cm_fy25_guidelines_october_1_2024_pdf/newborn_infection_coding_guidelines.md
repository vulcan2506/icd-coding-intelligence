# Newborn Infection Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Newborn Infection Coding Guidelines

**Key Behaviors:**
- If a newborn has a condition that may be either due to the birth process or community acquired and the documentation does not indicate which it is, the default is due to the birth process and the code from Chapter 16 should be used.
- If the condition is community-acquired, a code from Chapter 16 should not be assigned.
- For COVID-19 infection in a newborn, refer to guideline I.C.16.h.

**Requirements / Properties:**
- Documentation must indicate whether the infection is due to the birth process or community-acquired.
- If documentation is unclear regarding the source, Chapter 16 codes must be used by default.

**New in this version:**
- Specific reference to guideline I.C.16.h for COVID-19 infection in newborns.
