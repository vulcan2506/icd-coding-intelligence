# ICD-10 Poisoning Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Poisoning Coding Guidelines

**Key Behaviors:**
- First assign the appropriate code from categories T36-T50 for poisoning or reaction to improper use of medication.
- Assign the associated intent (accidental, intentional self-harm, assault, undetermined) as the 5th or 6th character.
- Code intent as accidental if unknown or unspecified, and only use undetermined if documentation explicitly states intent cannot be determined.
- Use additional code(s) for all manifestations of poisonings.
- Assign abuse or dependence as an additional code if a diagnosis of abuse or dependence of the substance exists.

**Requirements / Properties:**
- Documentation must specify the intent of the poisoning to assign the correct 5th or 6th character.
- If intent is not documented, it must be coded as accidental unless the record explicitly states it cannot be determined.
- All manifestations of the poisoning must be coded with additional codes.
- Abuse or dependence diagnoses must be coded as additional codes when present alongside poisoning.
- Drug interactions (including with alcohol) resulting in toxicity must be classified as poisoning.

**Deprecated in this version:**
- Coding poisoning as a disease process rather than an external cause.
- Assigning unspecified intent when documentation allows for determination of accidental, intentional, or undetermined.
- Failing to code manifestations of poisoning as additional codes.
- Omitting additional codes for abuse or dependence when present.
- Classifying drug interactions or nonprescribed drug combinations as non-poisoning conditions.

**New in this version:**
- Explicit requirement to code intent as accidental if unspecified or unknown.
- Mandatory use of undetermined intent only when documentation explicitly states intent cannot be determined.
- Requirement to code all manifestations of poisoning as additional codes.
- Inclusion of abuse or dependence as additional codes when present.
- Classification of drug-alcohol interactions and nonprescribed drug combinations as poisoning.
