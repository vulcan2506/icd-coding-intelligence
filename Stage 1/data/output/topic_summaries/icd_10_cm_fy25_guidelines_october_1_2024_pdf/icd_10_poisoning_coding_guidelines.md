# ICD-10 Poisoning Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Poisoning Coding Guidelines

**Key Behaviors:**
- Assign appropriate code from categories T36-T50 for poisoning or reaction to improper use of medication.
- Code intent as accidental if unknown or unspecified, and only use undetermined intent if documentation explicitly states it cannot be determined.
- Use additional codes for all manifestations of poisonings.
- Assign abuse or dependence as an additional code if a diagnosis of abuse or dependence of the substance exists.
- Classify drug interactions (including with alcohol) and prescription errors resulting in toxicity as poisoning.
- Classify errors made in drug prescription or administration by any provider, nurse, patient, or other person as poisoning.
- Classify reactions resulting from the interaction of a drug and alcohol as poisoning.

**Mandatory Coding Criteria:**
- Documentation must specify the intent (accidental, intentional self-harm, assault, or undetermined) or default to accidental.
- All manifestations of the poisoning must be coded using additional codes.
- Abuse or dependence diagnoses must be documented to be assigned as additional codes.
- Reaction must result from improper use, overdose, wrong substance, wrong route, or interaction.
- Insulin pump malfunction poisonings must follow Section I.C.4 guidelines.
