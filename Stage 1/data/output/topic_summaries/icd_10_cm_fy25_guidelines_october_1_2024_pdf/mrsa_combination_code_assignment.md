# MRSA Combination Code Assignment
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** MRSA Combination Code Assignment

**Key Behaviors:**
- Assign the appropriate combination code for the condition (e.g., sepsis, pneumonia) that includes the causal organism.
- Do not assign code B95.62 (Methicillin resistant Staphylococcus aureus infection) as an additional code because the combination code includes the organism.
- Do not assign a code from subcategory Z16.11 (Resistance to penicillins) as an additional diagnosis.

**Mandatory Coding Criteria:**
- The infection must be documented as due to methicillin resistant Staphylococcus aureus (MRSA).
- The condition must have a combination code that includes the causal organism.
- Follow Section C.1. for instructions on coding and sequencing of sepsis and severe sepsis.
