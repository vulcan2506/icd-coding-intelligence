# Sepsis Coding Sequencing Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Sepsis Coding Sequencing Rules

**Key Behaviors:**
- Underlying systemic infection sequenced first if severe sepsis is present on admission and meets principal diagnosis definition
- Code from subcategory R65.2 assigned as secondary diagnosis when severe sepsis develops during an encounter
- Infection following a procedure code sequenced first if known for sepsis following a postprocedural wound
- Noninfectious condition code sequenced first if it meets principal diagnosis definition and leads to infection resulting in sepsis
- Only one code from category R65 assigned, specifically R65.2 for severe sepsis cases
- For infections following infusion, transfusion, therapeutic injection, or immunization, code the specific infection category (T80.2 or T88.0-) first before the sepsis code
- When both a noninfectious condition and the resulting infection meet the definition of principal diagnosis, either may be assigned as the principal diagnosis

**Requirements / Properties:**
- Severe sepsis must meet the definition of principal diagnosis to be sequenced first with underlying infection
- Provider must be queried if documentation is unclear whether severe sepsis was present on admission
- Code from subcategory R65.2 cannot be assigned as a principal diagnosis
- Additional codes must be assigned for any acute organ dysfunction associated with severe sepsis
- Infectious agent must be identified with an additional code when applicable
- Use an additional code to identify the infectious agent when applicable for sepsis cases
- Assign additional codes for any acute organ dysfunction associated with severe sepsis

**Deprecated in this version:**
- Assigning code from subcategory R65.1, Systemic inflammatory response syndrome (SIRS) of non-infectious origin, when noninfectious condition leads to infection resulting in severe sepsis
- Assigning code from subcategory R65.2 as a principal diagnosis
- Failing to sequence infection following a procedure code first when sepsis follows a postprocedural wound infection

**New in this version:**
- Explicit rule that code from subcategory R65.2 can never be assigned as a principal diagnosis
- Mandatory sequencing of infection following a procedure code before sepsis code when site of infection is known
- Requirement to assign additional codes for acute organ dysfunction alongside severe sepsis codes
- Clarification that only one code from category R65 should be assigned regardless of etiology
- Guidance to query provider when documentation is unclear regarding presence of severe sepsis on admission
