# ICD-10-CM Diagnosis Code Reporting
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Diagnosis Code Reporting

**Key Behaviors:**
- Each unique ICD-10-CM diagnosis code may be reported only once for an encounter.
- This restriction applies to bilateral conditions when there are no distinct codes identifying laterality.
- This restriction applies when two different conditions are classified to the same ICD-10-CM diagnosis code.
- Documentation should describe the patient's condition using specific diagnoses as well as symptoms, problems, or reasons for the encounter.
- There are ICD-10-CM codes available to describe all documented conditions including symptoms and problems.

**Requirements / Properties:**
- A unique ICD-10-CM diagnosis code must not be reported more than once per encounter.
- Documentation must include specific diagnoses, symptoms, problems, or reasons for the encounter to ensure accurate reporting.
