# ICD-10-CM Diagnosis Code Reporting
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Diagnosis Code Reporting

**Key Behaviors:**
- Each unique ICD-10-CM diagnosis code may be reported only once for an encounter.
- This restriction applies to bilateral conditions when there are no distinct codes identifying laterality.
- This restriction applies when two different conditions are classified to the same ICD-10-CM diagnosis code.
- Documentation should describe the patient's condition using terminology that includes specific diagnoses as well as symptoms, problems, or reasons for the encounter.
- ICD-10-CM codes are available to describe all documented conditions including symptoms and problems.

**Mandatory Coding Criteria:**
- Ensure no duplicate reporting of the same unique ICD-10-CM diagnosis code within a single encounter.
- Verify whether distinct codes exist for laterality before applying the single-code rule to bilateral conditions.
- Confirm if multiple conditions map to the same ICD-10-CM code before reporting them separately.
- Documentation must include specific diagnoses alongside symptoms, problems, or reasons for the encounter.
- Select appropriate ICD-10-CM codes that cover the full scope of documented patient conditions.
