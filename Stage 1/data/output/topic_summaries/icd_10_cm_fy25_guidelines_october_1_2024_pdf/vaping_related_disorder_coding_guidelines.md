# Vaping-Related Disorder Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Vaping-Related Disorder Coding Guidelines

**Key Behaviors:**
- Assign code U07.0 as the principal diagnosis for patients presenting with conditions related to vaping.
- Assign only code U07.0 for lung injury due to vaping without additional codes for the injury itself.
- Code additional manifestations separately, such as acute respiratory failure (J96.0-) or pneumonitis (J68.0).
- Do not code associated respiratory signs and symptoms (e.g., cough, shortness of breath) separately when a definitive diagnosis is established.
- Code gastrointestinal symptoms (e.g., diarrhea, abdominal pain) separately when present.

**Requirements / Properties:**
- A definitive diagnosis must be established before coding associated respiratory signs and symptoms as separate conditions.
- Code U07.0 must be used as the principal diagnosis for any condition related to vaping.
- Specific manifestation codes (e.g., J96.0-, J68.0) are required only for conditions other than the primary vaping-related disorder or lung injury.

**Deprecated in this version:**
- Coding associated respiratory signs and symptoms separately when a definitive vaping-related diagnosis is established.
- Assigning multiple codes for lung injury due to vaping beyond code U07.0.

**New in this version:**
- Introduction of code U07.0 specifically for Vaping-related disorder.
- Mandatory assignment of U07.0 as the principal diagnosis for vaping-related conditions.
- Explicit guidance to exclude separate coding for respiratory symptoms when a definitive diagnosis exists.
