# Vaping-Related Disorder Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Vaping-Related Disorder Coding Guidelines

**Key Behaviors:**
- Assign code U07.0 as the principal diagnosis for patients presenting with vaping-related conditions.
- Assign only code U07.0 for lung injury due to vaping without additional codes.
- Code additional manifestations separately, such as acute respiratory failure (J96.0-) or pneumonitis (J68.0).
- Do not code associated respiratory signs and symptoms (e.g., cough, shortness of breath) separately when a definitive diagnosis is established.
- Code gastrointestinal symptoms (e.g., diarrhea, abdominal pain) separately.

**Mandatory Coding Criteria:**
- A definitive diagnosis must be established before coding associated respiratory signs and symptoms is prohibited.
- Code U07.0 must be assigned as the principal diagnosis for vaping-related conditions.
- Specific manifestation codes (e.g., J96.0-, J68.0) are required for conditions other than lung injury.
