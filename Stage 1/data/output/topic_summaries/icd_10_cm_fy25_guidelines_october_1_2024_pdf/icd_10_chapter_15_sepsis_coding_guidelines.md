# ICD-10 Chapter 15 Sepsis Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Sepsis Coding for Pregnancy and Musculoskeletal Conditions

**Key Behaviors:**
- Assign Chapter 15 sepsis codes for sepsis complicating abortion, pregnancy, childbirth, and puerperium
- Assign Chapter 19 codes for current acute injuries rather than Chapter 13 for recurrent conditions
- Assign Chapter 13 codes for chronic or recurrent bone, joint, or muscle conditions
- Query the provider if documentation is insufficient to determine the best code for a condition
- Assign specific infection codes as additional diagnoses when assigning Chapter 15 sepsis codes

**Mandatory Coding Criteria:**
- A code for the specific type of infection must be assigned as an additional diagnosis when assigning a Chapter 15 sepsis code
- If severe sepsis is present, a code from subcategory R65.2 (Severe sepsis) must be assigned
- Codes for associated organ dysfunction(s) must be assigned as additional diagnoses if severe sepsis is present
- Current acute injuries must be coded to the appropriate injury code from Chapter 19
- Chronic or recurrent conditions should generally be coded with a code from Chapter 13
