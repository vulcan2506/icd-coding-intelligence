# ICD-10 Chapter 15 Sepsis Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Chapter 15 Sepsis Coding Guidelines

**Key Behaviors:**
- Assign a code for the specific type of infection as an additional diagnosis when coding sepsis complicating abortion, pregnancy, childbirth, and the puerperium.
- If severe sepsis is present, assign a code from subcategory R65.2, Severe sepsis, as an additional diagnosis.
- Assign codes for associated organ dysfunction(s) as additional diagnoses when severe sepsis is present.
- Query the provider if it is difficult to determine from the documentation which code best describes a condition.
- Code current, acute injuries to the appropriate injury code from chapter 19 rather than chapter 13.

**Requirements / Properties:**
- Documentation must specify the specific type of infection to assign the correct additional diagnosis code.
- Clinical documentation must confirm the presence of severe sepsis to trigger the use of subcategory R65.2.
- Documentation must identify associated organ dysfunction(s) to support the assignment of additional organ dysfunction codes.
- Provider query is mandatory when documentation is insufficient to distinguish between acute injury (chapter 19) and chronic/recurrent conditions (chapter 13).
- Coding of sepsis complicating pregnancy-related conditions requires adherence to Section I.C.15 guidelines.

**Deprecated in this version:**
- Coding musculoskeletal conditions resulting from healed injuries to chapter 19 instead of chapter 13.
- Coding recurrent bone, joint, or muscle conditions to chapter 19 instead of chapter 13.
- Failing to assign a code for the specific type of infection alongside a sepsis code in pregnancy-related complications.
- Omitting codes for associated organ dysfunction(s) when severe sepsis is documented.
- Assigning unspecified codes without querying the provider when documentation is ambiguous.

**New in this version:**
- Mandatory assignment of subcategory R65.2 for severe sepsis cases.
- Requirement to code associated organ dysfunction(s) as additional diagnoses in severe sepsis scenarios.
- Explicit instruction to query providers for ambiguous documentation regarding acute vs. chronic conditions.
