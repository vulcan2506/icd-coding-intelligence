# ICD-10 Coding Guidelines for Post-Abortion Hemorrhage
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Coding Guidelines for Post-Abortion Hemorrhage

**Key Behaviors:**
- Assign code O04.6 for delayed or excessive hemorrhage following induced termination of pregnancy
- Do not assign code O72.1 for post-abortion conditions

**Mandatory Coding Criteria:**
- Condition must be hemorrhage post elective abortion
- Condition must be delayed or excessive hemorrhage following induced termination of pregnancy

**Superseded Guidance:**
- Code O72.1 for post-abortion conditions
