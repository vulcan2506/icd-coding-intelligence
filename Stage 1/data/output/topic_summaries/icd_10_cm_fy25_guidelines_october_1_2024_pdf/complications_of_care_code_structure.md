# Complications of Care Code Structure
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Complications of Care Code Structure

**Key Behaviors:**
- Some complications of care codes include the external cause within the code itself.
- The code specifies both the nature of the complication and the type of procedure that caused it.
- No separate external cause code indicating the type of procedure is necessary for these codes.

**Requirements / Properties:**
- The code must explicitly include the nature of the complication.
- The code must explicitly include the type of procedure that caused the complication.

**Deprecated in this version:**
- Assigning a separate external cause code indicating the type of procedure for codes that already include this information.
