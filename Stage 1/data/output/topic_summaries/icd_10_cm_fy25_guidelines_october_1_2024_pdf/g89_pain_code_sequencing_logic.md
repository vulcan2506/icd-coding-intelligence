# G89 Pain Code Sequencing Logic
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** G89 Pain Code Sequencing Logic

**Key Behaviors:**
- If the encounter is for pain control or pain management, assign the code from category G89 followed by the code identifying the specific site of pain.
- If the encounter is for any other reason except pain control or pain management, and a related definitive diagnosis has not been established, assign the code for the specific site of pain first, followed by the appropriate code from category G89.

**Requirements / Properties:**
- The encounter must be explicitly for pain control or pain management to sequence G89 codes first.
- A related definitive diagnosis must be confirmed by the provider before sequencing site-specific pain codes first.
- The specific site of pain must be identified by a corresponding code (e.g., M54.2) when sequencing G89 codes first.
