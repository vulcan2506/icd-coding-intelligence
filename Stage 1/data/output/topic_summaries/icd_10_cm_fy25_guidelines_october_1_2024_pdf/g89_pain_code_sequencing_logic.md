# G89 Pain Code Sequencing Logic
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** G89 Pain Code Sequencing Logic

**Key Behaviors:**
- Assign G89 code followed by site-specific pain code if the encounter is for pain control or pain management.
- Assign site-specific pain code first followed by G89 code if the encounter is for any other reason and a definitive diagnosis has not been established.

**Mandatory Coding Criteria:**
- Encounter must be explicitly for pain control or pain management to sequence G89 first.
- A related definitive diagnosis must be confirmed by the provider to sequence site-specific pain code first.
- If no definitive diagnosis is established, the site-specific pain code takes precedence over the G89 code.
