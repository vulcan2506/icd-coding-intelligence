# COVID-19 Nonrespiratory Manifestation Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** COVID-19 Nonrespiratory Manifestation Coding

**Key Behaviors:**
- Assign code U07.1, COVID-19, as the principal/first-listed diagnosis when the reason for the encounter is a nonrespiratory manifestation.
- Assign code(s) for the specific manifestation(s) as additional diagnoses.

**Requirements / Properties:**
- The reason for the encounter or admission must be a nonrespiratory manifestation of COVID-19 (e.g., viral enteritis).
- Code U07.1 must be used as the principal diagnosis in these scenarios.

**New in this version:**
- Mandatory assignment of U07.1 as the principal diagnosis for nonrespiratory manifestations.
