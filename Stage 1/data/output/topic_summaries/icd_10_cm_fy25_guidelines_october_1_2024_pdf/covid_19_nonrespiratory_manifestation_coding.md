# COVID-19 Nonrespiratory Manifestation Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** COVID-19 Nonrespiratory Manifestation Coding

**Key Behaviors:**
- Assign code U07.1 as the principal/first-listed diagnosis when the encounter is for a nonrespiratory manifestation of COVID-19
- Assign specific codes for the nonrespiratory manifestation(s) as additional diagnoses

**Mandatory Coding Criteria:**
- The reason for the encounter or admission must be a nonrespiratory manifestation of COVID-19 (e.g., viral enteritis)
- Code U07.1 must be assigned as the principal/first-listed diagnosis
