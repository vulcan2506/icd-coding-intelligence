# Drug and Chemical Poisoning Coding Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Drug and Chemical Poisoning Coding Rules

**Key Behaviors:**
- Do not code directly from the Table of Drugs and Chemicals; always refer back to the Tabular List.
- Code each drug, medicinal, or biological substance individually if two or more are taken, unless a combination code is listed.
- Assign code from subcategory T50.91 for multiple unspecified drugs, medicaments, or biological substances taken.

**Mandatory Coding Criteria:**
- Verification against the Tabular List is mandatory before assigning any code.
- Individual coding is required for multiple substances unless a specific combination code exists.
- Use subcategory T50.91 specifically when the substances are unspecified.
