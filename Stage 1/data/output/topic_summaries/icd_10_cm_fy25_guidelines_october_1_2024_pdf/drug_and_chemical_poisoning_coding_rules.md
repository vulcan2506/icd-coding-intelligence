# Drug and Chemical Poisoning Coding Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Drug and Chemical Poisoning Coding Rules

**Key Behaviors:**
- Do not code directly from the Table of Drugs and Chemicals; always refer back to the Tabular List.
- Code each drug, medicinal, or biological substance individually if two or more are taken, unless a combination code is listed.
- Assign code from subcategory T50.91 for multiple unspecified drugs, medicaments, or biological substances taken.

**Requirements / Properties:**
- Verification that the substance is listed in the Tabular List rather than the Table of Drugs and Chemicals.
- Confirmation of whether a specific combination code exists before assigning individual codes.
- Documentation must specify the number of unspecified substances to justify use of subcategory T50.91.

**Deprecated in this version:**
- Coding directly from the Table of Drugs and Chemicals without referencing the Tabular List.

**New in this version:**
- Mandatory use of subcategory T50.91 for multiple unspecified drugs, medicaments, and biological substances.
