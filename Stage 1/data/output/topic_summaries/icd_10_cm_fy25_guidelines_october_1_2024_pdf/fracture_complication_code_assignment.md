# Fracture Complication Code Assignment
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Fracture Complication Code Assignment

**Key Behaviors:**
- 7th character A is assigned when the patient is receiving active treatment for the fracture, regardless of provider continuity.
- 7th character D is assigned for encounters after active treatment is completed during the healing or recovery phase.
- Other 7th characters are used for subsequent encounters treating problems associated with healing, such as malunions, nonunions, and sequelae.
- Complications of surgical treatment during the healing or recovery phase must be coded with appropriate complication codes.
- Assignment of the 7th character is based on active treatment status rather than whether the provider is seeing the patient for the first time.

**Mandatory Coding Criteria:**
- Active treatment must be ongoing for 7th character A assignment.
- Active treatment must be completed for 7th character D assignment.
- Problems must be associated with the healing process for other 7th character assignments.
- Complications must be specifically related to surgical treatment for fracture repairs.
- Coding must follow Section I.C.19 guidelines for traumatic fractures.
