# COVID-19 Acute Respiratory Failure Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** COVID-19 Acute Respiratory Failure Coding

**Key Behaviors:**
- Assign code U07.1 as the principal/first-listed diagnosis when the encounter reason is a respiratory manifestation of COVID-19.
- Assign code(s) for the specific respiratory manifestation(s) as additional diagnoses.
- Assign code J96.0- for acute respiratory failure due to COVID-19 alongside U07.1.

**Mandatory Coding Criteria:**
- The reason for the encounter or admission must be a respiratory manifestation of COVID-19.
- Code U07.1 must be assigned as the principal diagnosis for respiratory manifestations of COVID-19.
- Code J96.0- must be used specifically for acute respiratory failure due to COVID-19.
