# COVID-19 Acute Respiratory Failure Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** COVID-19 Acute Respiratory Failure Coding

**Key Behaviors:**
- Assign code U07.1 as the principal/first-listed diagnosis when the encounter reason is a respiratory manifestation of COVID-19.
- Assign code(s) for the specific respiratory manifestation(s) as additional diagnoses alongside U07.1.
- For acute respiratory failure due to COVID-19, assign code U07.1 in conjunction with code J96.0-.

**Requirements / Properties:**
- The reason for the encounter or admission must be explicitly a respiratory manifestation of COVID-19.
- Code U07.1 must be used as the principal diagnosis for COVID-19 related respiratory encounters.
- Code J96.0- must be assigned to specify acute respiratory failure when present with COVID-19.

**New in this version:**
- Use of code U07.1 as the principal diagnosis for COVID-19 respiratory manifestations.
- Mandatory pairing of U07.1 with J96.0- for acute respiratory failure cases.
