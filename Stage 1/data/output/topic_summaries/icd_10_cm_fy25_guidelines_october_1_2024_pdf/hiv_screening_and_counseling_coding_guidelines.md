# HIV Screening and Counseling Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** HIV Screening and Counseling Coding Guidelines

**Key Behaviors:**
- Assign code Z11.4 for encounters where the patient is being seen to determine HIV status
- Use additional codes for any associated high-risk behavior if applicable during screening
- Code signs and symptoms if a patient with signs or symptoms is being seen for HIV testing
- Assign code Z71.7 for encounters where a patient returns to be informed of negative HIV test results
- Apply previous guidelines and assign appropriate codes if the HIV test result is positive

**Mandatory Coding Criteria:**
- Documentation must confirm the purpose of the encounter (screening vs. symptomatic testing)
- Documentation must specify associated high-risk behaviors to support additional coding
- Documentation must confirm that counseling was provided to justify the use of code Z71.7
