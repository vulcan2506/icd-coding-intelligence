# HIV Screening and Counseling Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** HIV Screening and Counseling Coding Guidelines

**Key Behaviors:**
- Assign code Z11.4 for encounters where the patient is being seen to determine HIV status
- Use additional codes for any associated high-risk behavior if applicable during screening
- Code signs and symptoms if a patient with symptoms is being seen for HIV testing
- Assign code Z71.7 for encounters where a patient returns to be informed of negative HIV test results
- Apply previous guidelines and assign appropriate codes if HIV test results are positive

**New in this version:**
- Code Z11.4 for Encounter for screening for human immunodeficiency virus [HIV]
- Code Z71.7 for Human immunodeficiency virus [HIV] counseling
