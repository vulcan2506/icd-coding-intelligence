# ICD-10-CM Coding Syntax Definitions
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Coding Syntax Definitions

**Key Behaviors:**
- Brackets enclose synonyms, alternative wording, or explanatory phrases in the Tabular List and identify manifestation codes in the Alphabetic Index.
- Parentheses enclose nonessential modifiers that may be present or absent without affecting the code number.
- Nonessential modifiers in the Alphabetic Index apply to subterms following a main term unless mutually exclusive with a subentry, in which case the subentry takes precedence.
- Colons in the Tabular List follow incomplete terms requiring one or more modifiers to make them assignable to a category.
- Subentries take precedence over nonessential modifiers when the two are mutually exclusive.

**Requirements / Properties:**
- Brackets must be used to identify manifestation codes in the Alphabetic Index.
- Nonessential modifiers must not affect the code number assignment.
- Incomplete terms in the Tabular List must be completed with modifiers following a colon before assignment.
- Subentries must take precedence over nonessential modifiers when mutually exclusive.
- Parentheses must enclose supplementary words that are optional for code assignment.
