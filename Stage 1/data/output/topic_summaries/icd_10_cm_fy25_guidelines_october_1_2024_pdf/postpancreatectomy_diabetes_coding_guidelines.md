# Postpancreatectomy Diabetes Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Postpancreatectomy Diabetes Coding

**Key Behaviors:**
- Assign code E89.1 for postpancreatectomy diabetes mellitus (lack of insulin due to surgical removal of all or part of the pancreas).
- Assign a code from category E13 as the principal or first-listed diagnosis.
- Assign a code from subcategory Z90.41, Acquired absence of pancreas, as an additional code.

**Mandatory Coding Criteria:**
- Diagnosis must be documented as postpancreatectomy diabetes mellitus resulting from surgical removal of all or part of the pancreas.
- Code E89.1 must be assigned for the secondary diabetes condition.
- Code from category E13 must be the principal or first-listed diagnosis.
- Code from subcategory Z90.41 must be included as an additional code.
