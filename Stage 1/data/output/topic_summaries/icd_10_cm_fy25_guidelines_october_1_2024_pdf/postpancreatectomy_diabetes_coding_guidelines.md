# Postpancreatectomy Diabetes Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Postpancreatectomy Diabetes Coding Guidelines

**Key Behaviors:**
- Assign code E89.1 for postpancreatectomy diabetes mellitus due to lack of insulin from surgical removal of the pancreas.
- Assign a code from category E13 as the principal or first-listed diagnosis.
- Assign a code from subcategory Z90.41, Acquired absence of pancreas, as an additional code.

**Requirements / Properties:**
- Clinical documentation must confirm the diabetes is secondary to pancreatectomy (surgical removal of all or part of the pancreas).
- Documentation must specify the lack of insulin as the cause of the diabetes.
- Sequencing must follow Tabular List instructions for categories E08, E09, and E13.
