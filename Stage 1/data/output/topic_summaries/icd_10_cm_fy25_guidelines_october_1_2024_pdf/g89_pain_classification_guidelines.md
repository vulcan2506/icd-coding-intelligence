# G89 Pain Classification Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** G89 Pain Classification and Exclusion Rules

**Key Behaviors:**
- Assign code F45.41 for pain exclusively related to psychological disorders, excluding category G89 codes.
- Use code F45.42 with a category G89 code when documentation indicates a psychological component for acute or chronic pain.
- Do not assign category G89 codes if pain is unspecified, post-thoracotomy, postprocedural, or neoplasm-related without further specification.
- Avoid assigning category G89 codes if the underlying definitive diagnosis is known, unless the encounter is solely for pain control.
- Assign category G89 codes alongside site-specific pain codes if the G89 code provides additional information about pain acuity or chronicity.

**Requirements / Properties:**
- Documentation must explicitly state that pain is exclusively related to psychological disorders to use F45.41.
- Documentation must confirm a psychological component exists to use F45.42 in conjunction with G89 codes.
- The encounter reason must be pain control/management, not underlying condition management, to justify G89 assignment when a definitive diagnosis is known.
- Pain must be specified as acute or chronic, post-thoracotomy, postprocedural, or neoplasm-related to allow G89 assignment.
- Site-specific pain codes must be used in conjunction with G89 codes if the G89 code adds detail about pain duration or nature.

**Deprecated in this version:**
- Assigning category G89 codes when the underlying definitive diagnosis is known and the encounter is for that condition management.
- Assigning category G89 codes for unspecified, post-thoracotomy, postprocedural, or neoplasm-related pain without specific documentation.
- Assigning category G89 codes in conjunction with F45.41 for pain related to psychological disorders.
- Using category G89 codes as the principal diagnosis when an admission is for a procedure treating the underlying condition.
- Failing to assign site-specific pain codes when the G89 code does not fully describe the site of pain.

**New in this version:**
- Explicit instruction to use F45.42 with G89 codes for pain with psychological components.
- Clarification that G89 codes may be used with site-specific pain codes if they provide additional information.
- Specific exclusion note under category G89 prohibiting assignment with F45.41.
- Guidance on principal diagnosis assignment for procedures treating underlying conditions versus pain management.
- Requirement to query or document specific pain characteristics (acute/chronic, etc.) before assigning G89 codes.
