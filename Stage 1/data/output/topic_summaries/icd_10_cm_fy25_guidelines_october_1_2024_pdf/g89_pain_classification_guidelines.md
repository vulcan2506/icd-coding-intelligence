# G89 Pain Classification Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** G89 Pain Classification and Exclusion Rules

**Key Behaviors:**
- Do not assign category G89 codes if the pain is exclusively related to psychological disorders (use F45.41 instead).
- Assign code F45.42 with a G89 code when there is documentation of a psychological component for acute or chronic pain.
- Do not assign G89 codes if the pain is not specified as acute, chronic, post-thoracotomy, postprocedural, or neoplasm-related.
- Do not assign G89 codes if the underlying definitive diagnosis is known, unless the encounter is specifically for pain control/management.
- Assign the underlying condition as the principal diagnosis for procedures aimed at treating it, excluding G89 codes.

**Mandatory Coding Criteria:**
- Documentation must indicate the pain is acute or chronic, post-thoracotomy, postprocedural, or neoplasm-related to qualify for G89 codes.
- The underlying definitive diagnosis must be unknown or the encounter must be solely for pain control/management to assign G89 codes.
- G89 codes may be used with site-specific pain codes if the G89 code provides additional information about the nature of the pain.
- Code F45.41 must be used exclusively for pain related to psychological disorders, precluding G89 assignment.
- Code F45.42 must be used in conjunction with a G89 code when a psychological component is documented alongside acute or chronic pain.
- Codes from category G89 may be used in conjunction with codes from other categories and chapters to provide more detail about acute or chronic pain and neoplasm-related pain, unless otherwise indicated.
- Codes from category G89 may be used in conjunction with codes that identify the site of pain (including codes from chapter 18) if the category G89 code provides additional information about the nature of the pain.
