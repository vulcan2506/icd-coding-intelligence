# Code O80 Assignment Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Code O80 Assignment Guidelines

**Key Behaviors:**
- Assign when patient is admitted for full-term normal delivery with single healthy infant and no complications
- Always assign as principal diagnosis
- Do not use if any other chapter 15 code is needed to describe a current complication
- May use additional codes from other chapters if not related to or complicating the pregnancy

**Mandatory Coding Criteria:**
- Full-term delivery
- Normal delivery
- Single infant
- Healthy infant
- No complications antepartum, during delivery, or postpartum
