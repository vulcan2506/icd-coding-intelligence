# Code O80 Assignment Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Code O80 Assignment Guidelines

**Key Behaviors:**
- Assign Code O80 when a patient is admitted for a full-term normal delivery and delivers a single, healthy infant without any complications.
- Code O80 is always a principal diagnosis.
- Do not use Code O80 if any other code from chapter 15 is needed to describe a current complication of the antenatal, delivery, or postnatal period.
- Additional codes from other chapters may be used with Code O80 if they are not related to or are in any way complicating the pregnancy.

**Requirements / Properties:**
- Patient must be admitted for a full-term normal delivery.
- Delivery must result in a single, healthy infant.
- No complications must be present antepartum, during the delivery, or postpartum during the delivery episode.
- No other Chapter 15 codes must be required to describe a current complication of the antenatal, delivery, or postnatal period.
