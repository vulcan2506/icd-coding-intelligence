# Non-Pressure Ulcer Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Non-Pressure Ulcer Coding Guidelines

**Key Behaviors:**
- Assign unstageable pressure ulcer code (L89.--0) only when stage cannot be clinically determined due to eschar or grafts; use unspecified stage (L89.--9) if stage is not documented.
- If the stage of an unstageable pressure ulcer is revealed after debridement, assign only the code for the stage revealed following debridement.
- For ulcers present on admission but healed at discharge, assign the code for the site and stage/severity at the time of admission.
- If documentation is unclear regarding whether a non-pressure ulcer is current or healing, query the provider.
- No code is assigned if documentation states a non-pressure ulcer is completely healed at the time of admission.

**Requirements / Properties:**
- Clinical documentation must explicitly state the stage or severity of the pressure ulcer to assign a specific code.
- Documentation must clarify whether the ulcer is current (new) or healing to avoid misclassification.
- For unstageable ulcers, documentation must confirm the presence of eschar or grafts preventing stage determination.
- If a pressure ulcer progresses to a higher stage during an inpatient stay, separate codes must be assigned for admission stage and highest reported stage.
- If documentation lacks stage or severity information for healing ulcers, assign the unspecified stage or severity code.

**Deprecated in this version:**
- Assigning unspecified stage codes (L89.--9) when the ulcer is clinically unstageable due to eschar or grafts.
- Assigning codes for non-pressure ulcers that are completely healed at the time of admission.
- Failing to query the provider when documentation is unclear about whether a non-pressure ulcer is current or healing.
- Assigning a single code for a pressure ulcer that progresses to a higher stage during an inpatient stay without coding both stages.
- Using unspecified stage codes when the ulcer is documented as unstageable rather than using the specific unstageable code.

**New in this version:**
- Specific instruction to assign only the code for the stage revealed after debridement for previously unstageable ulcers.
- Mandatory assignment of admission stage/severity codes for ulcers that heal during the encounter.
- Requirement to assign unspecified severity codes for healing non-pressure ulcers when severity is not documented.
- Explicit rule prohibiting code assignment for non-pressure ulcers completely healed at admission.
- Clarification that unstageable codes (L89.--0) are distinct from unspecified stage codes (L89.--9).
