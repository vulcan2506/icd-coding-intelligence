# Non-Pressure Ulcer Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Non-Pressure Ulcer Coding Guidelines

**Key Behaviors:**
- Assign code for unspecified stage (L89.--9) when there is no documentation regarding the stage of the pressure ulcer.
- If the stage of an unstageable pressure ulcer is revealed after debridement, assign only the code for the stage revealed following debridement.
- Assign the appropriate pressure ulcer stage code based on documentation for ulcers described as healing; if stage is unknown, assign unspecified stage.
- Query the provider if documentation is unclear whether the patient has a current (new) pressure ulcer or is being treated for a healing pressure ulcer.
- For ulcers present on admission but healed at discharge, assign the code for the site and stage of the pressure ulcer at the time of admission.
- Assign two separate codes (admission stage and highest stage) if a pressure ulcer progresses to a higher stage during an inpatient stay.
- Do not assign a code if documentation states the non-pressure ulcer is completely healed at the time of admission.

**Mandatory Coding Criteria:**
- Assignment of unstageable pressure ulcer code (L89.--0) must be based on clinical documentation indicating the stage cannot be determined.
- Unspecified stage code (L89.--9) is required when stage documentation is absent.
- Code for unspecified severity is required if documentation does not provide information about the severity of a healing non-pressure ulcer.
- Provider query is mandatory if it is unclear whether the condition is a current/new ulcer or a healing ulcer.
- Coding for healed-at-discharge ulcers must reflect the site and stage/severity at the time of admission.
