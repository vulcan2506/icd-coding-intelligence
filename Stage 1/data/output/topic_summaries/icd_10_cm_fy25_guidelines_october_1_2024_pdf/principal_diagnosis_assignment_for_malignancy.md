# Principal Diagnosis Assignment for Malignancy
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Principal Diagnosis Assignment for Malignancy

**Key Behaviors:**
- Designate primary malignancy as principal diagnosis if chiefly responsible for admission and treatment is directed at the primary site.
- Assign Z51.-code as principal diagnosis if chemotherapy, immunotherapy, or external beam radiation therapy is chiefly responsible for the encounter.
- Sequence primary site first, followed by any metastatic sites for treatment of primary malignancy.
- Do not use symptoms, signs, or ill-defined conditions associated with an existing malignancy to replace the malignancy as principal diagnosis.
- Use Z51.-code for encounters related to prophylactic organ removal.

**Mandatory Coding Criteria:**
- Malignancy must be chiefly responsible for occasioning the admission/encounter.
- Treatment must be directed at the primary site.
- Provider must document uncertainty if coding suspected/probable conditions.
- Underlying diagnosis must be listed as secondary when Z51.-code is principal.
- Primary site must be sequenced before metastatic sites.
- Underlying diagnosis or problem must be listed as a secondary diagnosis when a Z51.-code is assigned as the principal diagnosis.
