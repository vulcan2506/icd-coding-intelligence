# Perinatal Newborn Code Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Perinatal Newborn Code Guidelines

**Key Behaviors:**
- Refer to Section I.C.16 for further instruction on newborn code usage
- Assign Z76.1 for encounters regarding health supervision and care of foundlings
- Assign Z00.1- for routine child health examinations
- Assign Z38 for liveborn infants based on place of birth and type of delivery

**Mandatory Coding Criteria:**
- Condition must be a foundling for Z76.1
- Condition must be a routine child health examination for Z00.1-
- Condition must be a liveborn infant for Z38
