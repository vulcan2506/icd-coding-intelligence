# Observation Diagnosis Coding Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Observation Diagnosis Coding Rules

**Key Behaviors:**
- Assign a code for the medical condition as the first-listed diagnosis when a patient is admitted for observation.
- Code the reason for the surgery as the first reported diagnosis when a patient presents for outpatient surgery and develops complications requiring admission to observation.
- Code complications as secondary diagnoses when they occur during outpatient surgery requiring observation admission.

**Requirements / Properties:**
- The patient must be admitted specifically for observation status.
- The medical condition must be the primary reason for the observation admission.
- Complications must require admission to observation status to be coded as secondary diagnoses.
