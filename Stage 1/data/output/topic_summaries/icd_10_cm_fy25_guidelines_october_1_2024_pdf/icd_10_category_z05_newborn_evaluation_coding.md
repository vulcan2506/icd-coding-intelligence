# ICD-10 Category Z05 Newborn Evaluation Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Category Z05 Newborn Evaluation Coding

**Key Behaviors:**
- Assign a code from category Z05 to identify instances when a healthy newborn is evaluated for a suspected condition determined after study not to be present.
- Do not use a code from category Z05 when the patient is documented to have signs or symptoms of a suspected problem; in such cases code the sign or symptom.

**Requirements / Properties:**
- The newborn must be documented as healthy after the evaluation.
- The suspected condition must be ruled out after study.
- There must be no documentation of signs or symptoms of the suspected problem.
