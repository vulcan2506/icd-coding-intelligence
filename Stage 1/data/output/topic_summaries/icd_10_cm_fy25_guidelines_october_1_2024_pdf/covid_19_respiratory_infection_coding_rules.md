# COVID-19 Respiratory Infection Coding Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** COVID-19 Respiratory Infection Coding Rules

**Key Behaviors:**
- Assign codes U07.1, COVID-19, and J12.82 for pneumonia confirmed as due to COVID-19.
- Assign codes U07.1 and J22 for COVID-19 associated with lower respiratory infection NOS or acute respiratory infection NOS.
- Assign codes U07.1 and J98.8 for COVID-19 associated with respiratory infection NOS.

**Requirements / Properties:**
- Pneumonia must be confirmed as due to COVID-19 to assign J12.82.
- Documentation must specify the relationship between COVID-19 and the respiratory infection type (e.g., pneumonia, NOS, acute).

**New in this version:**
- Mandatory assignment of code U07.1 for all documented COVID-19 cases.
- Specific code J12.82 introduced for pneumonia due to coronavirus disease 2019.
- Specific code J22 introduced for unspecified acute lower respiratory infection in the context of COVID-19.
