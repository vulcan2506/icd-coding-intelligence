# Z28 Underimmunization Code Mapping
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Z28 Underimmunization Code Mapping

**Key Behaviors:**
- Map Z28 to Z29 for encounters regarding underimmunization status
- Map Z28 to Z40 for encounters regarding prophylactic surgery
- Map Z28 to Z41 for encounters regarding procedures for purposes other than remedying health state

**Requirements / Properties:**
- The encounter must be specifically for underimmunization status (Z28.3) to avoid mapping to Z29
- Documentation must distinguish between underimmunization status and other prophylactic measures to ensure correct code assignment

**Deprecated in this version:**
- Assigning Z28 without verifying the specific nature of the encounter (e.g., underimmunization vs. other prophylactic measures)
- Using unspecified codes when the encounter type can be clearly categorized under Z29, Z40, or Z41

**New in this version:**
- Explicit mapping rules for Z28 to Z29, Z40, and Z41 based on encounter specifics
- Requirement to exclude Z28.3 (Underimmunization status) from general Z28 mappings to ensure accurate coding
