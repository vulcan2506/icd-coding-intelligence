# Z28 Underimmunization Code Mapping
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Z28 Underimmunization Code Mapping

**Key Behaviors:**
- Map Z28 encounters to Z29 for general underimmunization status
- Map Z28 encounters to Z40 for prophylactic surgery contexts
- Map Z28 encounters to Z41 for other prophylactic measures contexts

**Mandatory Coding Criteria:**
- Exclude Z28.3 (Underimmunization status) from mapping to Z29
- Exclude Z28.3 from mapping to Z40
- Exclude Z28.3 from mapping to Z41
