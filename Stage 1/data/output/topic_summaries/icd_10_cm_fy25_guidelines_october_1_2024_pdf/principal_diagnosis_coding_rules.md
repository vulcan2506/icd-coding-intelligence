# Principal Diagnosis Coding Rules
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Principal Diagnosis Coding Rules

**Key Behaviors:**
- Respiratory failure may be listed as a secondary diagnosis if it occurs after admission, or if it is present on admission, but does not meet the definition of principal diagnosis.
- Codes for symptoms, signs, and ill-defined conditions from Chapter 18 are not to be used as principal diagnosis when a related definitive diagnosis has been established.

**Mandatory Coding Criteria:**
- Respiratory failure must meet the definition of principal diagnosis to be listed as such; otherwise, it is secondary.
- A related definitive diagnosis must be established before using a symptom or sign code as a principal diagnosis.
