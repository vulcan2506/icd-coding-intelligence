# O80 Delivery Code Usage for Z37.0
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** O80 Delivery Code Usage for Z37.0

**Key Behaviors:**
- Z37.0 is the only outcome of delivery code appropriate for use with O80
- Other Z37 codes must not be paired with O80

**Requirements / Properties:**
- The claim must document a single live birth outcome
- The delivery code O80 must be present on the claim

**Deprecated in this version:**
- Using Z37.1-Z37.9 with O80
