# O80 Delivery Code Usage for Z37.0
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** O80 Delivery Code Usage for Z37.0

**Key Behaviors:**
- Z37.0 is the only outcome of delivery code appropriate for use with O80.

**Mandatory Coding Criteria:**
- The outcome of delivery must be a single live birth to assign Z37.0 with O80.
