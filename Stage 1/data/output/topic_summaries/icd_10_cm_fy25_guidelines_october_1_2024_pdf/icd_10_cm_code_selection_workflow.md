# ICD-10-CM Code Selection Workflow
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Code Selection Workflow

**Key Behaviors:**
- Locate the diagnosis term in the Alphabetic Index first, then verify the code in the Tabular List.
- Read and follow instructional notations found in both the Alphabetic Index and the Tabular List.
- Use both the Alphabetic Index and Tabular List for all code assignments; the Alphabetic Index does not always provide the full code.
- Select the full code including laterality and applicable 7th characters only from the Tabular List.
- Interpret a dash (-) at the end of an Alphabetic Index entry as an indicator that additional characters are required.

**Mandatory Coding Criteria:**
- Verification of the code in the Tabular List is mandatory after locating the term in the Alphabetic Index.
- Instructional notations in both the Alphabetic Index and Tabular List must be read and followed.
- Full code selection (including laterality and 7th characters) must be completed in the Tabular List.
- A dash (-) in the Alphabetic Index entry necessitates checking for additional required characters.
- Even if a dash is absent in the Alphabetic Index, the Tabular List must be consulted to verify if a 7th character is required.
