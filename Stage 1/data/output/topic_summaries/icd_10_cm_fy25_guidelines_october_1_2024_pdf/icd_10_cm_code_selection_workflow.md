# ICD-10-CM Code Selection Workflow
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Code Selection Workflow

**Key Behaviors:**
- Locate the diagnosis term in the Alphabetic Index first, then verify the code in the Tabular List.
- Read and follow instructional notations appearing in both the Alphabetic Index and the Tabular List.
- Use both the Alphabetic Index and Tabular List to locate and assign the full code, including laterality and 7th characters.
- Recognize that a dash (-) at the end of an Alphabetic Index entry indicates additional characters are required.
- Refer to the Tabular List to verify if a 7th character is required, even if a dash is not present in the Alphabetic Index.

**Requirements / Properties:**
- Full code selection including laterality and applicable 7th characters must be verified in the Tabular List.
- Instructional notations in both the Alphabetic Index and Tabular List must be read and followed.
- A dash (-) in the Alphabetic Index entry is a mandatory indicator that additional characters are required.
