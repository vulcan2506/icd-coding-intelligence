# Severe Sepsis Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Severe Sepsis Coding Guidelines

**Key Behaviors:**
- Assign code for underlying systemic infection first, followed by code from subcategory R65.2 for severe sepsis.
- Do not assign severe sepsis code unless severe sepsis or associated acute organ dysfunction is explicitly documented.
- Query provider if blood cultures are negative/inconclusive but clinical evidence of sepsis exists.
- Query provider if the term 'urosepsis' is used to clarify if it is synonymous with sepsis.
- Query provider if acute organ dysfunction is not clearly associated with the sepsis.

**Mandatory Coding Criteria:**
- Documentation must explicitly state severe sepsis or an associated acute organ dysfunction.
- Minimum of 2 codes required: one for underlying systemic infection and one from subcategory R65.2.
- If causal organism is not documented, assign code A41.9 for the infection.
- Additional codes for associated acute organ dysfunction are required.
- Acute organ dysfunction must be clearly associated with the sepsis to assign severe sepsis code.
