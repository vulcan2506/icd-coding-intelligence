# Severe Sepsis Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Severe Sepsis Coding Guidelines

**Key Behaviors:**
- Assign code A41.9 for unspecified organism if the causal organism is not further specified.
- Do not assign a code from subcategory R65.2 unless severe sepsis or an associated acute organ dysfunction is documented.
- Query the provider if blood cultures are negative or inconclusive but clinical evidence of sepsis exists.
- Query the provider if the term 'urosepsis' is used, as it is nonspecific and has no default code.
- Query the provider if the relationship between acute organ dysfunction and sepsis is unclear.

**Requirements / Properties:**
- A code from subcategory R65.2 requires documentation of severe sepsis or an associated acute organ dysfunction.
- Coding of severe sepsis requires a minimum of 2 codes: one for the underlying systemic infection and one from subcategory R65.2.
- Additional codes for associated acute organ dysfunction are mandatory when present.
- An acute organ dysfunction must be clearly associated with the sepsis to assign the severe sepsis code.
- If the causal organism is not documented, code A41.9 must be assigned for the infection.

**Deprecated in this version:**
- Assigning a code from subcategory R65.2 without documented severe sepsis or acute organ dysfunction.
- Considering 'urosepsis' synonymous with sepsis without provider clarification.
- Assigning severe sepsis codes when acute organ dysfunction is related to a condition other than sepsis.
- Assigning severe sepsis codes without the required minimum of 2 codes (infection + R65.2).
- Failing to query the provider when documentation regarding organ dysfunction association is unclear.

**New in this version:**
- Explicit requirement to query the provider for negative or inconclusive blood cultures in the presence of clinical sepsis evidence.
- Mandatory assignment of code A41.9 when the causal organism is not documented.
- Clarification that 'urosepsis' is not synonymous with sepsis and requires provider clarification.
- Requirement to document the association between acute organ dysfunction and sepsis before coding.
- Mandatory inclusion of additional codes for associated acute organ dysfunction in severe sepsis cases.
