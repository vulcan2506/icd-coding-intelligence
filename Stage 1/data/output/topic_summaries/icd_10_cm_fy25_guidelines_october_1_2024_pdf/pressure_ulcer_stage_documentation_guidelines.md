# Pressure Ulcer Stage Documentation Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Pressure Ulcer Stage Documentation Guidelines

**Key Behaviors:**
- ICD-10-CM pressure ulcer stage codes must be assigned based on explicit clinical documentation of the stage
- Codes may also be assigned using terms found in the Alphabetic Index
- Provider query is required when clinical terms describe the stage but are not in the Alphabetic Index and the stage is undocumented
- System logic validates the presence of stage documentation before finalizing the code assignment
- Undocumented stages with non-indexed clinical terms trigger a mandatory provider interaction workflow
