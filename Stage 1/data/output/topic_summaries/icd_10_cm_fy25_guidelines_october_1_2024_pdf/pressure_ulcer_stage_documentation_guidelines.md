# Pressure Ulcer Stage Documentation Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Pressure Ulcer Stage Coding Guidelines

**Key Behaviors:**
- Assign pressure ulcer stage code based on clinical documentation of the stage
- Assign stage code based on documentation of terms found in the Alphabetic Index
- Query the provider if clinical terms describing the stage are not in the Alphabetic Index and the stage is not documented
- Do not assign a stage code without supporting clinical documentation or Alphabetic Index terms
- Ensure documentation explicitly confirms the stage before code assignment

**Mandatory Coding Criteria:**
- Clinical documentation must specify the pressure ulcer stage
- Documentation must include terms found in the Alphabetic Index if clinical stage terms are absent
- Provider query is required when stage is undocumented and no Alphabetic Index terms exist
