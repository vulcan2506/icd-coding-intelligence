# ICD-10-CM Code Specificity Requirements
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Code Specificity Requirements

**Key Behaviors:**
- Diagnosis codes must be used and reported at their highest number of characters available.
- Codes must be assigned to the highest level of specificity documented in the medical record.
- A three-character code is to be used only if it is not further subdivided by additional characters.

**Mandatory Coding Criteria:**
- Codes must include the full number of characters required for that specific code.
- The 7th character must be included if applicable for the specific code.
- Codes with 3, 4, 5, 6, or 7 characters must be utilized to provide greater detail where available.
