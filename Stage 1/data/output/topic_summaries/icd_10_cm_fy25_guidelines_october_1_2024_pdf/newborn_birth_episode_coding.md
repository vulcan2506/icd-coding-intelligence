# Newborn Birth Episode Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Newborn Birth Episode Coding

**Key Behaviors:**
- Assign a code from category Z38 as the principal diagnosis for the birth episode in a newborn record.
- A code from category Z38 is assigned only once, to a newborn at the time of birth.
- Do not use a code from category Z38 at a receiving hospital if the newborn is transferred from another institution.
- Use a code from category Z38 only on the newborn record, not on the mother's record.

**Requirements / Properties:**
- The code must be assigned at the time of birth.
- The code must be used exclusively on the newborn's medical record.
