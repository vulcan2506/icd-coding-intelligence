# Fetal Surgery Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Fetal Surgery Coding Guidelines

**Key Behaviors:**
- Assign diagnosis code from category O35 to identify the fetal condition when surgery is performed on the fetus.
- Assign the appropriate procedure code for the procedure performed.
- Do not use Chapter 16 perinatal codes on the mother's record to identify fetal conditions.
- Code surgery performed in utero on a fetus as an obstetric encounter.

**Mandatory Coding Criteria:**
- Surgery must be performed on the fetus.
- The fetal condition must be identified using category O35 codes.
- The encounter must be coded as an obstetric encounter.
