# Fetal Surgery Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Fetal Surgery Coding Guidelines

**Key Behaviors:**
- Assign a diagnosis code from category O35 to identify the fetal condition when surgery is performed on the fetus.
- Assign the appropriate procedure code for the specific procedure performed on the fetus.
- Do not use Chapter 16 perinatal codes on the mother's record to identify fetal conditions.
- Code surgery performed in utero on a fetus as an obstetric encounter.

**Requirements / Properties:**
- Diagnosis code must be from category O35 (Maternal care for known or suspected fetal abnormality and damage).
- Procedure code must accurately reflect the procedure performed on the fetus.
- Perinatal codes from Chapter 16 are prohibited for identifying fetal conditions on the mother's record.

**Deprecated in this version:**
- Using Chapter 16 perinatal codes on the mother's record to identify fetal conditions.
- Coding in utero fetal surgery as a non-obstetric encounter.

**New in this version:**
- Mandatory assignment of category O35 diagnosis codes for fetal conditions in fetal surgery cases.
