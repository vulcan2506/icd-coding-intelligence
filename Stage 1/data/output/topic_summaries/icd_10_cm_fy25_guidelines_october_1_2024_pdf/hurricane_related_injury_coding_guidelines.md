# Hurricane-Related Injury Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Hurricane-Related Injury Coding Guidelines

**Key Behaviors:**
- Assign appropriate external cause codes for injuries not directly caused by the hurricane (e.g., motor vehicle accidents) while excluding code X37.0-
- If the cause of an injury is unclear regarding hurricane involvement, assume the injury is due to the hurricane and assign code X37.0-
- Combine code X37.0- with other applicable external cause codes such as X30-, X31-, or X38- when relevant to the injury scenario

**Requirements / Properties:**
- Injuries must be determined to be a direct result of the hurricane to assign code X37.0-
- Documentation must clarify whether an injury is a direct result of the hurricane or a separate event (e.g., motor vehicle accident)
- Code X37.0- must not be assigned if the injury is clearly caused by a non-hurricane event

**Deprecated in this version:**
- Assigning code X37.0- for injuries that are clearly not a direct result of the hurricane (e.g., motor vehicle accidents)

**New in this version:**
- Explicit instruction to assume hurricane causation when the relationship between the injury and the hurricane is unclear
- Inclusion of specific external cause codes X30-, X31-, and X38- as applicable alternatives alongside X37.0-
