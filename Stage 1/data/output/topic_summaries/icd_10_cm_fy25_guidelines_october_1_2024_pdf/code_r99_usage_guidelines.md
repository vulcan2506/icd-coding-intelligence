# Code R99 Usage Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Code R99 Usage Guidelines

**Key Behaviors:**
- Code R99 is restricted to cases where a deceased patient is brought to a facility and pronounced dead upon arrival.
- Code R99 must not be used to represent the discharge disposition of death.

**Mandatory Coding Criteria:**
- Patient must have already died prior to arrival at the facility.
- Death must be pronounced upon arrival in the emergency department or healthcare facility.
