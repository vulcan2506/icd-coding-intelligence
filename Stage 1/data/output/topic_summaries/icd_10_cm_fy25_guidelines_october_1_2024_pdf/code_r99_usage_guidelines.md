# Code R99 Usage Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Code R99 Usage Guidelines

**Key Behaviors:**
- Code R99 is used only when a deceased patient is brought to a facility and pronounced dead upon arrival.
- Code R99 is not used to represent the discharge disposition of death.

**Requirements / Properties:**
- The patient must have already died prior to arrival at the healthcare facility.
- Death must be pronounced at the facility upon arrival.

**Deprecated in this version:**
- Using Code R99 for discharge disposition of death.
