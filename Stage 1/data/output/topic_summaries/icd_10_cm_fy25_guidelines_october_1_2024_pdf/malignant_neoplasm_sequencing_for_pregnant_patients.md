# Malignant Neoplasm Sequencing for Pregnant Patients
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Malignant Neoplasm Sequencing for Pregnant Patients

**Key Behaviors:**
- Code from subcategory O9A.1- must be sequenced first for malignant neoplasms in pregnant patients
- Appropriate code from Chapter 2 must follow to indicate the specific type of neoplasm

**Requirements / Properties:**
- Patient must be pregnant at the time of diagnosis or treatment
- Malignant neoplasm must be documented as complicating pregnancy, childbirth, or the puerperium
- Both the complication code (O9A.1-) and the neoplasm code (Chapter 2) must be present on the claim

**Deprecated in this version:**
- Sequencing the malignant neoplasm code from Chapter 2 before the O9A.1- complication code
- Omitting the O9A.1- code when a malignant neoplasm complicates the pregnancy

**New in this version:**
- Mandatory use of subcategory O9A.1- for all malignant neoplasms complicating pregnancy, childbirth, and the puerperium
