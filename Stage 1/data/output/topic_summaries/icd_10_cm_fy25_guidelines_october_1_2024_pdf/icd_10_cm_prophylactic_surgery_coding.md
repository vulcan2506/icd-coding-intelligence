# ICD-10-CM Prophylactic Surgery Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10-CM Prophylactic Surgery Coding

**Key Behaviors:**
- Assign category Z40 as the principal or first-listed code for encounters specifically for prophylactic organ removal.
- Follow the Z40 code with appropriate codes to identify associated risk factors such as genetic susceptibility or family history.
- Assign a code for an existing malignancy in addition to a Z40.0 code when prophylactic removal is performed at a different site to prevent new primary or metastatic disease.
- Do not assign a Z40.0 code if the organ removal is for the treatment of an existing malignancy.
- Use subcategory Z40.0 specifically for encounters related to risk factors for malignant neoplasms.

**Requirements / Properties:**
- The encounter must be specifically for prophylactic removal of an organ.
- Documentation must identify the associated risk factor (e.g., genetic susceptibility, family history).
- If an existing malignancy is present, it must be documented and coded alongside the prophylactic surgery code.
- The procedure must not be for the treatment of a malignancy at the same site.
- The Z40.0 code must be used only when the surgery is for risk factors related to malignant neoplasms.

**Deprecated in this version:**
- Coding organ removal for treatment of a malignancy as a prophylactic surgery (Z40) instead of a treatment code.
- Omitting the code for an existing malignancy when prophylactic surgery is performed at a different site.
- Using Z40 codes without identifying the associated risk factor.
- Assigning Z40.0 for non-malignant risk factors when the guideline specifies its use for malignant neoplasm risk factors.
- Listing the prophylactic surgery code as secondary when it is the principal reason for the encounter.

**New in this version:**
- Explicit requirement to code the associated risk factor (genetic susceptibility or family history) alongside the Z40 code.
- Clarification that Z40.0 is reserved for risk factors related to malignant neoplasms.
- Mandatory inclusion of an existing malignancy code when prophylactic surgery is performed to prevent metastasis or new primary at another site.
- Strict prohibition against using Z40 for therapeutic organ removal related to active malignancy treatment.
- Designation of Z40 as the principal diagnosis for encounters solely for prophylactic organ removal.
