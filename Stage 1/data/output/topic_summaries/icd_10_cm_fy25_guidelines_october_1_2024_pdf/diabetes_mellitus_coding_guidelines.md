# Diabetes Mellitus Coding Guidelines
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Diabetes Mellitus Coding Guidelines

**Key Behaviors:**
- Diabetes mellitus codes are combination codes including type, body system affected, and complications.
- Codes should be sequenced based on the reason for the particular encounter.
- As many codes as necessary should be assigned to describe all complications of the disease.
- If the type of diabetes is not documented, the default is E11.-, Type 2 diabetes mellitus.
- Additional codes from category Z79 should be assigned to identify long-term use of insulin, oral hypoglycemic drugs, or injectable non-insulin antidiabetic drugs.
- Codes from categories E08, E09, and E13 are used to identify complications/manifestations associated with secondary diabetes mellitus.
- Secondary diabetes mellitus is defined as diabetes always caused by another condition or event (e.g., cystic fibrosis, malignant neoplasm of pancreas, pancreatectomy, adverse effect of drug, or poisoning).

**Mandatory Coding Criteria:**
- Documentation must indicate the type of diabetes mellitus; otherwise, default to E11.-.
- Documentation must specify the use of insulin, oral hypoglycemics, or injectable non-insulin drugs to assign Z79 codes.
- If both oral hypoglycemic drugs and insulin are used, assign both Z79.4 and Z79.84.
- If insulin and injectable non-insulin antidiabetic drugs are used, assign Z79.4 and Z79.85.
- Code Z79.4 should not be assigned if insulin is given temporarily to bring a type 2 patient's blood sugar under control during an encounter.
