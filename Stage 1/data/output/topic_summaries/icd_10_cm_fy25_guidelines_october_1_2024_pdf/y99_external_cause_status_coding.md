# Y99 External Cause Status Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Y99 External Cause Status Coding

**Key Behaviors:**
- Assign a Y99 code whenever any other external cause code is assigned, except for poisonings, adverse effects, misadventures, or late effects.
- Use Y99 to indicate work status (military, non-military at work, or non-work activity) at the time the event occurred.
- Do not assign a Y99 code if no other external cause codes are applicable for the encounter.
- Record only one Y99 code per medical record, used exclusively at the initial encounter for treatment.
- Do not assign code Y99.9 if the status is not stated in the documentation.

**Requirements / Properties:**
- Presence of another applicable external cause code (cause or activity) for the encounter.
- Explicit documentation of the individual's work status at the time of the event.
- Initial encounter for treatment to ensure the code is recorded only once.
- Exclusion of poisonings, adverse effects, misadventures, and late effects from Y99 assignment.
- Documentation must specify the status; if unspecified, Y99.9 must not be assigned.

**Deprecated in this version:**
- Assigning Y99 codes when no other external cause codes are present.
- Recording multiple Y99 codes within a single medical record.
- Using Y99 codes for poisonings, adverse effects, misadventures, or late effects.
- Assigning Y99.9 when the status is documented as unknown or not stated.
- Applying Y99 codes at encounters other than the initial treatment encounter.

**New in this version:**
- Mandatory assignment of Y99 codes alongside Activity codes for all applicable external cause scenarios.
- Strict limitation of Y99 usage to the initial encounter for treatment only.
- Prohibition of Y99.9 assignment when status is not explicitly stated in the record.
