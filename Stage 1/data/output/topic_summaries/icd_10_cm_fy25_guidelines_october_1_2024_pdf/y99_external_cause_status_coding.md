# Y99 External Cause Status Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Y99 External Cause Status Coding

**Key Behaviors:**
- Assign a Y99 code whenever any other external cause code is assigned for an encounter, including activity codes.
- Use Y99 codes to indicate work status (military activity, non-military person at work, or non-work activity) at the time of the event.
- Combine Y99 status codes with other external cause codes such as transport accidents and falls.
- Do not assign Y99 codes if no other external cause codes (cause or activity) are applicable for the encounter.
- Record only one Y99 code per medical record, used exclusively at the initial encounter for treatment.

**Mandatory Coding Criteria:**
- Other external cause codes (cause or activity) must be applicable for the encounter.
- The status must be stated; do not assign Y99.9 if the status is not stated.
- The code is only applicable to events other than poisonings, adverse effects, misadventures, or late effects.
- The code must be assigned at the initial encounter for treatment.
- Only one Y99 code should be recorded on the medical record.
