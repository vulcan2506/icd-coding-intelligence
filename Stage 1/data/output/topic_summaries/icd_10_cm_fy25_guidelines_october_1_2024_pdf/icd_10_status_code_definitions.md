# ICD-10 Status Code Definitions
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Status Code Definitions and Application

**Key Behaviors:**
- Status codes indicate a patient is a carrier of a disease, has sequelae of a past condition, or has prosthetic/mechanical devices, and are distinct from history codes which indicate the condition no longer exists.
- Status codes should not be used with diagnosis codes from body system chapters if the diagnosis code already includes the information provided by the status code.
- For encounters for weaning from a mechanical ventilator, assign code J96.1 (Chronic respiratory failure) followed by Z99.11 (Dependence on respirator status).
- Codes from category Z15 (Genetic susceptibility) should not be used as principal or first-listed codes unless the patient is being seen for genetic counseling associated with procreative management.
- BMI codes (Z68) should only be assigned when there is an associated, reportable diagnosis (such as obesity) and must not be assigned during pregnancy.
- Genetic carrier status indicates the person does not have the disease and is not at risk of developing it.
- Resistance to antimicrobial drugs requires sequencing the infection code first.

**Requirements / Properties:**
- Documentation must confirm the patient is a carrier, has sequelae, or has a prosthetic device to justify a status code.
- If a diagnosis code includes the information provided by a status code, the status code must not be used.
- For genetic susceptibility (Z15), the condition must not be the reason for the encounter unless it is a follow-up after completed treatment where the condition no longer exists.
- BMI codes require an associated reportable diagnosis and are prohibited during pregnancy.
- Physical restraint status (Z78.1) may only be reported if documented by the provider during the current encounter, not for temporary restraints during a procedure.
- Documentation must confirm the patient is a carrier, has sequelae, or has a prosthetic device.
- Z15 codes cannot be principal/first-listed unless for genetic counseling associated with procreative management.

**Deprecated in this version:**
- Using status codes with diagnosis codes from body system chapters when the diagnosis code already provides the necessary information.
- Using Z15 codes as principal or first-listed codes for patients with the condition to which they are susceptible.
- Assigning BMI codes (Z68) during pregnancy.
- Using Z79 for medication administered for a brief period to treat an acute illness or injury.
- Reporting physical restraint status (Z78.1) when the restraint was temporary and occurred during a procedure.

**New in this version:**
- Specific status codes for genetic carriers (Z14), genetic susceptibility (Z15), and resistance to antimicrobial drugs.
- Status codes for retained foreign body fragments and hormone sensitivity malignancy status.
- Status code Z33.1 for incidental pregnant state when pregnancy does not complicate the reason for visit.
- Status code Z66 for Do Not Resuscitate (DNR) status documented by the provider.
- Status codes for blood type (Z67) and body mass index (Z68) with specific assignment rules.
