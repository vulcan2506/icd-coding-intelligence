# ICD-10 Status Code Definitions
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Status Code Definitions and Application

**Key Behaviors:**
- Status codes indicate a patient is a carrier of a disease, has sequelae of a past disease, or has prosthetic/mechanical devices resulting from past treatment.
- Status codes are distinct from history codes; history codes indicate the patient no longer has the condition.
- Status codes should not be used with diagnosis codes from body system chapters if the diagnosis code already includes the information provided by the status code.
- For encounters for weaning from a mechanical ventilator, assign code J96.1 followed by Z99.11.
- BMI codes should only be assigned when there is an associated, reportable diagnosis and must not be assigned during pregnancy.
- Genetic carrier status indicates the person does not have the disease and is not at risk of developing it.
- Genetic susceptibility indicates the person has a gene that increases the risk of developing the disease.

**Mandatory Coding Criteria:**
- Codes from category Z15 should not be used as principal or first-listed codes unless the patient has the condition and it is the reason for the encounter.
- For genetic counseling associated with procreative management, code Z31.5 should be assigned as the first-listed code followed by a code from category Z15.
- For resistance to antimicrobial drugs, sequence the infection code first.
- Code Z66 (Do not resuscitate) may be used when documented by the provider that a patient is on do not resuscitate status at any time during the stay.
- BMI codes must not be assigned during pregnancy.
- For genetic counseling associated with procreative management, code Z31.5 must be assigned as the first-listed code followed by a code from category Z15.
- For resistance to antimicrobial drugs, the infection code must be sequenced first.
