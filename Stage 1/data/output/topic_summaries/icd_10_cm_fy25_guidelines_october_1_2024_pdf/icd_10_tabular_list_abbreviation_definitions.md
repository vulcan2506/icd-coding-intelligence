# ICD-10 Tabular List Abbreviation Definitions
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** ICD-10 Tabular List Abbreviation Definitions

**Key Behaviors:**
- NEC abbreviation represents 'other specified' conditions
- NEC entries identify codes used when a specific code is not available for a condition
- NOS abbreviation is the equivalent of unspecified
- NOS is used when documentation does not specify a particular condition detail
- Abbreviations guide coders to select appropriate codes when specific details are missing

**Mandatory Coding Criteria:**
- Tabular List must include NEC entries under codes to identify 'other specified' options
- Documentation must lack specific details for NOS codes to be applicable
