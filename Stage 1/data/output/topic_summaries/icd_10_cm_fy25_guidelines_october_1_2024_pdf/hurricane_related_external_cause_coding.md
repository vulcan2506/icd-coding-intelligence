# Hurricane-Related External Cause Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Hurricane-Related External Cause Coding

**Key Behaviors:**
- Assign external cause codes to identify the cause of injuries incurred as a result of the hurricane, capturing cause, intent, place, activity, and status.
- Sequence the appropriate injury code before any external cause codes.
- Do not assign external cause codes for encounters treating medical conditions when no injury, adverse effect, or poisoning is involved.
- Assign external cause codes for each encounter for care and treatment of the injury in all healthcare settings.
- Sequence codes for cataclysmic events (hurricane) before other external cause codes, except for child/adult abuse and terrorism.

**Requirements / Properties:**
- External cause codes must be supplemental to ICD-10-CM codes and never recorded as a principal diagnosis.
- Codes must fully explain the cause, including specific scenarios like building collapse or levee breaking related to the hurricane.
- Code X37.0- (Hurricane) must be assigned for injuries directly resulting from the hurricane or flooding caused by a levee breaking related to the hurricane.
- Code X38.- (Flood) must be assigned for injuries from flooding resulting directly from the storm.
- Code X36.0- (Collapse of dam or manmade structure) must not be assigned when the collapse is due to the hurricane; it is limited to collapses due to earth surface movements.

**Deprecated in this version:**
- Assigning external cause codes for encounters where no injury, adverse effect, or poisoning is involved.
- Using code X36.0- for collapses caused by storm surges directly from a hurricane.
- Recording external cause codes as the principal diagnosis.
- Failing to sequence the hurricane code (X37.0-) before other external cause codes for hurricane-related injuries.
- Omitting external cause codes for encounters treating hurricane-related injuries.

**New in this version:**
- Explicit instruction to assign code X37.0- for injuries from flooding caused by a levee breaking related to the hurricane.
- Clarification that code X38.- is for flooding resulting directly from the storm, distinct from hurricane-related flooding.
- Definition of healthcare settings to include any location where medical care is provided by licensed healthcare professionals for data capture purposes.
- Requirement to assign as many external cause codes as necessary to fully explain each cause (e.g., hurricane + building collapse).
- Specific exclusion of code X36.0- for hurricane-related collapses to prevent misclassification of storm surge events.
