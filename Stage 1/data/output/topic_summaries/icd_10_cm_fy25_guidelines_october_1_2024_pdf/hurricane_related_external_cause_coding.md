# Hurricane-Related External Cause Coding
*Source: ICD-10-CM FY25 Guidelines October 1, 2024.pdf*

**Feature:** Hurricane-Related External Cause Coding

**Key Behaviors:**
- Assign external cause codes to identify the cause of injuries incurred as a result of the hurricane.
- Sequence the appropriate injury code before any external cause codes.
- Assign external cause codes for each encounter for care and treatment of the injury.
- Sequence codes for cataclysmic events (hurricane) before other external cause codes, except abuse and terrorism.
- Assign multiple external cause codes if necessary to fully explain the cause (e.g., hurricane and building collapse).
- External cause codes capture the intent of the event (unintentional, accidental, or intentional such as suicide or assault).
- External cause codes capture the place where the event occurred.

**Mandatory Coding Criteria:**
- External cause codes must not be recorded as a principal diagnosis in non-inpatient settings.
- Codes should not be assigned for encounters treating medical conditions when no injury, adverse effect, or poisoning is involved.
- Code X37.0- (Hurricane) is used for injuries from the hurricane or flooding caused by a levee breaking related to the hurricane.
- Code X38.- (Flood) is used for injuries from flooding resulting directly from the storm.
- Code X36.0.- (Collapse of dam or manmade structure) must not be assigned when the cause is due to the hurricane; it is limited to collapses due to earth surface movements.
- Code X37.0- must include the appropriate 7th character.
- Code X38.- must include the appropriate 7th character.
