# COVID-19 Exposure Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** COVID-19 Exposure Coding Guidelines

**Key Behaviors:**
- Assign code Z20.822 for asymptomatic individuals with actual or suspected exposure to COVID-19.
- Assign code Z20.822 for symptomatic individuals with actual or suspected exposure where infection is ruled out or test results are inconclusive/unknown.
- Refer to guideline I.C.21.c.1 for additional guidance on category Z20 codes.
- Refer to guideline I.C.1.g.1.a if COVID-19 is confirmed.

**Mandatory Coding Criteria:**
- Condition must be asymptomatic with actual or suspected exposure.
- Condition must be symptomatic with actual or suspected exposure AND infection ruled out OR test results inconclusive/unknown.
- Confirmation of COVID-19 infection requires different coding per guideline I.C.1.g.1.a.
