# COVID-19 Exposure Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** COVID-19 Exposure Coding Guidelines

**Key Behaviors:**
- Assign code Z20.822 for asymptomatic individuals with actual or suspected exposure to COVID-19.
- Assign code Z20.822 for symptomatic individuals with actual or suspected exposure where infection is ruled out or test results are inconclusive/unknown.
- Refer to guideline I.C.21.c.1 for additional guidance on category Z20 code usage.
- Direct to guideline I.C.1.g.1.a if COVID-19 is confirmed.
- Distinguish between exposure status and confirmed infection status for appropriate code selection.

**Requirements / Properties:**
- Documentation must confirm asymptomatic status or rule out infection for symptomatic individuals to use Z20.822.
- Test results must be inconclusive or unknown for symptomatic individuals to use Z20.822.
- Confirmed COVID-19 cases must follow guideline I.C.1.g.1.a instead of exposure codes.
- Actual or suspected exposure must be documented to justify Z20.822 assignment.
- Guideline I.C.21.c.1 must be consulted for category Z20 code usage specifics.

**New in this version:**
- Code Z20.822 is designated for contact with and (suspected) exposure to COVID-19.
- Symptomatic individuals with ruled-out infection are coded as exposed rather than infected.
- Inconclusive or unknown test results for symptomatic individuals result in exposure coding.
