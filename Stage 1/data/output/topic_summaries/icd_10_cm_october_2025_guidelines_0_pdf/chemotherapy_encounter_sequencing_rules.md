# Chemotherapy Encounter Sequencing Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Chemotherapy Encounter Sequencing

**Key Behaviors:**
- Sequence the diagnosis chiefly responsible for outpatient therapeutic services first.
- Sequence other diagnoses (e.g., chronic conditions) as additional diagnoses.
- When the primary reason is chemotherapy or radiation therapy, list the appropriate Z code first.
- List the diagnosis or problem for which the service is being performed second when chemotherapy/radiation is the primary reason.

**Mandatory Coding Criteria:**
- The diagnosis must be shown in the medical record to be chiefly responsible for the outpatient services provided.
- The encounter must be for therapeutic services only.
- The primary reason for admission/encounter must be chemotherapy or radiation therapy to trigger the Z code exception.
