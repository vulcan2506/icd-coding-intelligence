# ICD-10 Poisoning Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Poisoning Coding Guidelines

**Key Behaviors:**
- Assign appropriate code from categories T36-T50 for poisoning or reaction to improper use of medication.
- Code intent as accidental if unknown or unspecified, and only use undetermined intent if documentation explicitly states it cannot be determined.
- Use additional codes for all manifestations of poisonings.
- Assign abuse or dependence as an additional code if a diagnosis of abuse or dependence of the substance exists.
- Classify drug toxicity or reactions resulting from interactions between nonprescribed and correctly prescribed drugs as poisoning.
- Include errors made in drug prescription or administration by providers, nurses, patients, or other persons as examples of poisoning.

**Mandatory Coding Criteria:**
- Intent must be documented to determine if it is accidental, intentional self-harm, assault, or undetermined.
- Additional codes are required for all manifestations of the poisoning.
- Abuse or dependence must be documented as a separate diagnosis to be assigned as an additional code.
- Documentation must specify that intent cannot be determined to use the undetermined intent code.
- The reaction must result from improper use, overdose, wrong substance, wrong route, or drug interactions.
