# Z37 Maternal Delivery Code Assignment
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Z37 Maternal Delivery Code Assignment

**Key Behaviors:**
- Include a code from category Z37 on every maternal record when a delivery has occurred.
- Do not use Z37 codes on subsequent records.
- Do not use Z37 codes on the newborn record.

**Mandatory Coding Criteria:**
- A delivery must have occurred to assign a Z37 code.
- The code must be assigned to the maternal record, not the newborn record.
