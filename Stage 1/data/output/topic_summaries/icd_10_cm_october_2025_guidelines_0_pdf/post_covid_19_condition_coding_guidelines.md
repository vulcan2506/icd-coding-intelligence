# Post-COVID-19 Condition Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Post-COVID-19 Condition Coding Guidelines

**Key Behaviors:**
- Assign codes for specific symptoms or conditions related to a previous COVID-19 infection if known, followed by code U09.9 for unspecified post-COVID-19 condition.
- Do not assign code U09.9 for manifestations of an active (current) COVID-19 infection.
- When a patient has a condition associated with a previous infection and develops a new active infection, assign U09.9 in conjunction with U07.1 to identify both conditions.
- Assign codes for specific conditions associated with the previous infection alongside codes for manifestations of the new active infection.
- Use code U09.9 to represent the post-COVID-19 condition when specific sequelae are not fully documented or specified.

**Requirements / Properties:**
- Documentation must confirm the condition is a sequela or associated symptom developing following a previous COVID-19 infection.
- The specific symptom or condition must be known and related to the previous infection to be coded individually before using U09.9.
- Active (current) COVID-19 infection must be ruled out or documented separately to avoid misclassification as a post-COVID-19 condition.
- If both previous and new active infections exist, documentation must support the presence of conditions associated with the prior infection.
- Code U09.9 is mandatory when the post-COVID-19 condition is unspecified but confirmed to follow a previous infection.

**Deprecated in this version:**
- Assigning code U09.9 for manifestations of an active (current) COVID-19 infection.
- Coding only the active infection without acknowledging associated post-COVID-19 conditions when both are present.
- Failing to assign specific condition codes when the sequelae are known and documented.
- Using U09.9 as a standalone code without verifying the history of a previous COVID-19 infection.
- Omitting code U07.1 when a new active infection coexists with post-COVID-19 conditions.

**New in this version:**
- Introduction of code U09.9 for unspecified post-COVID-19 condition.
- Requirement to code specific sequelae before using U09.9 if known.
- Guideline for concurrent coding of U09.9 and U07.1 for patients with both post-COVID-19 conditions and new active infections.
- Mandatory inclusion of codes for specific conditions associated with previous infection alongside manifestation codes for new active infection.
- Clarification that U09.9 is not for active infection manifestations but for conditions following prior infection.
