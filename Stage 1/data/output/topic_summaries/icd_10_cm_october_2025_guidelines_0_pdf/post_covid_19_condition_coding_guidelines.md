# Post-COVID-19 Condition Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Post-COVID-19 Condition Coding

**Key Behaviors:**
- Assign specific symptom or condition codes related to a previous COVID-19 infection along with code U09.9 for unspecified post-COVID-19 condition.
- Do not assign code U09.9 for manifestations of an active (current) COVID-19 infection.
- When a patient has conditions from a previous infection and develops a new active infection, assign U09.9 in conjunction with U07.1 to identify both the sequela and the active infection.

**Mandatory Coding Criteria:**
- The condition must be sequela of COVID-19 or associated symptoms/conditions developing following a previous infection.
- The specific symptom(s) or condition(s) related to the previous infection must be known.
- Code U09.9 must not be used if the manifestations are of an active (current) infection only.
