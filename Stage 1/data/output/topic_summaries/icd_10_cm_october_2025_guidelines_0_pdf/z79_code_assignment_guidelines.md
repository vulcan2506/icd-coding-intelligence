# Z79 Code Assignment Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Z79 Code Assignment and tPA Transfer Reporting

**Key Behaviors:**
- Assign Z79 codes for medications used prophylactically, for chronic conditions, or requiring lengthy treatment courses.
- Do not assign Z79 codes for medications administered for brief periods to treat acute illnesses or injuries.
- Assign code Z92.82 as a secondary diagnosis for patients transferred to a facility who received tPA within the last 24 hours prior to admission.
- Code Z92.82 applies to the receiving facility record only, not the transferring facility record.
- The condition for which tPA was administered must be assigned as the primary diagnosis before Z92.82.

**Mandatory Coding Criteria:**
- Documentation must indicate the medication is for prophylaxis, chronic treatment, or a lengthy course.
- Documentation must explicitly exclude brief treatment of acute illness or injury for Z79 assignment.
- Documentation must confirm tPA administration occurred within 24 hours prior to admission to the current facility.
- The primary diagnosis must be the condition treated by the tPA administration.
- Z92.82 is restricted to the receiving facility's claim record.
- The guideline applies even if the patient is still receiving the tPA at the time they are received into the current facility.
