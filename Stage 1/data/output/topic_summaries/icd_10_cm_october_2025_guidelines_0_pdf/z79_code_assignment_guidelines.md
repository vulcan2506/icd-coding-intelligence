# Z79 Code Assignment Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Z79 Code Assignment and tPA Status Guidelines

**Key Behaviors:**
- Assign Z79 codes only for medications used as prophylactic measures or for chronic conditions requiring lengthy treatment.
- Do not assign Z79 codes for medications administered for a brief period to treat acute illness or injury.
- Assign code Z92.82 for patients transferred to a facility who received tPA within the last 24 hours prior to admission.
- The condition code for which tPA was administered must be assigned as the primary diagnosis before Z92.82.
- Code Z92.82 is applicable only to the receiving facility record and not the transferring facility record.

**Requirements / Properties:**
- Documentation must confirm the medication is for an extended period as prophylaxis or treatment of a chronic condition.
- Documentation must indicate the patient received tPA within the last 24 hours prior to admission to the current facility.
- The condition treated by tPA (e.g., cerebrovascular disease, myocardial infarction) must be documented to justify the primary diagnosis.
- The receiving facility must document the transfer status and the timing of tPA administration relative to admission.
- Z79 codes are invalid if the medication course is brief and intended for acute illness or injury.
- The guideline applies even if the patient is still receiving the tPA at the time they are received into the current facility.

**Deprecated in this version:**
- Assigning Z79 codes for brief courses of antibiotics to treat acute bronchitis.
- Assigning Z79 codes for medications administered for a brief period to treat acute illness or injury.
- Using Z79 codes for prophylactic measures that are not extended in duration.
- Assigning Z92.82 in the transferring facility record instead of the receiving facility record.
- Failing to assign the condition code for tPA administration as the primary diagnosis.

**New in this version:**
- Explicit requirement to assign Z92.82 for status post tPA administration within 24 hours of transfer.
- Mandatory sequencing of the condition code for tPA administration before Z92.82.
- Clarification that Z92.82 applies only to the receiving facility record.
- Updated guidance on Z79 exclusions for acute illness or injury treatment.
- Specific exclusion of Z88.9, Z89, Z90, Z91.0-Z91.8, and Z92.82 from Z79 assignment.
