# Extranodal Lymphoma Coding Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Extranodal Lymphoma Coding Rules

**Key Behaviors:**
- Assign codes from categories C81-C85 with a final character identifying 'extranodal and solid organ sites' when malignant neoplasm metastasizes beyond lymph nodes.
- Do not assign a code for the secondary neoplasm of the affected solid organ in cases of extranodal metastasis.
- Use specific codes (e.g., C83.398) for diffuse large B-cell lymphoma involving extranodal and solid organ sites.

**Requirements / Properties:**
- Documentation must confirm metastasis beyond the lymph nodes to extranodal or solid organ sites.
- The primary diagnosis must be a malignant neoplasm of lymphoid tissue.

**Deprecated in this version:**
- Assigning a code for the secondary neoplasm of the affected solid organ when the condition is extranodal metastasis.

**New in this version:**
- Requirement to use final characters identifying 'extranodal and solid organ sites' for categories C81-C85.
