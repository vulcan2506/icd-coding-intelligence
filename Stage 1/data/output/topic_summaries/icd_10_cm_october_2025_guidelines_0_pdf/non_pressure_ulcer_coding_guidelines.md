# Non-Pressure Ulcer Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Non-Pressure Ulcer Coding Guidelines

**Key Behaviors:**
- Assign unstageable pressure ulcer code (L89.--0) only when stage cannot be clinically determined due to eschar or grafts.
- Assign unspecified stage code (L89.--9) when there is no documentation regarding the stage of the pressure ulcer.
- Assign only the code for the stage revealed following debridement if the stage is revealed after an encounter.
- No code is assigned if documentation states the non-pressure ulcer is completely healed at the time of admission.
- Query the provider if documentation is unclear regarding whether the patient has a current non-pressure ulcer or is being treated for a healing one.
- Assign unspecified stage code (L89.--9) if documentation describes a healing pressure ulcer but does not provide information about the stage.
- Assign unspecified severity code for non-pressure ulcers described as healing if documentation does not provide information about the severity.

**Mandatory Coding Criteria:**
- Assignment of unstageable pressure ulcer code must be based on clinical documentation indicating stage cannot be determined.
- Assignment of stage code should be guided by clinical documentation of the stage or terms found in the Alphabetic Index.
- If clinical terms describing the stage are not found in the Alphabetic Index and there is no documentation of the stage, the provider must be queried.
- For ulcers present on admission but healed at discharge, assign the code for the site and stage/severity at the time of admission.
- For patients admitted with ulcers evolving into another stage, assign two separate codes: one for admission stage and one for the highest stage reported.
