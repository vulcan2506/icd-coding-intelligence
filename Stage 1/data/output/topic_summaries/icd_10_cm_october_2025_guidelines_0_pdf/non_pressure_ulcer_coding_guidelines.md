# Non-Pressure Ulcer Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Non-Pressure Ulcer Coding Guidelines

**Key Behaviors:**
- Assign code for unstageable pressure ulcer (L89.--0) only if stage cannot be clinically determined due to eschar or grafts; use unspecified stage (L89.--9) if stage is not documented.
- If stage of an unstageable pressure ulcer is revealed after debridement, assign only the code for the stage revealed following debridement.
- Query the provider if documentation is unclear regarding whether a patient has a current (new) non-pressure ulcer or is being treated for a healing non-pressure ulcer.
- For ulcers present on admission but healed at discharge, assign the code for the site and severity at the time of admission.
- If a patient is admitted with a non-pressure ulcer that evolves to a higher stage during admission, assign two separate codes: one for admission stage and one for the highest stage reported.

**Requirements / Properties:**
- Assignment of unstageable pressure ulcer codes must be based on clinical documentation indicating the stage cannot be determined.
- No code is assigned if documentation states the non-pressure ulcer is completely healed at the time of admission.
- Healing non-pressure ulcers must be assigned codes based on documented severity; unspecified severity is used if not provided.
- Provider query is mandatory when documentation is unclear about the status (new vs. healing) of a non-pressure ulcer.
- Coding for evolving ulcers requires documentation of both the admission stage and the highest stage reached during the stay.
- Assignment of unstageable pressure ulcer codes must be guided by clinical documentation of the stage or terms found in the Alphabetic Index.
- Provider query is required for clinical terms describing the stage that are not found in the Alphabetic Index when there is no documentation of the stage.

**Deprecated in this version:**
- Assigning unspecified stage codes when clinical documentation explicitly indicates the ulcer is unstageable due to eschar or grafts.
- Assigning a single code for a non-pressure ulcer that progresses to a higher stage during an inpatient admission.
- Assigning codes for non-pressure ulcers without querying the provider when the distinction between new and healing ulcers is unclear.
- Assigning codes for non-pressure ulcers that are completely healed at the time of admission.
- Using unspecified severity codes when the medical record provides specific severity documentation for healing non-pressure ulcers.

**New in this version:**
- Explicit instruction to assign only the revealed stage code for unstageable ulcers after debridement.
- Mandatory provider query protocol for ambiguous documentation regarding new versus healing non-pressure ulcers.
- Requirement to assign admission-stage codes for non-pressure ulcers that heal before discharge.
- Dual-code assignment rule for non-pressure ulcers evolving to higher stages during inpatient stays.
- Clarification that unspecified stage (L89.--9) applies only when stage is not documented, distinct from unstageable codes.
