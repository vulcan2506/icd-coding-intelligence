# Postprocedural Infection Code Assignment
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Postprocedural Infection Code Assignment

**Key Behaviors:**
- Code assignment is based on the provider's documentation of the relationship between the infection and the procedure.
- Infections must be explicitly linked to a specific procedure to be classified as postprocedural complications.
- Coder must verify the temporal and causal connection between the procedure and the infection diagnosis.
- Documentation must distinguish between expected postoperative infections and unrelated infections.
- Absence of documented relationship precludes assignment of postprocedural infection codes.

**Requirements / Properties:**
- Provider documentation must explicitly state the infection is related to the procedure.
- Clinical evidence must support the causal link between the procedure and the infection.
- Documentation must specify the timing of the infection relative to the procedure.
- Provider clarification is required if the relationship between infection and procedure is unclear.
- Code assignment is invalid without documented confirmation of the postprocedural nature.

**Deprecated in this version:**
- Assigning postprocedural codes without explicit provider documentation of the relationship.
- Assuming an infection is postprocedural based solely on temporal proximity.
- Coding infections as postprocedural complications when they are known sequelae of the disease.
- Using unspecified codes for postprocedural infections without documented laterality or causality.
- Combining unrelated infection codes with procedure codes without establishing a causal link.

**New in this version:**
- Mandatory requirement for explicit documentation of the infection-procedure relationship.
- Enhanced query protocols for unclear postprocedural infection documentation.
- Updated definitions for distinguishing postprocedural infections from unrelated infections.
- Additional data elements required to support the causal link in claims processing.
- New guidelines for coding specific types of postprocedural infections based on documentation.
