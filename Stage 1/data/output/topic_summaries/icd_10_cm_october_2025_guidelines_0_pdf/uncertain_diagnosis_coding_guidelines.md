# Uncertain Diagnosis Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Uncertain Diagnosis Coding Guidelines

**Key Behaviors:**
- Code uncertain diagnoses as if they existed or were established for inpatient admissions to short-term, acute, long-term care, and psychiatric hospitals.
- Do not code diagnoses documented as 'probable', 'suspected', 'questionable', 'rule out', 'compatible with', 'consistent with', or 'working diagnosis' for outpatient encounters.
- For outpatient encounters, code the condition to the highest degree of certainty, such as symptoms, signs, abnormal test results, or other reason for the visit.
- For outpatient encounters, code the condition to the highest degree of certainty available, such as symptoms, signs, abnormal test results, or other reason for the visit.
- The guideline explicitly distinguishes coding practices for outpatient encounters from those used by short-term, acute care, long-term care, and psychiatric hospitals.

**Mandatory Coding Criteria:**
- Guideline applies only to inpatient admissions to short-term, acute, long-term care, and psychiatric hospitals.
- Basis for coding uncertain diagnoses must be the diagnostic workup, arrangements for further workup or observation, and initial therapeutic approach corresponding to the established diagnosis.
- For outpatient encounters, code must reflect the highest degree of certainty available (symptoms, signs, test results).
