# Superficial Injury Coding Exclusion
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Superficial Injury Coding Exclusion

**Key Behaviors:**
- Superficial injuries such as abrasions or contusions are not coded when associated with more severe injuries of the same site.
- Coder must identify the presence of a more severe injury at the same anatomical location before applying the exclusion rule.
- Only the code for the more severe injury should be assigned on the claim.

**Requirements / Properties:**
- Documentation must explicitly link the superficial injury to a more severe injury occurring at the same site.
- The severity of the associated injury must be clinically established to justify the exclusion of the superficial code.

**Deprecated in this version:**
- Coding both the superficial injury and the more severe injury as separate diagnoses for the same site.
- Assigning a code for an abrasion or contusion when a laceration or fracture exists at the identical location.

**New in this version:**
- Mandatory review of clinical notes to determine if a superficial injury is secondary to a more severe trauma event.
