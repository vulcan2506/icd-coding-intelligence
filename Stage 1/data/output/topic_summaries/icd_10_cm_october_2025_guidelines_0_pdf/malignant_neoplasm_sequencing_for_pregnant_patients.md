# Malignant Neoplasm Sequencing for Pregnant Patients
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Malignant Neoplasm Sequencing for Pregnant Patients

**Key Behaviors:**
- Code from subcategory O9A.1- must be sequenced first for pregnant patients with malignant neoplasms
- Appropriate code from Chapter 2 is sequenced second to indicate the type of neoplasm

**Mandatory Coding Criteria:**
- Patient must be pregnant
- Diagnosis must be a malignant neoplasm complicating pregnancy, childbirth, or the puerperium
- Code from subcategory O9A.1- must be assigned as the primary diagnosis
