# Sequela Code Sequencing Logic
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Sequela Code Sequencing Logic

**Key Behaviors:**
- Sequela codes must be sequenced following the code describing the sequelae of the complication.
- The sequela code is treated as a secondary diagnosis relative to the complication code.
- Coding order is determined by the causal relationship between the complication and its sequela.

**Requirements / Properties:**
- Documentation must explicitly link the sequela to a specific complication.
- The complication code must be present on the claim for the sequela code to be valid.

**Deprecated in this version:**
- Sequela codes listed as primary diagnoses when a complication code exists.
- Coding sequela without identifying the underlying complication.

**New in this version:**
- Mandatory sequencing rule requiring the complication code to precede the sequela code.
