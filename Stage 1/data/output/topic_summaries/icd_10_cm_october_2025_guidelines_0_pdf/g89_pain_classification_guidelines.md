# G89 Pain Classification Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** G89 Pain Classification and Exclusion Rules

**Key Behaviors:**
- Assign code F45.41 exclusively for pain related to psychological disorders, excluding category G89 codes.
- Use code F45.42 with a category G89 code when documentation indicates a psychological component for acute or chronic pain.
- Do not assign category G89 codes if pain is unspecified, post-thoracotomy, postprocedural, or neoplasm-related without further specification.
- Avoid assigning category G89 codes if the underlying definitive diagnosis is known, unless the encounter is solely for pain control.
- Assign category G89 codes alongside site-specific pain codes if the G89 code provides additional information about pain acuity or chronicity.

**Requirements / Properties:**
- Documentation must explicitly state a psychological component to justify using F45.42 with a G89 code.
- The encounter reason must be pain control/management, not underlying condition management, to allow G89 coding when a definitive diagnosis exists.
- Provider documentation must define the pain as acute or chronic to utilize subcategory G89.2, as no specific time frame defines chronicity.
- If the underlying condition is the reason for a procedure (e.g., spinal fusion), the underlying condition must be the principal diagnosis, excluding G89.
- G89 codes may only be combined with site-specific codes if the G89 code adds detail regarding pain acuity or chronicity not present in the site code.

**Deprecated in this version:**
- Assigning category G89 codes when the pain is unspecified, post-thoracotomy, postprocedural, or neoplasm-related.
- Assigning category G89 codes when the underlying definitive diagnosis is known and the encounter is for that condition's management.
- Using unspecified laterality codes when documentation is unclear (general rule applied here to pain context).
- Coding pain disorders with psychological factors using only F45.41 without the accompanying G89 code when indicated.
- Assigning G89 codes as the principal diagnosis when an admission is for a procedure treating the underlying condition.

**New in this version:**
- Explicit instruction to use F45.42 in conjunction with G89 codes for pain with psychological factors.
- Clarification that chronic pain classification (G89.2) relies entirely on provider documentation without a fixed time frame.
- Guidance allowing G89 codes to be used with site-specific codes to describe pain acuity or chronicity.
- Mandatory exclusion of G89 codes when the encounter is for managing a known underlying condition rather than pain itself.
- Requirement to query or rely on specific documentation for pain acuity before assigning G89 codes.
