# External Cause Code Application
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** External Cause Code Application

**Key Behaviors:**
- External cause codes may be used with any code in the range of A00.0-T88.9, Z00-Z99 representing a health condition due to an external cause.
- Codes are valid for injuries, infections, diseases due to an external source, and other conditions like heart attacks during strenuous activity.
- Use the full range of codes to completely describe the cause, intent, place of occurrence, activity, and patient status.
- Application is required for all injuries and other health conditions due to an external cause.

**Mandatory Coding Criteria:**
- The primary diagnosis code must represent a health condition due to an external cause (range A00.0-T88.9, Z00-Z99).
- Documentation must support the cause, intent, place of occurrence, activity, and patient status.
