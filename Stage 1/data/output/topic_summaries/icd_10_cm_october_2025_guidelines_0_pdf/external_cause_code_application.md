# External Cause Code Application
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** External Cause Code Application

**Key Behaviors:**
- External cause codes may be used with any code in the range of A00.0-T88.9, Z00-Z99 representing a health condition due to an external cause.
- Codes are valid for injuries, infections, diseases due to external sources, and other conditions like heart attacks during strenuous activity.
- The full range of codes must be used to describe the cause, intent, place of occurrence, activity, and patient status.
- Application is mandatory for all injuries and other health conditions due to an external cause.
- Codes must completely describe the event context including patient status and activity at the time of the event.

**Requirements / Properties:**
- The health condition must be due to an external cause to qualify for an external cause code.
- Documentation must support the cause, intent, place of occurrence, activity, and patient status.
- Codes must be assigned for all injuries and other health conditions resulting from external causes.
