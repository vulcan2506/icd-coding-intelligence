# Resistant Hypertension Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Resistant Hypertension Coding Guidelines

**Key Behaviors:**
- Assign code I1A.0 as an additional code when apparent treatment resistant hypertension, treatment resistant hypertension, or true resistant hypertension is documented by the provider.
- Sequence a code for the specific type of existing hypertension first if known.

**Requirements / Properties:**
- Provider must document apparent treatment resistant hypertension, treatment resistant hypertension, or true resistant hypertension.
- Patient must have hypertension with blood pressure remaining above goal despite use of antihypertensive medications.

**New in this version:**
- Code I1A.0 is designated for resistant hypertension scenarios.
