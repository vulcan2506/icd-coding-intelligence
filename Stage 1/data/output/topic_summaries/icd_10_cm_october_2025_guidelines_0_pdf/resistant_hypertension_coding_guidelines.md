# Resistant Hypertension Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Resistant Hypertension Coding

**Key Behaviors:**
- Assign code I1A.0 as an additional code when resistant hypertension is documented
- Sequence the specific type of existing hypertension code first if known

**Mandatory Coding Criteria:**
- Blood pressure must remain above goal despite use of antihypertensive medications
- Provider must document apparent treatment resistant hypertension, treatment resistant hypertension, or true resistant hypertension
