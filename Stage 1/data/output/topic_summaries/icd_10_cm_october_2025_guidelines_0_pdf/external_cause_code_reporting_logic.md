# External Cause Code Reporting Logic
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** External Cause Code Reporting Logic

**Key Behaviors:**
- Assign the external cause code with the appropriate 7th character (initial encounter, subsequent encounter, or sequela) for each encounter for which the injury or condition is being treated.
- Assignment of the 7th character for external cause should match the 7th character of the code assigned for the associated injury or condition for the encounter.
- Assign as many external cause codes as necessary to fully explain each cause.
- If only one external code can be recorded, assign the code most related to the principal diagnosis.
- If the format permits capture of additional external cause codes, report the cause/intent, including medical misadventures, of the additional events rather than codes for place, activity, or external status.

**Requirements / Properties:**
- The 7th character must correspond to the specific encounter type (initial, subsequent, or sequela) for the injury or condition being treated.
- The 7th character of the external cause code must match the 7th character of the associated injury or condition code for the same encounter.
- If multiple external cause codes are permitted, the cause/intent of additional events must be reported over codes for place, activity, or external status.
