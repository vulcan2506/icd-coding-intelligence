# Drug and Medicament Coding Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Drug and Medicament Coding Rules

**Key Behaviors:**
- Code each drug, medicinal, or biological substance individually when two or more are taken
- Use individual coding unless a specific combination code is listed in the Table of Drugs and Chemicals
- Assign code from subcategory T50.91 for multiple unspecified drugs, medicaments, or biological substances
- T50.91 covers poisoning, adverse effect, and underdosing of multiple unspecified substances
- Prioritize specific combination codes over individual coding if available in the table
