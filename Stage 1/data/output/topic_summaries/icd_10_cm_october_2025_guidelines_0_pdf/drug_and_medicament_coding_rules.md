# Drug and Medicament Coding Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Multiple Drug and Medicament Coding Rules

**Key Behaviors:**
- Code each drug, medicinal, or biological substance individually when two or more are taken
- Use combination codes only if explicitly listed in the Table of Drugs and Chemicals
- Assign subcategory T50.91 for multiple unspecified drugs, medicaments, or biological substances
- T50.91 covers poisoning, adverse effect, and underdosing scenarios for unspecified multiple substances
- Individual coding takes precedence over combination coding unless a specific combination code exists

**Mandatory Coding Criteria:**
- Verify if a specific combination code exists in the Table of Drugs and Chemicals before individual coding
- Confirm the number of substances taken to determine if T50.91 is applicable
