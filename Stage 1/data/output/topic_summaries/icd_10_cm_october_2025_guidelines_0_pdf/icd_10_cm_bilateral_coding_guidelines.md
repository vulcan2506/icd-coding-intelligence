# ICD-10-CM Bilateral Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10-CM Bilateral Coding Guidelines

**Key Behaviors:**
- Assign separate codes for left and right sides if no bilateral code is provided and the condition is bilateral.
- Assign the unspecified side code only if the side is not identified in the medical record.
- Assign the bilateral code for the first encounter treating a bilateral condition, even if only one side is treated.
- Assign the appropriate unilateral code for subsequent encounters where the condition no longer exists on the previously treated side.
- Base code assignment on medical record documentation from other clinicians if laterality is not documented by the patient's provider.

**Requirements / Properties:**
- The condition must be confirmed as bilateral to assign separate left and right codes when no bilateral code exists.
- The condition must still exist on both sides to assign the bilateral code for the first encounter.
- The condition must no longer exist on the previously treated side to assign a unilateral code for subsequent encounters.
- Conflicting medical record documentation regarding the affected side requires querying the patient's provider for clarification.
- Codes for 'unspecified' side should rarely be used unless documentation is insufficient and clarification is impossible.

**Deprecated in this version:**
- Assigning a complication code from chapter 19 for Breast implant associated anaplastic large cell lymphoma (BIA-ALCL).
- Assigning the bilateral code for subsequent encounters where the condition no longer exists on the previously treated site.
- Using unspecified side codes when documentation is sufficient to determine the affected side or when clarification can be obtained.

**New in this version:**
- Specific assignment of code C84.7A for BIA-ALCL and C84.7B for BIA-ALCL in remission.
- Explicit instruction to assign the bilateral code for the first encounter treating a bilateral condition, including for the encounter to treat the first side.
- Guidance to base code assignment on medical record documentation from other clinicians if laterality is not documented by the patient's provider.
