# ICD-10-CM Bilateral Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10-CM Bilateral Coding Guidelines

**Key Behaviors:**
- Assign separate left and right codes if no bilateral code exists and the condition is bilateral.
- Assign unspecified side codes only when documentation is insufficient and clarification is impossible.
- Use bilateral codes for separate encounters treating both sides if the condition persists on both sides.
- Switch to unilateral codes for subsequent encounters if the condition no longer exists on the previously treated side.
- Do not assign complication codes from chapter 19 for Breast implant associated anaplastic large cell lymphoma (BIA-ALCL).

**Mandatory Coding Criteria:**
- Medical record must explicitly identify the side or indicate bilateral involvement.
- Provider must be queried for clarification if conflicting documentation exists regarding the affected side.
- Bilateral code assignment requires the condition to still exist on both sides at the time of the encounter.
- Unilateral codes are required for subsequent encounters where the condition has resolved on one side.
- BIA-ALCL must be coded using specific lymphoma codes (C84.7A or C84.7B) rather than complication codes.
