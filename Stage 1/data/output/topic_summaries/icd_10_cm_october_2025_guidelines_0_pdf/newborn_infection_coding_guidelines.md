# Newborn Infection Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Newborn Infection Coding Guidelines

**Key Behaviors:**
- Default to birth process origin if documentation is unclear between birth process and community acquisition
- Use Chapter 16 codes for conditions presumed due to the birth process
- Do not assign Chapter 16 codes for confirmed community-acquired conditions in newborns

**Mandatory Coding Criteria:**
- Documentation must indicate the origin of the infection (birth process vs. community) to avoid defaulting
- Chapter 16 codes are mandatory only when the condition is determined to be due to the birth process

**Newly Introduced Guidance:**
- For COVID-19 infection in a newborn, see guideline I.C.16.h.
