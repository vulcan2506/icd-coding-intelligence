# ICD-10-CM Pediatric Encounter Codes
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10-CM Pediatric Encounter Code Definitions

**Key Behaviors:**
- Assigns code Z76.1 for encounters regarding health supervision and care of foundlings
- Assigns code range Z00.1- for routine child health examinations
- Assigns code Z38 for liveborn infants categorized by place of birth and type of delivery

**New in this version:**
- Z76.1 code for foundling care
- Z00.1- code range for routine child exams
- Z38 code for liveborn infant classification
