# Hurricane-Related Injury Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Hurricane-Related Injury Coding

**Key Behaviors:**
- Assign appropriate external cause codes for injuries not directly caused by the hurricane (e.g., motor vehicle accidents) while excluding code X37.0-
- If the hurricane's direct causation is unclear, assume the injury is due to the hurricane and assign code X37.0-
- Combine code X37.0- with other applicable external cause codes such as X30-, X31-, or X38-

**Requirements / Properties:**
- Injuries must be determined to be a direct result of the hurricane to assign code X37.0-
- Documentation must clarify whether the injury is a direct result of the hurricane or a separate event
- Code X37.0- must not be assigned if the injury is caused by a distinct event like a motor vehicle accident

**Deprecated in this version:**
- Assigning code X37.0- for injuries that are not a direct result of the hurricane
- Failing to assign additional applicable external cause codes when the hurricane is the direct cause

**New in this version:**
- Explicit instruction to assume hurricane causation when direct result is unclear
- Inclusion of specific related external cause codes (X30-, X31-, X38-) for combined coding scenarios
