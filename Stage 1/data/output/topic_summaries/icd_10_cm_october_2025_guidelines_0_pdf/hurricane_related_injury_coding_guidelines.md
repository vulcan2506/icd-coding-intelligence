# Hurricane-Related Injury Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Hurricane-Related Injury Coding

**Key Behaviors:**
- Assign appropriate external cause codes for injuries not directly caused by the hurricane (e.g., motor vehicle accidents).
- Do not assign code X37.0- for injuries that are not a direct result of the hurricane.
- If the cause is unclear, assume the injury is due to the hurricane and assign code X37.0-.
- Assign additional applicable external cause codes such as X30-, X31-, or X38- alongside X37.0- when relevant.
- Ensure external cause codes describe the specific cause of the injury.

**Mandatory Coding Criteria:**
- Determine if the injury is a direct result of the hurricane before assigning code X37.0-.
- If the hurricane is not the direct cause, assign the code corresponding to the actual cause of the injury.
- If the hurricane is the direct cause or the cause is unclear, assign code X37.0-.
- Include other applicable external cause codes (X30-, X31-, X38-) if they describe the injury context.
- External cause codes must accurately describe the cause of the morbidity.
