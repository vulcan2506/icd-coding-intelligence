# ICD-10-CM Code Specificity Requirements
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10-CM Code Specificity Requirements

**Key Behaviors:**
- Only codes are permissible for reporting, not categories or subcategories.
- Diagnosis codes must be used and reported at their highest number of characters available.
- Codes must be assigned to the highest level of specificity documented in the medical record.
- A three-character code is to be used only if it is not further subdivided.
- A code is invalid if it has not been coded to the full number of characters required, including the 7th character if applicable.

**Mandatory Coding Criteria:**
- Any applicable 7th character is required for the code to be valid.
- Codes must be composed of 3, 4, 5, 6, or 7 characters.
- Fourth, fifth, and sixth characters must be used to provide greater detail if the category allows subdivision.
- The code assigned must match the highest level of specificity documented in the medical record.
- Reporting must use specific codes rather than categories or subcategories.
