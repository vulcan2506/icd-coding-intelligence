# Sepsis and Localized Infection Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Sepsis and Localized Infection Coding Guidelines

**Key Behaviors:**
- Assign code for underlying systemic infection first if admission is for sepsis/severe sepsis with a localized infection
- Assign code for localized infection as a secondary diagnosis when sepsis is the primary reason for admission
- Assign code from subcategory R65.2 as a secondary diagnosis if the patient has severe sepsis
- Assign localized infection code first if sepsis/severe sepsis develops after admission
- Refer to guideline I.C.1.d.9 for hemolytic-uremic syndrome associated with sepsis

**Requirements / Properties:**
- Clinical documentation must confirm the presence of sepsis or severe sepsis at the time of admission
- Documentation must identify the specific localized infection (e.g., pneumonia, cellulitis) if present
- Documentation must establish the temporal relationship between infection onset and admission to determine coding sequence
- Severe sepsis must be explicitly documented to trigger the assignment of subcategory R65.2
- Hemolytic-uremic syndrome must be documented as associated with sepsis to trigger specific guideline references

**Deprecated in this version:**
- Coding localized infection as the primary diagnosis when sepsis is the reason for admission
- Omitting the R65.2 code when severe sepsis is documented
- Coding sepsis before the underlying infection without clinical evidence of systemic infection
- Failing to sequence diagnoses based on the timing of sepsis development relative to admission
- Ignoring guideline I.C.1.d.9 when hemolytic-uremic syndrome is associated with sepsis

**New in this version:**
- Mandatory assignment of subcategory R65.2 for severe sepsis as a secondary diagnosis
- Strict sequencing rules based on admission reason versus post-admission development of sepsis
- Requirement to link hemolytic-uremic syndrome to sepsis via specific guideline reference I.C.1.d.9
- Explicit prohibition of unspecified coding for sepsis without documented underlying infection
- Enforcement of clinical documentation standards to distinguish between localized and systemic infection
