# Type 1 STEMI/NSTEMI Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Type 1 STEMI/NSTEMI Coding Guidelines

**Key Behaviors:**
- Subcategories I21.0-I21.2 and code I21.3 are used for type 1 ST elevation myocardial infarction (STEMI).
- Code I21.4 is used for type 1 non-ST elevation myocardial infarction (NSTEMI) and nontransmural MIs.
- If a type 1 NSTEMI evolves to STEMI, assign the STEMI code.
- If a type 1 STEMI converts to NSTEMI due to thrombolytic therapy, it is still coded as STEMI.
- For encounters after the 4-week time frame where the patient is still receiving care related to the myocardial infarction, the appropriate aftercare code should be assigned, rather than a code from category I21.

**Requirements / Properties:**
- The myocardial infarction must meet the definition for 'other diagnoses' to be reported for encounters occurring while the MI is equal to, or less than, four weeks old.
- Codes from category I21 may continue to be reported for encounters occurring while the myocardial infarction is equal to, or less than, four weeks old.
- For old or healed myocardial infarctions not requiring further care, code I25.2, Old myocardial infarction, may be assigned.
- The ICD-10-CM codes must identify the site, such as anterolateral wall or true posterior wall.
- Documentation must distinguish between type 1 acute myocardial infarction and other types to select the correct subcategory.
- Codes from category I21 may continue to be reported for encounters occurring while the myocardial infarction is equal to, or less than, four weeks old, including transfers to another acute setting or a postacute setting.
