# Type 1 STEMI/NSTEMI Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Type 1 STEMI/NSTEMI Coding Guidelines

**Key Behaviors:**
- Subcategories I21.0-I21.2 and I21.3 are used for type 1 ST elevation myocardial infarction (STEMI).
- Code I21.4 is used for type 1 non-ST elevation myocardial infarction (NSTEMI) and nontransmural MIs.
- If a type 1 NSTEMI evolves to STEMI, assign the STEMI code.
- If a type 1 STEMI converts to NSTEMI due to thrombolytic therapy, it is still coded as STEMI.
- Codes from category I21 are reported for encounters within four weeks of the myocardial infarction.

**Mandatory Coding Criteria:**
- Codes must identify the site of the infarction (e.g., anterolateral wall or true posterior wall).
- The myocardial infarction must meet the definition for 'other diagnoses' to be reported for encounters occurring while equal to or less than four weeks old.
- Aftercare codes should be assigned rather than category I21 codes for encounters after the 4-week time frame if the patient is still receiving care related to the myocardial infarction.
- Code I25.2 may be assigned for old or healed myocardial infarctions not requiring further care.
- The encounter timing relative to the 4-week mark determines whether category I21 or aftercare codes are appropriate.
- Codes from category I21 may only be reported if the myocardial infarction meets the definition for 'other diagnoses'.
