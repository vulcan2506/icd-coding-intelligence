# Abuse and Neglect Coding Protocols
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Abuse and Neglect Coding Protocols

**Key Behaviors:**
- Sequence confirmed abuse/neglect codes (T74.-) before accompanying mental health or injury codes.
- Sequence suspected abuse/neglect codes (T76.-) before accompanying mental health or injury codes.
- Add external cause codes (X92-Y09) and perpetrator codes (Y07) only for confirmed abuse/neglect cases.
- Do not report external cause or perpetrator codes for suspected abuse/neglect cases.
- Use specific ruled-out encounter codes (Z04.71, Z04.72, Z04.41, Z04.42, Z04.81, Z04.82) instead of T76 codes when allegations are ruled out.

**Mandatory Coding Criteria:**
- Documentation must explicitly state 'abuse' or 'neglect' to assign confirmed codes (T74.-).
- Documentation must explicitly state 'suspected' to assign suspected codes (T76.-).
- Perpetrator code (Y07) requires the perpetrator to be known.
- External cause codes (X92-Y09) are required for confirmed cases to identify the cause of physical injuries.
- Ruled-out allegations require specific Z04 encounter codes rather than T76 codes.
- Perpetrator code (Y07) must only be assigned when the perpetrator of the abuse is known.
