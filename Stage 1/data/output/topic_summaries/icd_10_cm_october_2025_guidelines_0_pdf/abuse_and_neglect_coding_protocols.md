# Abuse and Neglect Coding Protocols
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Abuse and Neglect Coding Protocols

**Key Behaviors:**
- Sequence confirmed abuse/neglect codes (T74.-) first, followed by suspected codes (T76.-), then mental health or injury codes.
- Code confirmed abuse/neglect as T74.- and suspected abuse/neglect as T76.- based on medical record documentation.
- Add external cause codes (X92-Y09) and perpetrator codes (Y07) only for confirmed cases of abuse or neglect.
- Do not report external cause or perpetrator codes for suspected cases of abuse or neglect.
- Use specific ruled-out encounter codes (Z04.71, Z04.72, Z04.41, Z04.42, Z04.81, Z04.82) instead of T76 codes when abuse is ruled out.

**Requirements / Properties:**
- Documentation must explicitly state 'abuse' or 'neglect' to assign confirmed codes (T74.-).
- Documentation must explicitly state 'suspected' to assign suspected codes (T76.-).
- External cause and perpetrator codes are mandatory only when the abuse is confirmed and injuries are present.
- If abuse is ruled out during an encounter, specific Z04.- ruled-out codes must be used instead of T76 codes.
- Perpetrator codes (Y07) require the perpetrator to be known.
- Documentation must explicitly reference 'abuse in a pregnant patient' to trigger specific coding considerations per Section I.C.15.

**Deprecated in this version:**
- Using code T76.- for cases where abuse has been ruled out during the encounter.
- Reporting external cause or perpetrator codes for suspected cases of abuse or neglect.
- Failing to sequence confirmed codes (T74.-) before suspected codes (T76.-) or accompanying condition codes.

**New in this version:**
- Mandatory use of specific encounter codes Z04.71, Z04.72, Z04.41, Z04.42, Z04.81, and Z04.82 for ruled-out cases of physical, sexual, and exploitation abuse.
- Requirement to add perpetrator code Y07 specifically when the perpetrator is known in confirmed cases.
- Explicit instruction to sequence T74.- codes before T76.- codes and any accompanying mental health or injury codes.
