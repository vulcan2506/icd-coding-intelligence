# Immunization and Transition Counseling Codes
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Immunization and Transition Counseling Coding Standards

**Key Behaviors:**
- Code Z71.85 is used specifically for counseling regarding vaccine safety
- Code Z71.85 must not be used for providing general information on risks or side effects during routine vaccine administration
- Code Z71.87 is assigned when pediatric-to-adult transition counseling is the sole reason for the encounter
- Code Z71.87 is assigned when transition counseling is provided alongside treatment of other conditions
- When both transition counseling and medical treatment occur, codes for the medical condition and Z71.87 are assigned with sequencing based on encounter circumstances
- Code Z76.81 is assigned for expectant mother prebirth pediatrician visits
- Code Z76.81 is used to capture encounters related to maternal-fetal transition planning and early pediatric engagement

**Mandatory Coding Criteria:**
- Documentation must confirm the counseling is regarding vaccine safety to use Z71.85
- Documentation must confirm the encounter is for pediatric-to-adult transition to use Z71.87
- Sequencing of Z71.87 with other condition codes depends on the specific circumstances of the encounter
- Documentation must confirm the encounter is an expectant mother prebirth pediatrician visit to use Z76.81
