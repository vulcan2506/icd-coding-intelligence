# Immunization and Transition Counseling Codes
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Immunization and Transition Counseling Coding Rules

**Key Behaviors:**
- Code Z71.85 is used specifically for counseling regarding vaccine safety for patients or caregivers
- Code Z71.85 must not be used for providing general information about risks and side effects during routine vaccine administration
- Code Z71.87 is assigned when pediatric-to-adult transition counseling is the sole reason for the encounter
- Code Z71.87 is also assigned when transition counseling is provided alongside treatment of other conditions
- When both transition counseling and medical treatment occur, codes for the medical condition and Z71.87 are assigned with sequencing based on encounter circumstances

**Requirements / Properties:**
- Z71.85 requires explicit counseling on vaccine safety, not just general risk information
- Z71.87 requires transition counseling to be the primary or additional service provided during the encounter
- Sequencing rules apply when Z71.87 is combined with codes for treated medical conditions

**Deprecated in this version:**
- Using Z71.85 for general risk and side effect information during routine vaccine administration

**New in this version:**
- Code Z71.85 for Encounter for immunization safety counseling
- Code Z71.87 for Encounter for pediatric-to-adult transition counseling
- Code Z76.81 for Expectant mother prebirth pediatrician visit
