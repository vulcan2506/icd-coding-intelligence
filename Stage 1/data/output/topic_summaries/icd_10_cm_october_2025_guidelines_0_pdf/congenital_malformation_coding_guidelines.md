# Congenital Malformation Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Congenital Malformation Coding Guidelines

**Key Behaviors:**
- Assign codes from categories Q00-QA1 when a malformation, deformation, or chromosomal abnormality is documented.
- A malformation/deformation or chromosomal abnormality may be the principal/first-listed diagnosis or a secondary diagnosis.
- Assign additional codes for manifestations that are not an inherent component of the anomaly.
- Do not code separately for manifestations that are an inherent component of the anomaly when the code specifically identifies it.
- For birth admissions, sequence category Z38 (Liveborn infants) as the principal diagnosis followed by congenital anomaly codes.

**Mandatory Coding Criteria:**
- The condition must be documented as a malformation, deformation, or chromosomal abnormality.
- If the code assignment specifically identifies the anomaly, inherent manifestations should not be coded separately.
- For birth admissions, the Z38 code must be sequenced first based on place of birth and type of delivery.
- Codes from Chapter 17 may be used throughout the life of the patient if the condition is diagnosed by the provider.
- If a congenital malformation has been corrected, a personal history code must be used to identify the history.
