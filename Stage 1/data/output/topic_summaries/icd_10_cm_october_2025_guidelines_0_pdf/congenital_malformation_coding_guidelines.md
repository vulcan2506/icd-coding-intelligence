# Congenital Malformation Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Congenital Malformation Coding Guidelines

**Key Behaviors:**
- Assign codes from categories Q00-QA1 when a malformation, deformation, or chromosomal abnormality is documented.
- Assign additional codes for manifestations that are not an inherent component of the anomaly.
- Do not code separately for manifestations that are an inherent component of the identified anomaly.
- Sequence category Z38 as the principal diagnosis for birth admissions, followed by congenital anomaly codes.
- Use personal history codes if a congenital malformation has been corrected.

**Requirements / Properties:**
- The condition must be documented as a malformation, deformation, or chromosomal abnormality.
- For birth admissions, the place of birth and type of delivery must be documented to select the correct Z38 code.
- The provider must have diagnosed the condition for code assignment from Q00-QA1.
