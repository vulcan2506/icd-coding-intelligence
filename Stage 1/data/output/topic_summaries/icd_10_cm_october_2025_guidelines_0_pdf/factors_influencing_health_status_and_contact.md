# Factors Influencing Health Status and Contact
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Factors Influencing Health Status and Contact

**Key Behaviors:**
- Code factors influencing health status and contact with health services to describe the reason for the encounter or the nature of the contact.
- Follow-up codes are used to indicate that a patient is being monitored for a condition or treatment outcome.
- These codes should not be used to describe the primary diagnosis or the main reason for the encounter if a more specific disease code exists.

**Requirements / Properties:**
- Documentation must clearly identify the specific factor influencing health status or the nature of the contact with health services.
- The code must be supported by clinical notes indicating the patient's status or the reason for the visit.

**Deprecated in this version:**
- Using these codes as primary diagnosis codes when a specific disease or condition code is available.
- Assigning codes without explicit documentation of the factor influencing health status.

**New in this version:**
- Enhanced specificity required for follow-up codes to distinguish between routine monitoring and active intervention.
