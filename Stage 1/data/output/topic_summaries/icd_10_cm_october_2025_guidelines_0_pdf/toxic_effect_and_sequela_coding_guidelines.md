# Toxic Effect and Sequela Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Toxic Effect and Sequela Coding

**Key Behaviors:**
- Classify harmful substance ingestion or contact as a toxic effect using categories T51-T65.
- Assign the toxic effect code first, followed by codes for all associated manifestations.
- Ensure toxic effect codes include an associated intent classification (accidental, intentional self-harm, assault, or undetermined).
- Refer to Section I.B.10 for guidance on coding Sequela (Late Effects).

**Mandatory Coding Criteria:**
- Documentation must confirm the harmful substance was ingested or came in contact with the person.
- All associated manifestations of the toxic effect must be coded after the primary toxic effect code.
- An intent code must be assigned to the toxic effect.
