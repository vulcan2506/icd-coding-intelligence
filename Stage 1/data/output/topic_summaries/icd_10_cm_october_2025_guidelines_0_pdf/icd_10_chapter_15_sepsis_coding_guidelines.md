# ICD-10 Chapter 15 Sepsis Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Chapter 15 Sepsis Coding Guidelines

**Key Behaviors:**
- Assign a code for the specific type of infection as an additional diagnosis when coding sepsis complicating abortion, pregnancy, childbirth, and the puerperium.
- If severe sepsis is present, assign a code from subcategory R65.2, Severe sepsis, and codes for associated organ dysfunction(s) as additional diagnoses.
- Code current, acute injuries to the appropriate injury code from chapter 19 rather than chapter 13.
- Code chronic or recurrent bone, joint, or muscle conditions with codes from chapter 13.
- Query the provider if it is difficult to determine from the documentation which code best describes a condition.

**Requirements / Properties:**
- Documentation must specify the specific type of infection for sepsis complicating pregnancy-related conditions.
- Documentation must explicitly indicate severe sepsis to trigger the use of subcategory R65.2.
- Documentation must identify associated organ dysfunction(s) to be coded alongside severe sepsis.
- Documentation must distinguish between current acute injuries and chronic/recurrent conditions to select the correct chapter.
- Provider query is mandatory when documentation is insufficient to determine the best descriptive code.
