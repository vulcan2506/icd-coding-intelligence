# ICD-10 Chapter 15 Sepsis Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Sepsis Coding with Pregnancy and Organ Dysfunction

**Key Behaviors:**
- Assign a code for the specific type of infection as an additional diagnosis when coding sepsis complicating abortion, pregnancy, childbirth, or the puerperium.
- Assign a code from subcategory R65.2 (Severe sepsis) and codes for associated organ dysfunction(s) as additional diagnoses if severe sepsis is present.

**Mandatory Coding Criteria:**
- A code for the specific type of infection must be assigned as an additional diagnosis for sepsis complicating pregnancy-related conditions.
- Code R65.2 must be assigned if severe sepsis is present, along with codes for associated organ dysfunction(s).
