# Z28 Underimmunization Code Mapping
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Z28 Underimmunization Code Mapping

**Key Behaviors:**
- Map Z28 encounters to Z29 for general underimmunization status
- Map Z28 encounters to Z40 for prophylactic surgery contexts
- Map Z28 encounters to Z41 for other prophylactic measures contexts
- Map Z76.3 encounters to Z76.4 for boarder to healthcare facility
- Map Z76.3 encounters to Z91.1- for patient noncompliance with medical treatment

**Mandatory Coding Criteria:**
- Verify the encounter is not specifically Z28.3 (Underimmunization status) before mapping to Z29
- Ensure the specific context (e.g., surgery, other measures) matches the target code category
- Confirm the source code is Z28 or Z76.3 as per the mapping table
- Verify the specific subcode is not Z28.3 before mapping to Z29
- Verify the specific subcode is not Z41.9 before mapping to Z41
