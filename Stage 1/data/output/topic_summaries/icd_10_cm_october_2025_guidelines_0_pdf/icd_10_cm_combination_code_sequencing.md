# ICD-10-CM Combination Code Sequencing
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10-CM Combination Code Sequencing

**Key Behaviors:**
- When using combination codes for atherosclerotic heart disease with angina pectoris, it is not necessary to use an additional code for angina pectoris.
- A causal relationship can be assumed between atherosclerosis and angina pectoris unless documentation indicates the angina is due to something other than the atherosclerosis.
- If a patient with coronary artery disease is admitted due to an acute myocardial infarction (AMI), the AMI should be sequenced before the coronary artery disease.

**Requirements / Properties:**
- Documentation must indicate that angina is not due to a cause other than atherosclerosis to assume the causal relationship for combination codes.
- Admission status must confirm the patient is admitted due to acute myocardial infarction to sequence AMI before coronary artery disease.

**New in this version:**
- The combination codes I25.11 and I25.7 specifically cover atherosclerotic heart disease of native coronary arteries and atherosclerosis of coronary artery bypass grafts/transplanted hearts with angina pectoris.
- A causal relationship between atherosclerosis and angina pectoris is assumed unless documentation explicitly states the angina is due to a cause other than atherosclerosis.
