# ICD-10 Obesity and Hypoventilation Code Classification
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Obesity and Hypoventilation Code Classification

**Key Behaviors:**
- Category E66 includes codes for causes of obesity such as drug-induced obesity (E66.1)
- Category E66 includes codes for effects of obesity such as morbid obesity with alveolar hypoventilation (E66.2)
- Other categories contain specific obesity codes like E88.82 for MC4R pathway disruption
- Fifth character subcategory O99.21 covers obesity complicating pregnancy, childbirth, and the puerperium
- Classification distinguishes between obesity causes and obesity-related effects
