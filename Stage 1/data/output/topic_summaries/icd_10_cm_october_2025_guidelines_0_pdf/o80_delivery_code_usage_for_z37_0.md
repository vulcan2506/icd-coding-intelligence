# O80 Delivery Code Usage for Z37.0
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** O80 Delivery Code Usage for Z37.0

**Key Behaviors:**
- Z37.0 is the only outcome of delivery code appropriate for use with O80
- Other Z37 outcome codes must not be paired with O80

**Requirements / Properties:**
- The claim must document a single live birth outcome
- The delivery code O80 must be paired exclusively with Z37.0

**Deprecated in this version:**
- Pairing O80 with Z37.1 through Z37.9
- Using unspecified delivery outcome codes with O80

**New in this version:**
- Mandatory restriction of Z37.0 as the sole valid outcome for O80
