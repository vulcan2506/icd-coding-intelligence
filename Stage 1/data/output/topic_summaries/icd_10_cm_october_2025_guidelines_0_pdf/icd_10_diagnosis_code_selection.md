# ICD-10 Diagnosis Code Selection
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Diagnosis Code Selection

**Key Behaviors:**
- Use appropriate codes from A00.0 through T88.9 to identify diagnoses, symptoms, conditions, problems, complaints, or other reasons for the encounter.
- Utilize codes from the Z00-Z99.8 range to identify diagnoses, symptoms, conditions, problems, complaints, or other reasons for the encounter.
- Apply codes from the U00-U85 range to identify diagnoses, symptoms, conditions, problems, complaints, or other reasons for the encounter.

**Requirements / Properties:**
- Codes must be selected from the specified ranges (A00.0–T88.9, Z00–Z99.8, U00–U85) to accurately reflect the reason for the encounter.
