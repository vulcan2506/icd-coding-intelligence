# Unqualified Visual Loss Coding Logic
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Unqualified Visual Loss Coding Logic

**Key Behaviors:**
- Assign code H54.3 if blindness or low vision of both eyes is documented without a specific visual impairment category.
- Assign a code from H54.6- if blindness or low vision in one eye is documented without a specific visual impairment category.
- Assign code H54.7 if visual loss is documented without any information regarding whether one or both eyes are affected.

**Mandatory Coding Criteria:**
- Documentation must specify 'blindness' or 'low vision' for H54.3 or H54.6- assignments.
- Documentation must lack a specific visual impairment category for H54.3 or H54.6- assignments.
- Documentation must lack information about the number of eyes affected for H54.7 assignment.
