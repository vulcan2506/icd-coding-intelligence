# Unqualified Visual Loss Coding Logic
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Unqualified Visual Loss Coding Logic

**Key Behaviors:**
- Assign code H54.3 if blindness or low vision of both eyes is documented without a visual impairment category.
- Assign a code from H54.6- if blindness or low vision in one eye is documented without a visual impairment category.
- Assign code H54.7 if visual loss is documented without information on whether one or both eyes are affected.

**Requirements / Properties:**
- Documentation must specify 'blindness' or 'low vision' to trigger unqualified visual loss codes.
- Documentation must indicate the number of eyes affected (both or one) to distinguish between H54.3 and H54.6-.
- Absence of a documented visual impairment category is required to apply the unqualified visual loss codes.
