# Antepartum Complication Trimester Coding Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Antepartum Complication Trimester Coding Rules

**Key Behaviors:**
- Assign the trimester character based on when the complication developed, not the trimester of discharge.
- If the condition developed prior to the current admission, assign the trimester character for the time of admission/encounter.
- Do not use the discharge trimester if the patient remains hospitalized into a subsequent trimester after the complication onset.

**Requirements / Properties:**
- Clinical documentation must clearly identify the specific trimester when the complication first developed.
- Documentation must distinguish between new complications and pre-existing conditions to determine the correct assignment point.

**Deprecated in this version:**
- Assigning the trimester character based on the date of discharge when the complication onset occurred in a prior trimester.
- Using the discharge trimester as the primary determinant for antepartum complication codes in multi-trimester admissions.

**New in this version:**
- Mandatory assignment of the trimester character corresponding to the onset of the complication regardless of length of stay.
