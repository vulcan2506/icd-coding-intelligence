# Antepartum Complication Trimester Coding Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Antepartum Complication Trimester Coding Rules

**Key Behaviors:**
- Assign trimester character based on when the complication developed, not the discharge trimester.
- If condition developed prior to admission or is pre-existing, assign trimester character based on the time of admission/encounter.

**Mandatory Coding Criteria:**
- Patient must be admitted for complications of pregnancy.
- If admission spans multiple trimesters, use the trimester of complication onset.
- If condition is pre-existing, use the trimester at the time of admission.
