# Glaucoma Coding Assignment
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Glaucoma Coding Assignment

**Key Behaviors:**
- Assign as many codes from category H40 as needed to identify the type of glaucoma, the affected eye, and the glaucoma stage.
- Report only the code for the type of glaucoma, bilateral, with the seventh character for the stage when both eyes are the same type and stage and a bilateral code exists.
- Assign the appropriate code for each eye rather than the code for bilateral glaucoma when each eye has a different type or stage and laterality is distinguished.
- Assign one code for each type of glaucoma with the appropriate seventh character for the stage when eyes have different types and laterality is not distinguished.
- Assign a code for the type of glaucoma for each eye with the seventh character for the specific stage when eyes have the same type but different stages and laterality is not distinguished.

**Requirements / Properties:**
- Documentation must identify the type of glaucoma, the affected eye, and the glaucoma stage.
- Laterality must be documented to determine if codes for each eye or a bilateral code is appropriate.
- The seventh character must correspond to the specific glaucoma stage documented for each eye.
- If a bilateral code exists and both eyes match, only the bilateral code should be reported.
- If laterality is not distinguished in the classification, separate codes must be assigned for each eye type or stage.
