# Glaucoma Coding Assignment
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Glaucoma Coding Assignment

**Key Behaviors:**
- Assign as many codes from category H40 as needed to identify the type of glaucoma, the affected eye, and the glaucoma stage.
- Report only the code for bilateral glaucoma with the seventh character for the stage when both eyes are the same type and stage and a bilateral code exists.
- Report only one code for the type of glaucoma with the appropriate seventh character for the stage when both eyes are the same type and stage but no bilateral code exists.
- Assign the appropriate code for each eye rather than the code for bilateral glaucoma when each eye has a different type or stage and laterality is distinguished.
- Assign one code for each type of glaucoma with the appropriate seventh character for the stage when each eye has a different type and laterality is not distinguished.

**Mandatory Coding Criteria:**
- Documentation must identify the type of glaucoma, the affected eye, and the glaucoma stage.
- If bilateral glaucoma is present, determine if both eyes share the same type and stage to decide between a single bilateral code or separate eye codes.
- If laterality is not distinguished in the classification (e.g., subcategories H40.10, H40.20), assign separate codes for each eye if types or stages differ.
- Use the seventh character to indicate the glaucoma stage when applicable.
- Do not report a code for bilateral glaucoma if the eyes have different types or stages.
- When laterality is not distinguished in the classification (e.g., subcategories H40.10, H40.20), assign separate codes for each eye if the types or stages differ.
