# Type 2 Diabetes Remission Coding Logic
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Type 2 Diabetes Remission Coding Logic

**Key Behaviors:**
- Code E11.A is assigned when provider documentation explicitly states Type 2 diabetes mellitus is in remission
- Provider query is triggered if documentation is unclear regarding whether remission has been achieved
- The term 'resolved' is explicitly excluded as a synonym for remission in coding decisions
