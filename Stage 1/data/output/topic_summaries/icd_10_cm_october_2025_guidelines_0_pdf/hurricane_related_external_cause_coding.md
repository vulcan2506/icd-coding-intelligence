# Hurricane-Related External Cause Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Hurricane-Related External Cause Coding

**Key Behaviors:**
- Assign external cause codes to identify the cause of injuries incurred as a result of the hurricane.
- Sequence the appropriate injury code before any external cause codes.
- Assign external cause codes for each encounter for care and treatment of the injury.
- Sequence hurricane codes (X37.0-) before other external cause codes, except for abuse and terrorism.
- Do not assign collapse codes (X36.0-) when the cause is due to a hurricane; use hurricane codes instead.
- External cause codes capture the intent of the event (unintentional, accidental, intentional, suicide, or assault).
- External cause codes capture the place where the event occurred.

**Mandatory Coding Criteria:**
- External cause codes must not be recorded as a principal diagnosis in non-inpatient settings.
- Codes should only be assigned when an injury, adverse effect, or poisoning is involved.
- Assign codes X37.0- for injuries from the hurricane or flooding caused by a levee breaking related to the hurricane.
- Assign code X38.- for injuries from flooding resulting directly from the storm.
- Use code X36.0- only for collapses of man-made structures due to earth surface movements, not storm surges.
- Assign as many external cause of morbidity codes as necessary to fully explain each cause.
- Assign code X37.0- when an injury is incurred as a result of flooding caused by a levee breaking related to the hurricane.
