# Hurricane-Related External Cause Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Hurricane-Related External Cause Coding

**Key Behaviors:**
- Assign external cause codes to identify the cause of injuries incurred as a result of the hurricane, capturing cause, intent, place, activity, and status.
- Sequence the appropriate injury code before any external cause codes.
- Do not assign external cause codes for encounters treating medical conditions where no injury, adverse effect, or poisoning is involved.
- Assign external cause codes for each encounter for care and treatment of the injury in all healthcare settings.
- Sequence cataclysmic event codes (like hurricanes) before other external cause codes, except for abuse and terrorism.

**Requirements / Properties:**
- External cause codes must be supplemental and never recorded as a principal diagnosis.
- The injury code must be sequenced before the external cause code.
- Codes must fully explain the cause, including specific details like building collapse or flooding.
- Code X37.0- must be used for injuries directly from the hurricane or levee breaking related to it.
- Code X38.- must be used for injuries from flooding resulting directly from the storm.

**Deprecated in this version:**
- Assigning code X36.0- (Collapse of dam or manmade structure) when the cause of the collapse is due to the hurricane.
- Using code X36.0- for collapses caused by storm surges directly from a hurricane.
- Assigning external cause codes for encounters where no injury, adverse effect, or poisoning is involved.

**New in this version:**
- Use of code X37.0- for injuries incurred as a direct result of the hurricane.
- Use of code X37.0- when an injury is incurred as a result of flooding caused by a levee breaking related to the hurricane.
- Use of code X38.- for injuries from flooding resulting directly from the storm.
- Requirement to assign as many external cause codes as necessary to fully explain each cause.
- Consideration of any location where medical care is provided by licensed healthcare professionals as a healthcare setting for coding purposes.
