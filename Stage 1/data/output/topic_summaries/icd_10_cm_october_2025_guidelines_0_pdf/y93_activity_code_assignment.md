# Y93 Activity Code Assignment
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Y93 Activity Code Assignment

**Key Behaviors:**
- Assign a code from category Y93 to describe the patient's activity at the time the injury or health condition occurred.
- An activity code is used only once, at the initial encounter for treatment.
- Only one code from Y93 should be recorded on a medical record.
- Do not assign Y93.9, Unspecified activity, if the activity is not stated.
- A code from category Y93 is appropriate for use with external cause and intent codes if identifying the activity provides additional information about the event.

**Requirements / Properties:**
- The activity must be documented as occurring at the time the injury or other health condition occurred.
- The code must be assigned only at the initial encounter for treatment.
- The activity must be explicitly stated; if not stated, Y93.9 must not be assigned.
- Only a single Y93 code is permitted per medical record.
- The code is not applicable to poisonings, adverse effects, misadventures, or sequela.

**Deprecated in this version:**
- Assigning Y93.9 when the activity is not stated.
- Recording more than one Y93 code on a medical record.
- Using Y93 codes for poisonings, adverse effects, misadventures, or sequela.
