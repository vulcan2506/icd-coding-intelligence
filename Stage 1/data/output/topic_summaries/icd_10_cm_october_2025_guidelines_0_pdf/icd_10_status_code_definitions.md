# ICD-10 Status Code Definitions
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Status Code Definitions and Application

**Key Behaviors:**
- Status codes indicate a patient is a carrier of a disease or has sequelae/residuals of a past condition, including prosthetic devices.
- Status codes are distinct from history codes, which indicate the patient no longer has the condition.
- Status codes should not be used with diagnosis codes from body system chapters if the diagnosis code already includes the status information.
- For encounters involving weaning from a mechanical ventilator, assign code J96.1 followed by Z99.11.
- Genetic susceptibility codes (Z15) should generally not be used as principal or first-listed codes unless the susceptibility is the reason for the encounter.
- Status codes include the presence of prosthetic or mechanical devices resulting from past treatment.
- Genetic carrier status (Z14) indicates a person carries a gene associated with a disease that may be passed to offspring, but the person does not currently have the disease.

**Mandatory Coding Criteria:**
- The status must affect the course of treatment or its outcome to be informative.
- If a diagnosis code includes the information provided by the status code, the status code should not be used.
- For genetic susceptibility, if the patient has the condition, code the current condition first; if the condition no longer exists, code follow-up first.
- For genetic counseling encounters, code Z31.5 as the first-listed code followed by category Z15 codes.
- Status codes must be assigned when the patient is a carrier or has sequelae, even if the condition is not currently active.
- For genetic susceptibility (Z15), if the patient has the condition, code the current condition first; if the condition no longer exists, code follow-up first.
