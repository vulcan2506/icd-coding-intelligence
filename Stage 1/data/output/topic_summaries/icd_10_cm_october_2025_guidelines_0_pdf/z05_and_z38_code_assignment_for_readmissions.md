# Z05 and Z38 Code Assignment for Readmissions
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Z05 Code Assignment for Readmissions

**Key Behaviors:**
- Codes from category Z05 may be assigned as a principal or first-listed code for readmissions
- Codes from category Z05 may be assigned as a principal or first-listed code for encounters when the code from category Z38 no longer applies
- Codes from category Z05 are restricted to use only for healthy newborns and infants
- Codes from category Z05 are used when no condition after study is found to be present
- Assignment of Z05 codes is permitted only when the initial Z38 code is no longer applicable

**Mandatory Coding Criteria:**
- Documentation must confirm the newborn/infant is healthy with no condition after study found
- The initial Z38 code must no longer apply to the current encounter or readmission
