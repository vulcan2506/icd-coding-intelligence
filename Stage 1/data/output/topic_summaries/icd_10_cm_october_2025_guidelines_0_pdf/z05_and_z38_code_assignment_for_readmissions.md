# Z05 and Z38 Code Assignment for Readmissions
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Z05 Code Assignment for Readmissions

**Key Behaviors:**
- Codes from category Z05 may be assigned as a principal or first-listed code for readmissions
- Codes from category Z05 may be assigned as a principal or first-listed code for encounters when Z38 codes no longer apply
- Codes from category Z05 are restricted to use only for healthy newborns and infants
- Codes from category Z05 are used when no condition after study is found to be present
- Z05 codes can replace Z38 codes in subsequent encounters if the newborn status changes
