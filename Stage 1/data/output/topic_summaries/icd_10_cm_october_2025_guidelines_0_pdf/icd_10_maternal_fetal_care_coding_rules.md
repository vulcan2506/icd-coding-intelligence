# ICD-10 Maternal Fetal Care Coding Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Maternal Fetal Care Coding Rules

**Key Behaviors:**
- Assign 7th character '0' for single gestations or when documentation is insufficient to determine the affected fetus.
- Codes from categories O35 and O36 are assigned only when the fetal condition modifies maternal management (e.g., requiring diagnostic studies or special care).
- The existence of a fetal condition alone does not justify assigning a code from categories O35 or O36 to the mother's record.

**Requirements / Properties:**
- 7th character must be assigned for categories O31, O32, O33.3-O33.6, O35, O36, O40, O41, O60.1, O60.2, O64, and O69 to identify the affected fetus.
- Clinical documentation must explicitly link the fetal condition to a modification of maternal management for O35/O36 codes.
- The 7th character '0' must be assigned when it is not possible to clinically determine which fetus is affected.
- The 7th character '0' must be assigned when the documentation in the record is insufficient to determine the fetus affected and it is not possible to obtain clarification.
