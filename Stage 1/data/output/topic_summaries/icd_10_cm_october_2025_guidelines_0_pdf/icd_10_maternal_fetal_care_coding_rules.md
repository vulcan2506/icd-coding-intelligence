# ICD-10 Maternal Fetal Care Coding Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Maternal Fetal Care Coding Rules

**Key Behaviors:**
- Assign 7th character to identify the fetus for which the complication code applies in specific categories (O31, O32, O33.3-O33.6, O35, O36, O40, O41, O60.1, O60.2, O64, O69).
- Assign 7th character '0' for single gestations.
- Assign 7th character '0' when documentation is insufficient to determine the affected fetus and clarification is not possible.
- Assign 7th character '0' when it is not clinically possible to determine which fetus is affected.
- Codes from categories O35 and O36 are assigned only when the fetal condition modifies maternal management (e.g., requiring diagnostic studies, special care, or termination).

**Mandatory Coding Criteria:**
- The fetal condition must actually be responsible for modifying the management of the mother to assign codes from categories O35 and O36.
- Assignment of O35/O36 codes is not justified solely by the existence of the fetal condition without impact on maternal care.
- 7th character assignment is required for specific categories to identify the affected fetus.
