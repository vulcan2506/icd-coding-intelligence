# CKD Post-Transplant Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** CKD Post-Transplant Coding Guidelines

**Key Behaviors:**
- Assign appropriate N18 code for the patient's stage of CKD and code Z94.0 for kidney transplant status.
- Do not code CKD alone as a transplant complication.
- Query the provider if documentation is unclear regarding the presence of a transplant complication.
- Code documented transplant complications (e.g., failure, rejection) according to section I.C.19.g.
- Distinguish between residual CKD and active transplant complications based on clinical documentation.

**Requirements / Properties:**
- Clinical documentation must explicitly state the stage of CKD to assign the correct N18 code.
- Documentation must confirm the presence of a kidney transplant to assign Z94.0.
- Documentation must clearly identify specific transplant complications (e.g., rejection, failure) to trigger complication coding.
- Provider query is mandatory when the distinction between residual CKD and transplant complication is ambiguous.
- Coding of CKD as a complication is prohibited unless specific complication criteria are met.

**Deprecated in this version:**
- Coding CKD alone as a complication of the kidney transplant.
- Assigning transplant complication codes without explicit documentation of failure, rejection, or other specific complications.
- Failing to assign Z94.0 when a kidney transplant status is present.
- Assuming CKD resolves post-transplant without verifying current functional status.
- Coding unspecified complications without querying the provider for clarification.

**New in this version:**
- Mandatory assignment of Z94.0 (Kidney transplant status) alongside N18 codes for residual CKD.
- Explicit prohibition of coding CKD as a complication unless specific transplant failure or rejection is documented.
- Requirement to query providers for clarification when the nature of post-transplant renal issues is unclear.
- Reference to section I.C.19.g for specific coding of documented transplant complications.
- Differentiation between chronic kidney disease (N18) and acute transplant complications based on clinical documentation.
