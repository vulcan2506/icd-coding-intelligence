# CKD Post-Transplant Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** CKD Post-Transplant Coding Guidelines

**Key Behaviors:**
- Assign appropriate N18 code for patient's stage of CKD regardless of transplant status
- Code Z94.0 for kidney transplant status
- Do not code CKD as a transplant complication unless specific complication is documented
- Query provider if documentation is unclear regarding transplant complications
- Refer to section I.C.19.g for coding specific transplant complications like failure or rejection

**Mandatory Coding Criteria:**
- Presence of CKD alone does not constitute a transplant complication
- Specific documentation of transplant complication (failure, rejection, etc.) required to code as complication
- Provider query required when documentation is unclear about transplant complications
