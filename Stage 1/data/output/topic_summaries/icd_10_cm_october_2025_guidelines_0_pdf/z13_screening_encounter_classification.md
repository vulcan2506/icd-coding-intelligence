# Z13 Screening Encounter Classification
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Z13 Screening Encounter Classification

**Key Behaviors:**
- Screening is testing for disease in seemingly well individuals for early detection, distinct from diagnostic examinations for suspected diagnoses.
- A screening code may be a first-listed code if the visit reason is specifically the screening exam.
- A screening code may be used as an additional code if screening is done during an office visit for other health problems.
- A screening code is not necessary if the screening is inherent to a routine examination, such as a pap smear during a pelvic exam.
- If a condition is discovered during screening, the code for the condition may be assigned as an additional diagnosis.

**Requirements / Properties:**
- The Z code indicates that a screening exam is planned.
- A procedure code is required to confirm that the screening was performed.
- The sign or symptom must not be used to explain the reason for the test if it is a diagnostic examination.
