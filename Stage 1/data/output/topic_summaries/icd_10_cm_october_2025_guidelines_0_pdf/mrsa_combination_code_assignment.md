# MRSA Combination Code Assignment
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** MRSA Combination Code Assignment

**Key Behaviors:**
- Assign the appropriate combination code for the condition (e.g., sepsis, pneumonia) that includes the causal organism.
- Do not assign code B95.62, Methicillin resistant Staphylococcus aureus infection as the cause of diseases classified elsewhere, as an additional code.
- Do not assign a code from subcategory Z16.11, Resistance to penicillins, as an additional diagnosis.

**Requirements / Properties:**
- The infection must be diagnosed as due to methicillin resistant Staphylococcus aureus (MRSA).
- The condition must have a combination code that includes the causal organism.
- Sequencing instructions for sepsis and severe sepsis must be followed per Section C.1.

**Deprecated in this version:**
- Assigning code B95.62 as an additional code when a combination code is available.
- Assigning a code from subcategory Z16.11 as an additional diagnosis for MRSA infections.

**New in this version:**
- Use of specific combination codes like A41.02 for Sepsis due to Methicillin resistant Staphylococcus aureus.
- Use of specific combination codes like J15.212 for Pneumonia due to Methicillin resistant Staphylococcus aureus.
