# MRSA Combination Code Assignment
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** MRSA Combination Code Assignment

**Key Behaviors:**
- Assign the appropriate combination code for the condition that includes both the infection type and the MRSA organism.
- Do not assign code B95.62 (Methicillin resistant Staphylococcus aureus infection) as an additional code when a combination code is used.
- Do not assign a code from subcategory Z16.11 (Resistance to penicillins) as an additional diagnosis.

**Mandatory Coding Criteria:**
- The infection must be documented as due to methicillin resistant Staphylococcus aureus (MRSA).
- The condition must have a combination code available that includes the causal organism.
- Sequencing instructions for sepsis and severe sepsis must be followed per Section C.1.
