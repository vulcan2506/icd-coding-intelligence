# Code Assignment and Diagnosis Exception Logic
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Code Assignment and Diagnosis Exception Logic

**Key Behaviors:**
- Code assignment is based on the provider's diagnostic statement that the condition exists, not on clinical criteria used to establish the diagnosis.
- Exceptions allow code assignment based on documentation from non-provider clinicians (e.g., dietitians, nurses) for specific metrics like BMI, pressure ulcer stage, and coma scale.
- If conflicting medical record documentation exists from the same or different clinicians, the patient's provider must be queried for clarification.
- Codes for BMI, coma scale, NIHSS, blood alcohol level, SDOH, and underimmunization status must only be reported as secondary diagnoses.
- The associated diagnosis (e.g., obesity, acute stroke) for these metrics must be documented by the patient's provider.
- Code assignment may be based on documentation from clinicians other than the patient's provider for specific metrics if permitted by regulatory or accreditation requirements or internal hospital policies.
- The list of exceptions for non-provider documentation includes laterality and firearm injury intent, in addition to BMI, ulcer stage, coma scale, NIHSS, SDOH, blood alcohol level, and underimmunization status.

**Mandatory Coding Criteria:**
- Provider's statement that the patient has a particular condition is sufficient for code assignment.
- Associated diagnosis for BMI, coma scale, NIHSS, blood alcohol level, SDOH, and underimmunization status must be documented by the patient's provider.
- Specific metrics (BMI, ulcer depth/stage, coma scale, NIHSS, SDOH, underimmunization) may be documented by other permitted clinicians.
- Conflicting documentation requires a query to the patient's provider.
- Exception codes must be reported as secondary diagnoses.
- The associated diagnosis for laterality and firearm injury intent must be documented by the patient's provider.
- Laterality and firearm injury intent codes must only be reported as secondary diagnoses.
