# Symptom and Sign Code Reporting Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Symptom and Sign Code Reporting Guidelines

**Key Behaviors:**
- Codes describing symptoms and signs are acceptable for reporting when a related definitive diagnosis has not been established by the provider.
- Chapter 18 (R00-R99) contains many, but not all, codes for symptoms.
- Symptom codes should be used as a temporary placeholder until a confirmed diagnosis is established.

**Requirements / Properties:**
- A related definitive diagnosis must not be established (confirmed) by the provider for symptom codes to be valid.
- Documentation must explicitly indicate that the diagnosis remains unconfirmed.

**Deprecated in this version:**
- Using symptom codes when a definitive diagnosis is already confirmed by the provider.
- Assuming all codes in Chapter 18 represent definitive diagnoses rather than symptoms.

**New in this version:**
- Explicit reference to Section I.B.18 regarding the use of Signs/Symptom/Unspecified Codes.
