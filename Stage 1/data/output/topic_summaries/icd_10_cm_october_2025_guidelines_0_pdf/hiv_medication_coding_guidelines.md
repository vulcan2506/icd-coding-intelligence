# HIV Medication Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** HIV Medication and Status Coding

**Key Behaviors:**
- Assign code B20 for patients with documented HIV disease, HIV-related illness, or AIDS managed on antiretroviral medications.
- Assign code Z21 for patients with documented HIV positive status managed on antiretroviral medication only if there is no documentation of HIV disease, illness, or AIDS.
- Assign code Z79.899 as an additional code to identify the long-term use of antiretroviral medications.
- Assign code Z11.4 for encounters where a patient without signs or symptoms is tested for HIV.
- Assign code Z71.7 for encounters regarding follow-up of negative HIV test results or for counseling during testing.
- Assign code R75 for patients with documentation of inconclusive HIV serology.
- Assign code Z11.4 with an additional code for associated high-risk behavior if applicable during HIV screening encounters.

**Mandatory Coding Criteria:**
- Code B20 must be assigned on every subsequent admission/encounter once an HIV-related illness has developed.
- Patients previously diagnosed with any HIV illness (B20) must never be assigned to R75 or Z21.
- Code Z11.4 should not be assigned if HIV signs or symptoms are present.
- Codes from Chapter 15 (O98.7) always take sequencing priority for pregnancy, childbirth, or puerperium cases.
- Code Z71.7 may be assigned if counseling is provided during the encounter for the test.
- For pregnancy, childbirth, or puerperium cases with symptomatic HIV disease, assign code O98.7 followed by code B20 and additional codes for HIV-related illnesses.
- For pregnancy, childbirth, or puerperium cases with asymptomatic HIV infection status, assign code O98.7 followed by code Z21.
