# HIV Medication Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** HIV Medication and Status Coding Guidelines

**Key Behaviors:**
- Assign code B20 for patients with documented HIV disease, HIV-related illness, or AIDS currently managed on antiretroviral medications.
- Assign code Z21 for patients with documented HIV positive status managed on antiretroviral medication only if there is no additional documentation of HIV disease, illness, or AIDS.
- Assign code Z11.4 for encounters where a patient without signs or symptoms is tested for HIV, with additional codes for associated high-risk behavior if applicable.
- Assign code Z71.7 for encounters regarding HIV test results (negative or counseling) and do not assign Z11.4 if HIV signs or symptoms are present.
- Assign code Z79.899 as an additional code to identify the long-term use of antiretroviral medications.

**Requirements / Properties:**
- Code B20 must be assigned on every subsequent admission/encounter once an HIV-related illness has developed.
- Patients previously diagnosed with any HIV illness (B20) must never be assigned to R75 or Z21.
- Codes from Chapter 15 (Pregnancy, Childbirth, and the Puerperium) always take sequencing priority when applicable.
- Code Z11.4 should not be assigned if HIV signs or symptoms are present during the encounter.
- Code Z79.899 may be assigned as an additional code to identify long-term use of antiretroviral medications.
- Assign code R75 for patients with documentation of inconclusive HIV serology.
- Assign code B20 for patients with a prior diagnosis of an HIV-related illness on every subsequent admission/encounter.

**Deprecated in this version:**
- Assigning code R75 to patients with a prior diagnosis of an HIV-related illness.
- Assigning code Z21 to patients previously diagnosed with any HIV illness (B20).
- Assigning code Z11.4 for encounters where HIV signs or symptoms are present.
- Failing to assign code B20 on subsequent encounters for patients with documented HIV-related illness.
- Omitting code Z79.899 when identifying long-term antiretroviral medication use.
- Assigning code Z21 to patients with documented asymptomatic HIV infection status or HIV-positive status when additional documentation of HIV disease, HIV-related illness, or AIDS is present.

**New in this version:**
- Explicit instruction to assign code Z79.899 as an additional code for long-term antiretroviral medication use.
- Clarification that code Z71.7 may be assigned for HIV counseling during testing encounters.
- Requirement to code signs and symptoms separately if a patient presents for HIV testing with symptoms.
- Specific guidance for coding HIV infection in pregnancy, childbirth, and the puerperium with subcategory O98.7.
- Mandatory sequencing priority for Chapter 15 codes over other HIV-related codes in pregnancy contexts.
- Mandatory sequencing priority for Chapter 15 codes (O98.7) over other HIV-related codes in pregnancy contexts.
