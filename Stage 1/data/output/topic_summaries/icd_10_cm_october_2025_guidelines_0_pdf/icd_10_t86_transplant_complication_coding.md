# ICD-10 T86 Transplant Complication Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 T86 Transplant Complication Coding

**Key Behaviors:**
- Codes under category T86 are used for both complications and rejection of transplanted organs.
- A transplant complication code is only assigned if the complication affects the function of the transplanted organ.
- Two codes are required to fully describe a transplant complication: the appropriate code from category T86 and a secondary code that identifies the complication.
- Pre-existing conditions or conditions developing after the transplant are not coded as complications unless they affect the function of the transplanted organs.
- Refer to specific guidelines I.C.21, I.C.2, and I.C.1.d.4 for organ removal status, malignant neoplasm association, and sepsis sequencing.

**Requirements / Properties:**
- The complication must affect the function of the transplanted organ to be coded.
- Both a T86 category code and a secondary complication code must be assigned.
- Conditions must be verified as affecting transplanted organ function before being classified as complications.
- Refer to guideline I.C.21 for determining transplant organ removal status.
- Refer to guideline I.C.2 for coding malignant neoplasms associated with transplanted organs.
- Refer to guideline I.C.1.d.4 for sequencing sepsis due to infection in a transplanted organ.
