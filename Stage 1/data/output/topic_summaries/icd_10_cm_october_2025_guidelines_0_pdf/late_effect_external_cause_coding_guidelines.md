# Late Effect External Cause Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Late Effect External Cause Coding Application

**Key Behaviors:**
- Use a late effect external cause code for subsequent visits when a late effect of the initial injury is being treated
- Do not use a late effect external cause code for subsequent visits for follow-up care (e.g., to assess healing, to receive rehabilitative therapy) of the injury when no late effect of the injury has been documented

**Mandatory Coding Criteria:**
- Documentation must confirm the presence of a late effect of the initial injury before assigning the code
