# Late Effect External Cause Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Late Effect External Cause Coding Logic

**Key Behaviors:**
- Assign a late effect external cause code for subsequent visits when a late effect of the initial injury is being treated
- Do not assign a late effect external cause code for subsequent visits if only follow-up care is provided (e.g., assessing healing or rehabilitative therapy)
- Exclude late effect external cause codes when no late effect of the injury has been documented in the claim
