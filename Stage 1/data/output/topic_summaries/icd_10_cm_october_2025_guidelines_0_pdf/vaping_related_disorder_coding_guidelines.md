# Vaping-Related Disorder Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Vaping-Related Disorder Coding Guidelines

**Key Behaviors:**
- Assign code U07.0 as the principal diagnosis for patients presenting with vaping-related conditions.
- Assign only code U07.0 for lung injury due to vaping without additional codes.
- Assign additional codes for other manifestations such as acute respiratory failure or pneumonitis.
- Do not code separately for associated respiratory signs and symptoms (e.g., cough, shortness of breath) when a definitive diagnosis is established.
- Code separately for gastrointestinal symptoms such as diarrhea and abdominal pain.

**Mandatory Coding Criteria:**
- A definitive diagnosis must be established before suppressing separate coding for respiratory signs and symptoms.
- Code U07.0 is required as the principal diagnosis for vaping-related conditions.
- Additional codes are required for specific manifestations like acute respiratory failure or pneumonitis.
