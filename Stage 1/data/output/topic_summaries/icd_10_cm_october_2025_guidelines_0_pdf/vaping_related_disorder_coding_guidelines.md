# Vaping-Related Disorder Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Vaping-Related Disorder Coding Guidelines

**Key Behaviors:**
- Assign code U07.0 as the principal diagnosis for patients presenting with vaping-related conditions.
- Assign only code U07.0 for lung injury due to vaping without additional codes for the injury itself.
- Code additional manifestations separately, such as acute respiratory failure (J96.0-) or pneumonitis (J68.0).
- Do not code associated respiratory signs and symptoms (e.g., cough, shortness of breath) separately when a definitive diagnosis is established.
- Code gastrointestinal symptoms (e.g., diarrhea, abdominal pain) separately when present.

**Requirements / Properties:**
- A definitive diagnosis of a vaping-related condition must be established before coding associated respiratory symptoms.
- Code U07.0 must be used as the principal diagnosis for all vaping-related presentations.
- Specific manifestation codes (e.g., J96.0-, J68.0) are required only for conditions other than the primary lung injury.

**Deprecated in this version:**
- Coding associated respiratory signs and symptoms separately when a definitive vaping-related diagnosis exists.
- Assigning multiple codes for the primary lung injury due to vaping instead of using only U07.0.

**New in this version:**
- Introduction of code U07.0 specifically for Vaping-related disorder.
- Mandatory assignment of U07.0 as the principal diagnosis for vaping-related conditions.
- Explicit guidance to exclude separate coding for respiratory symptoms in the presence of a definitive vaping diagnosis.
