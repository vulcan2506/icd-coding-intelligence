# COVID-19 Acute Bronchitis Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** COVID-19 Acute Bronchitis Coding Guidelines

**Key Behaviors:**
- Assign code U07.1 for acute bronchitis confirmed as due to COVID-19
- Assign code J20.8 for acute bronchitis due to other specified organisms when confirmed as due to COVID-19
- Assign code U07.1 and J40 for bronchitis not otherwise specified (NOS) due to COVID-19

**Requirements / Properties:**
- Acute bronchitis must be confirmed as due to COVID-19 to apply these specific code combinations
- Documentation must distinguish between acute bronchitis and bronchitis not otherwise specified (NOS)

**New in this version:**
- Use of code U07.1 in combination with J20.8 for confirmed COVID-19 acute bronchitis
- Use of code U07.1 in combination with J40 for confirmed COVID-19 bronchitis NOS
