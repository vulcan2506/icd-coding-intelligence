# Aftercare Encounter Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Aftercare Encounter Coding Guidelines

**Key Behaviors:**
- Aftercare Z codes should not be used if treatment is directed at a current, acute disease; instead, the diagnosis code must be used.
- Aftercare Z codes should not be used for aftercare of injuries; assign the acute injury code with the appropriate 7th character for subsequent encounter.
- Aftercare codes are generally first listed to explain the specific reason for the encounter, unless a specific type of aftercare is provided in addition to the reason.
- Status Z codes may be used with aftercare Z codes to indicate the nature of the aftercare, except when the aftercare code already indicates the type of status.
- Aftercare codes should be used in conjunction with other aftercare codes or diagnosis codes to provide better detail on the specifics of an aftercare encounter visit.

**Requirements / Properties:**
- Treatment must be directed at the healing or recovery phase, or for long-term consequences of a disease, rather than a current acute disease.
- For injury aftercare, the acute injury code with the appropriate 7th character for subsequent encounter must be assigned instead of an aftercare Z code.
- Certain aftercare Z code categories require a secondary diagnosis code to describe the resolving condition or sequelae.
- Sequencing of multiple aftercare codes depends on the specific circumstances of the encounter.
- Status codes should not be used when the aftercare code already indicates the type of status (e.g., Z43.0 with Z93.0).

**Deprecated in this version:**
- Using aftercare Z codes for current, acute diseases.
- Using aftercare Z codes for aftercare of injuries.
- Using status codes when the aftercare code indicates the type of status.
- Listing the reason for encounter before the aftercare code when the aftercare is the primary reason.
- Failing to assign a secondary diagnosis for aftercare Z code categories that require it.

**New in this version:**
- Use of specific aftercare Z code categories such as Z42 (plastic and reconstructive surgery), Z43 (attention to artificial openings), and Z44 (fitting and adjustment of external prosthetic device).
- Requirement to use acute injury codes with 7th characters for subsequent encounters instead of aftercare Z codes for injuries.
- Guidance on using status Z codes in conjunction with aftercare Z codes to indicate the nature of the aftercare.
- Instruction to list aftercare codes first to explain the specific reason for the encounter.
- Mandate to use additional codes to provide better detail on the specifics of an aftercare encounter visit.
- The aftercare codes are generally found in the following categories: Z42 Encounter for plastic and reconstructive surgery following medical procedure or healed injury, Z43 Encounter for attention to artificial openings, and Z44 Encounter for fitting and adjustment of external prosthetic device.
