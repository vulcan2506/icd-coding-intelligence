# Aftercare Encounter Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Aftercare Encounter Coding Guidelines

**Key Behaviors:**
- Use aftercare Z codes for continued care during healing/recovery or long-term consequences after initial treatment.
- Do not use aftercare Z codes if treatment is directed at a current, acute disease; use the diagnosis code instead.
- Do not use aftercare Z codes for aftercare of injuries; assign the acute injury code with the appropriate 7th character for subsequent encounter.
- Generally list the aftercare Z code first to explain the reason for the encounter, unless a specific type of aftercare is provided in addition to another reason.
- Use status Z codes with aftercare Z codes to indicate the nature of the aftercare, unless the aftercare code already indicates the type of status.
- An aftercare code may be reported as an additional code when a specific type of aftercare is provided in addition to the reason for the encounter.
- The sequencing of multiple aftercare codes depends on the specific circumstances of the encounter.

**Mandatory Coding Criteria:**
- Initial treatment of the disease must have been performed before assigning aftercare codes.
- Patient must require continued care during the healing or recovery phase, or for long-term consequences.
- For injury aftercare, the acute injury code must be assigned with the appropriate 7th character for subsequent encounter.
- Certain aftercare Z code categories require a secondary diagnosis code to describe the resolving condition or sequelae.
- Status codes should not be used when the aftercare code already indicates the type of status (e.g., Z43.0 with Z93.0).
