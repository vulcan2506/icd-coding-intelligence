# Obesity Code Severity and Assignment Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Obesity Code Severity Assignment and Documentation Rules

**Key Behaviors:**
- Obesity class codes in subcategory E66.81 require a fifth character to convey severity
- Provider must document the obesity class in the medical record for code assignment
- Obesity class codes can be reported with other obesity codes from Chapters 4 and 15
- If both class 3 obesity and morbid obesity are documented, only class 3 obesity code is assigned
- Code Z79.4 is excluded if insulin is given temporarily for secondary diabetes control

**Requirements / Properties:**
- Medical record must explicitly document the obesity class for E66.81 codes
- Provider documentation is mandatory for valid assignment of obesity class codes

**Deprecated in this version:**
- Assignment of code Z79.4 when insulin is given temporarily for secondary diabetes control

**New in this version:**
- Fifth character requirement for E66.81 obesity class codes to specify severity
