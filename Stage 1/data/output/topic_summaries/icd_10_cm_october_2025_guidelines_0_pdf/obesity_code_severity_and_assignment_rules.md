# Obesity Code Severity and Assignment Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Obesity Code Severity and Assignment Rules

**Key Behaviors:**
- Obesity class codes in subcategory E66.81 require a fifth character to convey severity
- The obesity class must be documented in the medical record by the provider for code assignment
- Obesity class codes can be reported with other obesity codes in Chapters 4 and 15 to fully describe the condition
- If both class 3 obesity and morbid obesity are documented, only a code for class 3 obesity should be assigned as it is more specific
- Code Z79.4 should not be assigned if insulin is given temporarily to bring a secondary diabetic patient's blood sugar under control during an encounter

**Mandatory Coding Criteria:**
- Provider documentation of obesity class is required for code assignment
- Fifth character must be used for E66.81 codes to indicate severity
- Documentation must distinguish between temporary and long-term insulin use for Z79.4 assignment
