# Primary Malignancy Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Primary Malignancy Coding and Sequencing

**Key Behaviors:**
- Assign Z85 Personal history codes when a primary malignancy is excised/eradicated with no further treatment or evidence of existing malignancy at that site.
- Code secondary malignant neoplasms to the affected site as the principal diagnosis with Z85 as a secondary code.
- Assign the neoplasm code as principal diagnosis when surgical removal is followed by adjunct chemotherapy or radiation during the same episode.
- Assign Z51.0, Z51.11, or Z51.12 as principal diagnosis for encounters chiefly for antineoplastic therapy administration.
- Sequence the malignancy code as principal diagnosis for brachytherapy encounters, explicitly excluding Z51.0.
- Assign Z51.0, Z51.11, or Z51.12 as the first-listed diagnosis when multiple types of antineoplastic therapy are administered, with one reported as secondary.
- Sequence the malignancy code as principal diagnosis for brachytherapy complications, whereas Z51.0 is sequenced as principal for external beam therapy complications.

**Mandatory Coding Criteria:**
- Verify no further treatment is directed to the site before assigning Z85 for excised malignancies.
- Ensure documentation confirms the malignancy is suspected or probable if coding borderline diagnoses.
- Assign Z85.0-Z85.85 only for the former site of a primary malignancy, not for secondary malignancy sites.
- Use Z85.89 for the former site of either a primary or secondary malignancy.
- Sequence the malignancy code as principal diagnosis when complications arise during brachytherapy insertion/implantation.
- Verify there is no further treatment directed to the site before assigning Z85 for excised malignancies.
- Verify there is no evidence of any existing primary malignancy at the site before assigning Z85.
