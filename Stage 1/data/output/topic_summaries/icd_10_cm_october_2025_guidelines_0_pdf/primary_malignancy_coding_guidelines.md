# Primary Malignancy Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Primary Malignancy Coding Guidelines

**Key Behaviors:**
- Assign Z85 Personal history codes when a primary malignancy is excised/eradicated with no further treatment or evidence of existing malignancy at that site.
- Code secondary malignant neoplasms to the affected site as the principal diagnosis with Z85 as a secondary code.
- Assign the neoplasm code as the principal diagnosis when surgical removal is followed by adjunct chemotherapy or radiation during the same episode.
- Sequence Z51.0, Z51.11, or Z51.12 as the first-listed diagnosis for encounters chiefly for antineoplastic therapy administration.
- Sequence the malignancy code as principal for brachytherapy encounters, explicitly excluding Z51.0 assignment.
- Assign the malignancy code as principal diagnosis when complications (e.g., nausea, vomiting) develop during an encounter chiefly for antineoplastic therapy, followed by codes for the complications.
- Assign the malignancy code as principal diagnosis for brachytherapy encounters when complications develop, followed by codes for the complications.

**Requirements / Properties:**
- Documentation must confirm no further treatment is directed to the site and no evidence of existing primary malignancy to use Z85.
- Encounter reason must be chiefly for antineoplastic therapy administration to justify Z51.x codes as principal diagnosis.
- Malignancy code must be assigned as secondary diagnosis when therapy administration is the primary reason for the encounter.
- Z85 codes must only be assigned for the former site of a primary malignancy, not the site of a secondary malignancy.
- Z85.89 may be assigned for the former site(s) of either a primary or secondary malignancy.
- Documentation must specify that the encounter is chiefly for the administration of chemotherapy, immunotherapy, or external beam radiation therapy to justify Z51.x codes as the principal diagnosis.
- Documentation must confirm that no further treatment is directed to the site of a previously excised malignancy to validate the use of Z85 codes.

**Deprecated in this version:**
- Using Z51.0 for encounters involving insertion or implantation of radioactive elements (brachytherapy).
- Assigning Z85 codes for the site of a secondary malignancy instead of the former site of a primary malignancy.
- Coding the malignancy as secondary when the encounter is chiefly for therapy administration without specifying the therapy type.
- Using unspecified codes for malignancy sites when specific documentation of primary or secondary status is available.
- Failing to sequence complications as secondary diagnoses when the principal diagnosis is the therapy encounter code.

**New in this version:**
- Explicit instruction to sequence Z51.0 and Z51.1 codes together when multiple antineoplastic therapies are administered, with one reported as secondary.
- Clarification that Z85.89 is the specific code for the former site of either a primary or secondary malignancy.
- Requirement to assign the malignancy code as principal for brachytherapy encounters, distinguishing it from external beam radiation therapy.
- Guidance on sequencing complications (e.g., nausea, vomiting) as secondary diagnoses following the principal therapy encounter code.
- Mandate to use primary malignancy codes until treatment is completed if further treatment is directed to the site after excision.
