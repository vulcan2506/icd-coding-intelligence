# Excludes2 Note Logic Implementation
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Excludes2 Note Logic Implementation

**Key Behaviors:**
- Represents 'Not included here' logic where the excluded condition is not part of the represented condition.
- Allows a patient to have both the code with the Excludes2 note and the excluded code simultaneously.
- Permits the use of both the code and the excluded code together when clinically appropriate.

**Mandatory Coding Criteria:**
- The condition excluded must not be part of the condition represented by the code.
- Both conditions must be present in the patient at the same time for dual coding.
