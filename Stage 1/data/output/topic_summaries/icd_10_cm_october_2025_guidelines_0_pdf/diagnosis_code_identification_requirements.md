# Diagnosis Code Identification Requirements
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Diagnosis Code Identification Requirements

**Key Behaviors:**
- Codes from A00.0 through T88.9 must be used to identify diagnoses, symptoms, conditions, problems, complaints, or other reasons for the encounter/visit.
- Codes from Z00-Z99 must be used to identify diagnoses, symptoms, conditions, problems, complaints, or other reasons for the encounter/visit.
- Codes from U00-U85 must be used to identify diagnoses, symptoms, conditions, problems, complaints, or other reasons for the encounter/visit.

**Requirements / Properties:**
- The code must fall within the specified range of A00.0 through T88.9, Z00-Z99, and U00-U85.
- The code must accurately identify the diagnosis, symptom, condition, problem, complaint, or reason for the encounter/visit.
