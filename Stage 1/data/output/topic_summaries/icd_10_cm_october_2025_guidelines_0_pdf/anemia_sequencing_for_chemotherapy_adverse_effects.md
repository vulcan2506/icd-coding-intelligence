# Anemia Sequencing for Chemotherapy Adverse Effects
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Anemia Sequencing for Chemotherapy and Radiotherapy Adverse Effects

**Key Behaviors:**
- Sequence anemia code first when admission is for management of anemia associated with chemotherapy adverse effect and only treatment is for anemia.
- Sequence anemia code first when admission is for management of anemia associated with radiotherapy adverse effect.
- Follow anemia code with appropriate neoplasm code and adverse effect code (T45.1X5- for chemotherapy) or cause code (Y84.2 for radiotherapy).
- Use Y84.2 specifically for radiotherapy as the cause of abnormal reaction without mention of misadventure.
- Ensure adverse effect codes are linked to the specific administration type (chemotherapy vs radiotherapy).

**Mandatory Coding Criteria:**
- Admission or encounter must be specifically for management of anemia.
- Anemia must be documented as associated with an adverse effect of chemotherapy or immunotherapy.
- Only treatment provided must be for the anemia (no other concurrent treatments for the underlying condition).
- For radiotherapy, code Y84.2 must be used to indicate the cause of the abnormal reaction.
- Neoplasm code must be included to identify the underlying condition associated with the adverse effect.
- The encounter must be for management of anemia associated with an adverse effect of immunotherapy.
- The encounter must be for management of anemia associated with an adverse effect of radiotherapy.
