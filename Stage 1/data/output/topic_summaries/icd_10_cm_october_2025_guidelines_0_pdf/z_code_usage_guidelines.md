# Z Code Usage Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Z Code Usage and Specificity Guidelines

**Key Behaviors:**
- Z codes may be used as first-listed or secondary codes depending on the encounter circumstances.
- Certain Z codes are restricted to use only as first-listed or principal diagnosis codes.
- Nonspecific Z codes should generally be avoided in the inpatient setting due to lack of justification.
- Outpatient use of nonspecific Z codes is limited to instances where no further documentation permits more precise coding.
- Any sign, symptom, or reason for visit captured in another code should be used instead of a nonspecific Z code.

**Requirements / Properties:**
- Documentation must support the use of a Z code as a first-listed diagnosis in the inpatient setting.
- More precise coding must be available before assigning a nonspecific Z code in the outpatient setting.
- If a sign or symptom is captured in another code, that code must be used instead of a Z code.

**Deprecated in this version:**
- Using nonspecific Z codes in the inpatient setting when more specific codes are available.
- Assigning nonspecific Z codes in the outpatient setting when further documentation permits precise coding.
- Using Z codes for signs or symptoms that are already captured in other codes.

**New in this version:**
- Explicit restriction of certain Z codes to first-listed or principal diagnosis status only.
- Mandatory avoidance of nonspecific Z codes in favor of more precise codes when documentation allows.
- Specific code mappings for 'Encounter for administrative examinations, unspecified' (Z02.9) to more precise codes such as Z13.9, Z04.9, and Z41.9.
