# Z Code Usage Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Z Code Usage and Specificity

**Key Behaviors:**
- Z codes may be used as first-listed or secondary codes depending on the encounter circumstances.
- Certain Z codes are restricted to use only as first-listed or principal diagnosis codes.
- Use of non-specific Z codes in the inpatient setting is discouraged due to redundancy with other codes.
- Outpatient use of non-specific Z codes is limited to instances where no further documentation permits more precise coding.
- Any sign, symptom, or reason for visit captured in another code should be used instead of a non-specific Z code.

**Mandatory Coding Criteria:**
- Z codes must be appropriate for the specific healthcare setting (inpatient or outpatient).
- More precise codes must be used if documentation supports them over non-specific Z codes.
- Non-specific Z codes should only be assigned when no other code exists to capture the reason for the visit.
- Z codes must be used as either a first-listed or secondary code depending on the circumstances of the encounter.
- Certain Z codes are restricted to use only as first-listed or principal diagnosis codes.

**Superseded Guidance:**
- Encounter for administrative examinations, unspecified (Z02.9) is deprecated in favor of more specific codes such as Encounter for examination and observation for unspecified reason (Z04.9), Encounter for screening, unspecified (Z13.9), or Encounter for procedure for purposes other than remedying health state, unspecified (Z41.9).
- The use of the non-specific code Z02.9 is no longer permitted when more specific documentation exists for the reason for the visit.
