# ICD-10-CM Code Structure and Subdivision
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10-CM Code Structure and Subdivision

**Key Behaviors:**
- Codes are composed of 3, 4, 5, 6, or 7 characters.
- Three-character codes serve as the heading of a category.
- Categories may be further subdivided using fourth, fifth, sixth, or seventh characters.
- Subdivision provides greater specificity for the diagnosis.
- The structure allows for hierarchical coding from broad categories to specific instances.

**Mandatory Coding Criteria:**
- A three-character code must be used as the initial category heading.
- Additional characters (4th through 7th) must be used to achieve the required level of specificity.
- The code length must fall within the 3 to 7 character range.
