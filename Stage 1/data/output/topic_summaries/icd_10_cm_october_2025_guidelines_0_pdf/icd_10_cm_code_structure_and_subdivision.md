# ICD-10-CM Code Structure and Subdivision
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10-CM Code Structure and Subdivision

**Key Behaviors:**
- Codes are composed of 3, 4, 5, 6, or 7 characters.
- Three-character codes serve as the heading of a category.
- Categories may be further subdivided using fourth, fifth, sixth, or seventh characters.
- Subdivision characters provide greater specificity to the diagnosis.
- The structure allows for hierarchical coding from broad categories to specific instances.

**Requirements / Properties:**
- A three-character code must be assigned as the initial category heading.
- Additional characters must be used if the category allows for further subdivision.
- The final code must reflect the highest level of specificity available in the tabular list.
- All characters in the code must be valid according to the ICD-10-CM official guidelines.
- The code length must correspond to the specific condition's required detail level.
