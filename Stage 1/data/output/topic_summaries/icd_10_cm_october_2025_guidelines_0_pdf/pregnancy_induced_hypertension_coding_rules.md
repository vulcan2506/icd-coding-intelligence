# Pregnancy-Induced Hypertension Coding Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Pregnancy-Induced Hypertension Coding Rules

**Key Behaviors:**
- Assign code R03.0 for elevated blood pressure reading without a diagnosis of hypertension unless the patient has an established diagnosis of hypertension.
- Assign code O13.- for gestational [pregnancy-induced] hypertension without significant proteinuria.
- Assign code O14.- for pre-eclampsia.
- Codes O13.- and O14.- are used specifically for transient hypertension of pregnancy.

**Requirements / Properties:**
- Patient must not have an established diagnosis of hypertension to use code R03.0.
- Clinical documentation must distinguish between transient hypertension of pregnancy and established hypertension.
- Documentation must specify the presence or absence of significant proteinuria to differentiate between O13.- and O14.-.
