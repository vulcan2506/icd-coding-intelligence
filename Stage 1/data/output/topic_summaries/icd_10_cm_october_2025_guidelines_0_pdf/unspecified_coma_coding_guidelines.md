# Unspecified Coma Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Unspecified Coma Coding Guidelines

**Key Behaviors:**
- Assign code R40.20 when the underlying cause of the coma is not known.
- Assign code R40.20 when the cause is a traumatic brain injury and the coma scale is not documented.
- Do not report unspecified coma codes for patients with medically induced coma or sedated patients.
- Do not report individual or total Glasgow coma scale scores for patients with medically induced coma or sedated patients.

**Mandatory Coding Criteria:**
- Underlying cause of coma must be unknown to assign R40.20.
- Coma scale must be documented if the cause is a traumatic brain injury to avoid R40.20.
- Patient must not be in a medically induced coma or sedated state to assign R40.20.
