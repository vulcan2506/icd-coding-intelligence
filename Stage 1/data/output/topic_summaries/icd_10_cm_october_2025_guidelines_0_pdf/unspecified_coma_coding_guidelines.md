# Unspecified Coma Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Unspecified Coma Coding Guidelines

**Key Behaviors:**
- Assign code R40.20 when the underlying cause of the coma is not known.
- Assign code R40.20 when the cause is a traumatic brain injury and the coma scale is not documented.
- Do not report unspecified coma codes for patients with medically induced coma.
- Do not report unspecified coma codes for sedated patients.
- Do not report individual or total Glasgow coma scale scores for medically induced or sedated patients.

**Requirements / Properties:**
- The underlying cause of the coma must be unknown to justify R40.20.
- If the cause is traumatic brain injury, the coma scale must be documented in the medical record.
- Documentation must explicitly distinguish between natural coma and medically induced/sedated states.

**Deprecated in this version:**
- Reporting unspecified coma codes for medically induced comas.
- Reporting unspecified coma codes for sedated patients.
- Reporting Glasgow coma scale scores for medically induced or sedated patients.

**New in this version:**
- Mandatory assignment of R40.20 when TBI cause exists but coma scale is missing.
- Explicit prohibition against coding unspecified coma for sedated or medically induced states.
