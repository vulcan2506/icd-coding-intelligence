# MRSA Infection and Carrier Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** MRSA Infection and Carrier Coding

**Key Behaviors:**
- Assign appropriate infection code with B95.62 as the cause of disease when no combination code exists.
- Do not assign codes from subcategory Z16.11 for MRSA infections.
- Assign both Z22.322 and the MRSA infection code if a patient has both colonization and infection during admission.

**Mandatory Coding Criteria:**
- Documentation must confirm a current infection due to MRSA.
- The specific infection must lack a combination code that includes the causal organism.
- Code B95.62 must be used as the cause of disease for MRSA infections without a combination code.

**Superseded Guidance:**
- Assignment of codes from subcategory Z16.11 for MRSA infections.
