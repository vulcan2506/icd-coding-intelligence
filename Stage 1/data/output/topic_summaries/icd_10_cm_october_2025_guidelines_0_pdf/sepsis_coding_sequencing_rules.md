# Sepsis Coding Sequencing Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Sepsis Coding Sequencing and R65.2 Assignment

**Key Behaviors:**
- Underlying systemic infection sequenced first if severe sepsis is present on admission and meets principal diagnosis definition.
- Code from subcategory R65.2 can never be assigned as a principal diagnosis.
- If severe sepsis develops during an encounter, underlying infection and R65.2 code are assigned as secondary diagnoses.
- For infection following a procedure, the procedure-related infection code is sequenced first, followed by sepsis code and R65.2 if severe.
- Only one code from category R65 should be assigned; do not additionally assign R65.1 for non-infectious origin SIRS when severe sepsis is present.
- For infection-associated hemolytic-uremic syndrome with severe sepsis, assign code D59.31 as the principal diagnosis.
- When severe sepsis complicates abortion, pregnancy, childbirth, or the puerperium, refer to specific section I.C.15 guidelines.

**Mandatory Coding Criteria:**
- Severe sepsis must meet the definition of principal diagnosis to sequence underlying infection first.
- Provider must be queried if documentation is unclear whether severe sepsis was present on admission.
- Code from subcategory R65.2 must be assigned with additional codes for any acute organ dysfunction.
- For non-infectious conditions leading to infection, the non-infectious condition code is sequenced first if it meets principal diagnosis definition.
- If infection meets principal diagnosis definition, it should be sequenced before the non-infectious condition.
- For postprocedural wound infections, assign the site-specific infection code (T81.41-T81.43, T81.49, O86.00-O86.03) first, followed by the sepsis following procedure code (T81.44 or O86.04).
- For infections following infusion, transfusion, therapeutic injection, or immunization, assign the code from subcategory T80.2 or T88.0- first, followed by the specific infection code.
