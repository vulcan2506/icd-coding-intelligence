# MIS-C Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** MIS-C Coding Guidelines

**Key Behaviors:**
- Assign code U07.1 as the principal diagnosis for individuals with MIS and current COVID-19 infection
- Assign code M35.81 as an additional diagnosis for Multisystem inflammatory syndrome
- Assign code U09.9 for individuals with a history of COVID-19 who develop MIS
- Assign code Z20.822 for individuals with suspected exposure to COVID-19 who develop MIS without current infection or history
- Assign additional codes for any associated complications of MIS

**Requirements / Properties:**
- Current COVID-19 infection must be documented to assign U07.1 as the principal diagnosis
- History of COVID-19 must be documented to assign U09.9 as an additional diagnosis
- Known or suspected exposure to COVID-19 must be documented to assign Z20.822
- MIS diagnosis must be confirmed to assign M35.81
- Associated complications must be documented to justify additional coding

**New in this version:**
- Use of code U07.1 specifically for COVID-19 in MIS-C cases
- Use of code M35.81 for Multisystem inflammatory syndrome
- Use of code U09.9 for Post COVID-19 condition unspecified
- Use of code Z20.822 for contact with and suspected exposure to COVID-19
