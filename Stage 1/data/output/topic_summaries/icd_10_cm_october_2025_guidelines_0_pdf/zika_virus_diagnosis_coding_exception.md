# Zika Virus Diagnosis Coding Exception
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Zika Virus Diagnosis Coding Exception

**Key Behaviors:**
- Code only a confirmed diagnosis of Zika virus (A92.5) as documented by the provider.
- Confirmation does not require documentation of the type of test performed; the provider's diagnostic statement is sufficient.
- Assign code A92.5 regardless of the stated mode of transmission.
- Do not assign code A92.5 if the provider documents 'suspected', 'possible', or 'probable' Zika.
- For non-confirmed cases, assign a code explaining the reason for encounter (e.g., fever, rash) or Z20.821.

**Mandatory Coding Criteria:**
- Provider must explicitly document the condition as 'confirmed'.
- Documentation of the specific test type is not required for confirmation.
- Diagnosis must be documented as suspected, possible, or probable to trigger the exception (no A92.5).
- Alternative codes must explain the reason for encounter if Zika is not confirmed.
- Code Z20.821 is used for contact with or suspected exposure to Zika virus.
