# Zika Virus Diagnosis Coding Exception
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Zika Virus Diagnosis Coding Exception

**Key Behaviors:**
- Code only a confirmed diagnosis of Zika virus (A92.5) as documented by the provider without requiring test type documentation.
- Assign code A92.5 regardless of the stated mode of transmission.
- Do not assign code A92.5 if the provider documents 'suspected', 'possible', or 'probable' Zika.
- For non-confirmed cases, assign codes explaining the reason for encounter (e.g., fever, rash) or Z20.821 for exposure.
- This exception overrides the standard hospital inpatient guideline Section II, H regarding confirmation requirements.

**Requirements / Properties:**
- Provider must explicitly state the condition is 'confirmed' to assign code A92.5.
- Documentation of the specific diagnostic test type is not required for confirmation.
- Mode of transmission documentation does not affect the assignment of code A92.5 for confirmed cases.
- Use of terms 'suspected', 'possible', or 'probable' precludes the use of code A92.5.
- Alternative codes must describe the clinical reason for encounter or exposure status if Zika is not confirmed.

**Deprecated in this version:**
- Requiring documentation of the specific test type performed to confirm Zika virus diagnosis.
- Assigning code A92.5 for 'suspected', 'possible', or 'probable' Zika virus diagnoses.
- Withholding code A92.5 based on the stated mode of transmission when the diagnosis is confirmed.
- Applying standard hospital inpatient guideline Section II, H confirmation rules to Zika virus cases.
- Failing to assign alternative symptom or exposure codes when Zika virus is not confirmed.

**New in this version:**
- Explicit exception to hospital inpatient guideline Section II, H for Zika virus diagnosis coding.
- Acceptance of provider diagnostic statement alone as sufficient for 'confirmation' without test details.
- Mandatory assignment of code A92.5 irrespective of transmission mode for confirmed cases.
- Designated use of Z20.821 for contact with or suspected exposure to Zika virus.
- Requirement to code underlying symptoms (fever, rash, joint pain) when Zika is not confirmed.
