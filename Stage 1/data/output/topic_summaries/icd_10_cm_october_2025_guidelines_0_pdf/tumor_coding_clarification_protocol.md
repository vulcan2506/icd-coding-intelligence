# Tumor Coding Clarification Protocol
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Tumor Coding Clarification Protocol

**Key Behaviors:**
- Coder must query the provider if documentation is unclear regarding the status of multiple malignant tumors in the same organ.
- Multiple malignant tumors in the same organ may represent different primaries or metastatic disease depending on the site.
- Correct ICD-10-CM codes must be assigned only after clarifying the nature of each tumor.

**Requirements / Properties:**
- Documentation must clearly distinguish between different primaries and metastatic disease for multiple tumors in the same organ.
- Provider query is mandatory when the status of each tumor is unclear.
