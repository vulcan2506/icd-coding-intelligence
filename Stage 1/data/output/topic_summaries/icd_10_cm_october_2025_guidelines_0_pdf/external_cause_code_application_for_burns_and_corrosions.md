# External Cause Code Application for Burns and Corrosions
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** External Cause Code Application for Burns and Corrosions

**Key Behaviors:**
- Use external cause codes with burns and corrosions to identify the source of the injury
- Use external cause codes with burns and corrosions to identify the intent of the burn
- Use external cause codes with burns and corrosions to identify the place where the injury occurred

**Mandatory Coding Criteria:**
- External cause code must identify the source of the burn or corrosion
- External cause code must identify the intent of the burn or corrosion
- External cause code must identify the place where the burn or corrosion occurred
