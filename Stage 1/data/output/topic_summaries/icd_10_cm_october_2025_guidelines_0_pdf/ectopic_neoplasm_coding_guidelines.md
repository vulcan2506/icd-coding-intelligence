# Ectopic Neoplasm Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Ectopic Neoplasm Coding Logic and Workflow

**Key Behaviors:**
- Malignant neoplasms of ectopic tissue are coded to the site of origin rather than the location of the tissue
- Histological terms must be referenced in the Alphabetic Index before consulting the Neoplasm Table
- The Neoplasm Table is used to select the correct column based on the specific type of neoplasm
- The Tabular List must be referenced to verify the code and ensure no more specific site code exists
- Codes for genetic susceptibility to cancer (Z15.0) are referenced for factors influencing health status
- Ectopic pancreatic malignant neoplasms involving the stomach are specifically coded to malignant neoplasm of pancreas, unspecified (C25.9)

**Requirements / Properties:**
- Documentation must clearly indicate the histological term to determine the appropriate Neoplasm Table column
- Verification in the Tabular List is required to confirm the selected code and rule out more specific site codes
- The Alphabetic Index entry for 'adenoma' must be reviewed to check for instructional notes directing to 'see also neoplasm, by site, benign'
