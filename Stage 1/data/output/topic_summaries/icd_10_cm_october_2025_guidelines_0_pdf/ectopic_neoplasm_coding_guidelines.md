# Ectopic Neoplasm Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Ectopic Neoplasm Coding Logic and Verification

**Key Behaviors:**
- Code malignant neoplasms of ectopic tissue to the site of origin (e.g., ectopic pancreatic neoplasm in stomach is coded to pancreas)
- Reference the histological term in the Alphabetic Index before consulting the Neoplasm Table to determine the correct column
- Select the column in the Neoplasm Table that corresponds to the specific type of neoplasm documented
- Verify the selected code in the Tabular List to ensure a more specific site code does not exist
- Reference Section I.C.21 for information regarding Z15.0 codes for genetic susceptibility to cancer

**Mandatory Coding Criteria:**
- Documentation must specify the histological term to guide Neoplasm Table column selection
- Tabular List verification is required to confirm no more specific site code exists
