# Traumatic Coma Scale Coding Implementation
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Traumatic Coma Scale Coding Implementation

**Key Behaviors:**
- Coma scale codes (R40.21- to R40.24-) must be sequenced after the primary diagnosis code(s) and used in conjunction with traumatic brain injury codes.
- The 7th character indicating when the scale was recorded must match across all three codes used to complete the scale.
- At a minimum, report the initial score documented on presentation at the facility (e.g., by EMT or in the emergency department).
- If only the total score is documented, assign code R40.24-; if multiple scores are captured within the first 24 hours, assign only the code for the admission time score.
- Coma scores reported after admission but less than 24 hours later are not classified by ICD-10-CM.

**Requirements / Properties:**
- Codes must be used in conjunction with traumatic brain injury codes and cannot be used with R40.2A (Nontraumatic coma due to underlying condition).
- Documentation must include the initial score recorded at presentation to the facility.
- All three codes used to complete the scale must share the same 7th character indicating the time of recording.
- If multiple scores exist within the first 24 hours, only the score at the time of admission is reportable.
- Clinician documentation for coma scales must adhere to Section I.B.14 guidelines if not provided by the patient's provider.

**Deprecated in this version:**
- Using code R40.2A for traumatic coma scenarios.
- Reporting coma scores that occur after admission but within the first 24 hours as valid ICD-10-CM data.
- Assigning unspecified coma scale codes when specific subcategory codes (R40.21- to R40.24-) are available.
- Placing coma scale codes before the primary diagnosis code(s) in the claim sequence.
- Using mismatched 7th characters across the three codes required to complete the scale.

**New in this version:**
- Explicit requirement to sequence coma scale codes after the diagnosis code(s).
- Mandatory use of the 7th character to indicate the timing of the scale recording.
- Specific instruction to assign R40.24- when only the total score is documented.
- Clarification that only the admission-time score is reportable if multiple scores occur within 24 hours.
- Reference to Section I.B.14 for documentation standards by non-provider clinicians.
