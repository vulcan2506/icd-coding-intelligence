# Traumatic Coma Scale Coding Implementation
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Traumatic Coma Scale Coding Implementation

**Key Behaviors:**
- Coma scale codes (R40.21- to R40.24-) are used in conjunction with traumatic brain injury codes and sequenced after the diagnosis code(s).
- One code from each subcategory is required to complete the scale, with the 7th character indicating when the scale was recorded and matching across all three codes.
- At a minimum, the initial score documented on presentation at the facility (e.g., by EMT or in the ED) must be reported.
- If only the total score is documented, assign code R40.24-; if multiple scores are captured within the first 24 hours, assign only the code for the score at the time of admission.
- ICD-10-CM does not classify coma scores reported after admission but less than 24 hours later.
- Codes are primarily intended for use by trauma registries.
- Codes may be used in any setting where coma scale information is collected.

**Mandatory Coding Criteria:**
- Codes cannot be used with R40.2A (Nontraumatic coma due to underlying condition).
- The 7th character must match for all three codes in the scale.
- The initial score documented on presentation at the facility must be reported as a minimum.
- Only the score at the time of admission is assigned if multiple scores are captured within the first 24 hours.
- Clinician documentation requirements refer to Section I.B.14 for non-provider clinicians.
