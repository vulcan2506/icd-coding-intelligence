# ICD-10 Code Assignment for MSSA/MRSA Colonization
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Code Assignment for MSSA/MRSA Colonization

**Key Behaviors:**
- Assign code Z22.322 for patients documented as having MRSA colonization.
- Assign code Z22.321 for patients documented as having MSSA colonization.
- Recognize colonization as the presence of bacteria without necessarily causing illness.

**Mandatory Coding Criteria:**
- Documentation must explicitly state the patient is a carrier or suspected carrier of the specific organism.
- Provider must document the condition as colonization or carriage rather than an active disease process unless specified otherwise.
