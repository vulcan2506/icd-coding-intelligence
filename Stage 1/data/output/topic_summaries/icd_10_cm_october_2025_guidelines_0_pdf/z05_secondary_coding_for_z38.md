# Z05 Secondary Coding for Z38
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Z05 Secondary Coding for Z38

**Key Behaviors:**
- Assign Z05 codes as secondary codes following Z38 codes
- Z38 codes must be assigned as the primary diagnosis

**Mandatory Coding Criteria:**
- The infant must be liveborn
- The place of birth or type of delivery must be documented to justify the Z38 code
