# Individual Site Code Assignment Logic
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Individual Site Code Assignment Logic

**Key Behaviors:**
- Codes describing specified sites are assigned individually by site when documented
- When specified sites are not documented, assign the appropriate code for 'multiple sites'
- Documentation must explicitly identify the specific site to trigger individual site assignment
- Absence of site documentation defaults to the multiple sites code category
- Individual site assignment prevents grouping unrelated sites into a single unspecified code

**Mandatory Coding Criteria:**
- Clinical documentation must specify the anatomical site for individual code assignment
- If site specification is missing, the multiple sites code must be selected instead
