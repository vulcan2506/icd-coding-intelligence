# Hypertensive Disease Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Hypertensive Disease Coding with Heart Conditions

**Key Behaviors:**
- Assign code from category I11 for hypertension with heart conditions classified to I50.-, I51.4, I51.89, or I51.9, and use additional codes from I50 or I51 to identify the heart condition.
- Assign code from category I11 for hypertension with heart conditions classified to I51.5 or I51.7 without assigning an additional code for the specific heart condition.
- Code hypertension and heart conditions separately if documented as unrelated, sequencing based on admission circumstances.
- Assign appropriate codes from categories I10-I15 for existing states of hypertension under control by therapy.
- Assign appropriate codes from categories I10-I15 for uncontrolled hypertension, whether untreated or not responding to therapy.

**Requirements / Properties:**
- Provider must document the relationship between hypertension and heart conditions to determine if they are related or unrelated.
- Documentation must specify if hypertension is under control, uncontrolled, untreated, or not responding to therapy.
- Specific heart condition codes (I50.-, I51.4-I51.9) must be identified to determine if an additional code is required alongside I11.

**Deprecated in this version:**
- Assigning an additional code from category I50 or I51 when the heart condition is classified as I51.5 (Myocardial degeneration) or I51.7 (Cardiomegaly) in the presence of hypertension.
- Coding hypertension and heart conditions as a single code when the provider has documented them as unrelated.

**New in this version:**
- Explicit instruction to sequence codes based on the circumstances of the admission or encounter when coding unrelated hypertension and heart conditions.
