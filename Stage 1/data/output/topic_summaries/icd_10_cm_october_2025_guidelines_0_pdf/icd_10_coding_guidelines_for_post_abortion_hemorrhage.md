# ICD-10 Coding Guidelines for Post-Abortion Hemorrhage
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Coding Guidelines for Post-Abortion Hemorrhage

**Key Behaviors:**
- Assign code O04.6 for delayed or excessive hemorrhage following induced termination of pregnancy
- Do not assign code O72.1 for post-abortion conditions as it is restricted to postpartum scenarios

**Requirements / Properties:**
- The condition must be hemorrhage occurring after an elective abortion
- The hemorrhage must be classified as delayed or excessive
- The procedure must be an induced termination of pregnancy

**Deprecated in this version:**
- Assigning code O72.1 (Other immediate postpartum hemorrhage) for post-abortion hemorrhage cases

**New in this version:**
- Explicit exclusion of O72.1 for non-postpartum abortion-related hemorrhage
