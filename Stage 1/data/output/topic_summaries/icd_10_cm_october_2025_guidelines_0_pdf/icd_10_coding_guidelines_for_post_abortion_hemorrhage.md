# ICD-10 Coding Guidelines for Post-Abortion Hemorrhage
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Coding of Post-Abortion Hemorrhage

**Key Behaviors:**
- Assign code O04.6 for delayed or excessive hemorrhage following induced termination of pregnancy.
- Do not assign code O72.1 for post-abortion conditions.

**Mandatory Coding Criteria:**
- Condition must be hemorrhage following elective abortion.
- Hemorrhage must be delayed or excessive.

**Superseded Guidance:**
- Code O72.1 for post-abortion conditions
