# Alphabetic Index Instruction Logic
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Alphabetic Index Instruction Logic

**Key Behaviors:**
- The 'see' instruction following a main term indicates that another term must be referenced to locate the correct code.
- The 'see also' instruction indicates an additional main term that may provide useful entries but is not mandatory if the original term provides the necessary code.
- It is necessary to follow the 'see' note to the referenced main term to find the correct code.
- It is not necessary to follow the 'see also' note when the original main term provides the necessary code.
- The 'see also' instruction suggests alternative main terms that may offer additional Alphabetic Index entries.

**Requirements / Properties:**
- When a 'see' instruction is present, the coder must navigate to the referenced main term to locate the code.
- When a 'see also' instruction is present, the coder may consult the referenced term for additional entries if needed.
