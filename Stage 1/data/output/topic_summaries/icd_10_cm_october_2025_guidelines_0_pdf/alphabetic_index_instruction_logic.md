# Alphabetic Index Instruction Logic
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Alphabetic Index Instruction Logic

**Key Behaviors:**
- 'see' instruction indicates another term must be referenced to locate the correct code.
- 'see also' instruction indicates an additional main term may provide useful entries but is not mandatory if the original term provides the necessary code.

**Mandatory Coding Criteria:**
- When a 'see' instruction is present, the coder must navigate to the referenced main term to find the code.
- When a 'see also' instruction is present, the coder may skip it if the original main term yields the required code.
