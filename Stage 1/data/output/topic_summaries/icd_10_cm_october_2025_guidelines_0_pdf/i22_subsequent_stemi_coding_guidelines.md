# I22 Subsequent STEMI Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** I22 Subsequent STEMI Coding Logic and Sequencing

**Key Behaviors:**
- Assign category I22 code only when a new AMI occurs within 4 weeks of an initial type 1 or unspecified AMI
- Category I22 code must always be used in conjunction with a category I21 code
- Do not assign I22 for subsequent type 2, type 4, or type 5 AMIs; assign only I21.A1 or I21.A9 respectively
- If subsequent AMI is of a different type than the initial AMI, assign appropriate I21 codes for each type and do not assign I22
- Sequencing of I22 and I21 codes depends on the specific circumstances of the encounter

**Mandatory Coding Criteria:**
- Documentation must confirm the initial AMI was type 1 or unspecified to qualify for I22 usage
- Encounter must occur within the 4-week timeframe following the initial AMI
- Both initial and subsequent AMIs must be type 1 or unspecified to utilize category I22
