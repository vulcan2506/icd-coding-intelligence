# I22 Subsequent STEMI Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** I22 Subsequent STEMI Coding Logic

**Key Behaviors:**
- Assign category I22 only when a new AMI occurs within 4 weeks of an initial type 1 or unspecified AMI
- I22 codes must be used in conjunction with a code from category I21
- Do not assign I22 for subsequent type 2, type 4, or type 5 AMIs; assign I21.A1 or I21.A9 instead
- If subsequent AMI types differ from the initial AMI, assign appropriate I21 codes for each type and exclude I22
- Sequencing of I22 and I21 codes depends on the specific circumstances of the encounter

**Deprecated in this version:**
- Assigning code I22 for subsequent type 2, type 4, or type 5 myocardial infarctions
- Assigning code I22 when the initial and subsequent AMI types differ
