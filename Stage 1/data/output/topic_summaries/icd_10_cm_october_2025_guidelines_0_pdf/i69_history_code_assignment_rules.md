# I69 History Code Assignment Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** I69 History Code Assignment Logic

**Key Behaviors:**
- Codes from category I69 are excluded from assignment when the patient lacks neurologic deficits
- Assignment of I69 codes is contingent upon the presence of documented neurologic deficits
- The system enforces strict adherence to Section I.C.21.4 guidelines for personal history codes

**Requirements / Properties:**
- Clinical documentation must explicitly confirm neurologic deficits to trigger I69 code assignment
