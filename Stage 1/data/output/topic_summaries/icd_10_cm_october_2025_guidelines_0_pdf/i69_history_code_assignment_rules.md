# I69 History Code Assignment Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** I69 History Code Assignment Constraints

**Key Behaviors:**
- Codes from category I69 must not be assigned if the patient does not have neurologic deficits
- Assignment of I69 codes requires documented evidence of neurologic deficits
- Use of personal history codes must follow Section I.C.21.4 guidelines

**Mandatory Coding Criteria:**
- Documentation must confirm the presence of neurologic deficits before assigning I69 codes
- Coding must adhere to Section I.C.21.4 rules for personal history codes
