# ICD-10-CM Coding Syntax Definitions
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10-CM Coding Syntax Definitions

**Key Behaviors:**
- Brackets enclose synonyms, alternative wording, or explanatory phrases in the Tabular List and identify manifestation codes in the Alphabetic Index.
- Parentheses enclose nonessential modifiers that may be present or absent without affecting the code number, except when mutually exclusive with subentries.
- Colons indicate incomplete terms in the Tabular List requiring additional modifiers to become assignable to a category.
- 'Not elsewhere classifiable' represents 'other specified' for conditions where a specific code is unavailable.
- 'Not otherwise specified' is the equivalent of unspecified codes.
- Commas in the Alphabetic Index denote alternate verbiage, essential or nonessential modifiers, or alternatives for 'and/or'.

**Requirements / Properties:**
- Brackets must be used to identify manifestation codes in the Alphabetic Index.
- Nonessential modifiers in parentheses apply to subterms following a main term unless mutually exclusive.
- Incomplete terms following a colon must be completed with appropriate modifiers before assignment.
- Conditions without a specific code must use the 'Not elsewhere classifiable' (NEC) entry.
- Unspecified conditions must use the 'Not otherwise specified' equivalent.
