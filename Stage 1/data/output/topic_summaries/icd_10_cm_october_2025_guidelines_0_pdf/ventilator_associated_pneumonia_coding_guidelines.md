# Ventilator-Associated Pneumonia Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Ventilator-Associated Pneumonia Coding Guidelines

**Key Behaviors:**
- Code assignment is based on the provider's documentation of the relationship between the condition and the procedure.
- Code J95.851 should be assigned only when the provider has documented ventilator associated pneumonia (VAP).
- An additional code to identify the organism should also be assigned.
- Do not assign an additional code from categories J12-J18 to identify the type of pneumonia.
- If documentation is unclear regarding whether pneumonia is attributable to the mechanical ventilator, query the provider.

**Requirements / Properties:**
- Provider must specifically state that the pneumonia is ventilator-associated pneumonia.
- An additional code identifying the organism must be assigned.
- Documentation must establish the relationship between the condition and the procedure.

**Deprecated in this version:**
- Assigning an additional code from categories J12-J18 to identify the type of pneumonia.
- Assigning code J95.851 without specific provider documentation of VAP.

**New in this version:**
- Code J95.851 for Ventilator associated pneumonia.
- Mandatory assignment of an additional code to identify the organism (e.g., B96.5).
