# Ventilator-Associated Pneumonia Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Ventilator-Associated Pneumonia Coding

**Key Behaviors:**
- Code assignment is based on the provider's documentation of the relationship between the condition and the procedure.
- An additional code to identify the organism should be assigned alongside J95.851.
- Do not assign an additional code from categories J12-J18 to identify the type of pneumonia.
- Query the provider if documentation is unclear regarding whether pneumonia is attributable to the mechanical ventilator.
- An additional code to identify the specific organism causing the pneumonia must be assigned alongside code J95.851.

**Mandatory Coding Criteria:**
- Provider must specifically document ventilator associated pneumonia (VAP).
- Do not assign additional codes from categories J12-J18.
- Code J95.851 should not be assigned if the provider has not specifically stated the pneumonia is ventilator-associated.
