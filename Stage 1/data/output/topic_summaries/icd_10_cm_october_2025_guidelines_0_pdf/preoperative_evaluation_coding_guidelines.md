# Preoperative Evaluation Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Preoperative Evaluation Coding Guidelines

**Key Behaviors:**
- Sequence first a code from subcategory Z01.81 to describe pre-op consultations for patients receiving preoperative evaluations only.
- Assign a code for the condition to describe the reason for the surgery as an additional diagnosis.
- Code any findings related to the pre-op evaluation.

**Requirements / Properties:**
- Documentation must confirm the patient is receiving preoperative evaluations only.
- The reason for the surgery must be explicitly documented to assign an additional diagnosis code.
- Findings related to the pre-op evaluation must be documented to be coded.
