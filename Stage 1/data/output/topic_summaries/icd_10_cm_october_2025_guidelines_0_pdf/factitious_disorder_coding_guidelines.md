# Factitious Disorder Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Factitious Disorder Coding Guidelines

**Key Behaviors:**
- Assign code F68.1- for patients with documented factitious disorder imposed on self or Munchausen's syndrome.
- Assign code F68.A to the perpetrator's record for Munchausen's syndrome by proxy (MSBP) or factitious disorder imposed on another.
- Assign codes from categories T74 or T76 to the victim of MSBP based on whether the abuse is confirmed or suspected.

**Mandatory Coding Criteria:**
- Diagnosis must be documented as factitious disorder imposed on self or Munchausen's syndrome for F68.1- assignment.
- Diagnosis must be documented as factitious disorder imposed on another for F68.A assignment to the perpetrator.
- Abuse status (confirmed or suspected) must be determined to select between T74 and T76 categories for the victim.
