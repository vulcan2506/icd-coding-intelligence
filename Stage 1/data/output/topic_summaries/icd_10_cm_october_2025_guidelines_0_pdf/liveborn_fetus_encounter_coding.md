# Liveborn Fetus Encounter Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Liveborn Fetus Encounter Coding

**Key Behaviors:**
- Assign code Z33.2 for encounters involving an attempted termination of pregnancy that results in a liveborn fetus.
- Assign a code from category Z37 to represent the outcome of delivery for the liveborn fetus.

**Requirements / Properties:**
- Documentation must confirm the procedure was an attempted termination of pregnancy.
- Documentation must confirm the fetus was liveborn.
- Both the encounter code (Z33.2) and the outcome code (Z37) must be assigned.
