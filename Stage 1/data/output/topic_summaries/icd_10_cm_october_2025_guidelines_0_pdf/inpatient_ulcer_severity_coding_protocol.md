# Inpatient Ulcer Severity Coding Protocol
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Inpatient Ulcer Severity Coding Protocol

**Key Behaviors:**
- Assign two separate codes for non-pressure ulcers that progress to a higher severity level during an inpatient stay.
- The first code represents the site and severity level of the ulcer on admission.
- The second code represents the same ulcer site and the highest severity level reported during the stay.

**Requirements / Properties:**
- Documentation must confirm the patient was admitted with a non-pressure ulcer at a specific severity level.
- Documentation must confirm the ulcer progressed to a higher severity level during the inpatient stay.
- Clinicians other than the patient's provider must follow Section I.B.14. for pressure ulcer stage documentation.
