# Hypertension Comorbidity Coding Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Hypertension Comorbidity Coding Rules

**Key Behaviors:**
- Presume a causal relationship between hypertension and heart/kidney involvement if linked by the term 'with' in the Alphabetic Index, even without explicit provider documentation.
- Code hypertension and related conditions as related unless documentation explicitly states they are unrelated.
- Require explicit provider documentation linking conditions to code them as related when no relational terms (e.g., 'with', 'associated with') are present in the classification.

**Mandatory Coding Criteria:**
- Documentation must explicitly link conditions if the classification does not use relational terms like 'with' or 'due to'.
- Absence of explicit provider documentation does not preclude coding related conditions if the Alphabetic Index links them via 'with'.
- Explicit documentation stating conditions are unrelated overrides the presumption of a causal relationship.
