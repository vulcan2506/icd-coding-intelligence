# Gestational Diabetes Code Subcategory O24.4
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Gestational Diabetes Coding (O24.4)

**Key Behaviors:**
- Codes for gestational diabetes are restricted to subcategory O24.4.
- No other code from category O24 should be used with a code from O24.4.
- If treated with both diet and insulin, only the code for insulin-controlled is required.
- If treated with both diet and oral hypoglycemic medications, only the code for oral hypoglycemic drugs is required.
- Codes Z79.4, Z79.84, and Z79.85 should not be assigned with codes from subcategory O24.4.

**Mandatory Coding Criteria:**
- Patient must not have been diabetic prior to pregnancy.
- Condition must occur during the second or third trimester.
- Treatment modality determines the specific code (diet, insulin, or oral hypoglycemic).
- Abnormal glucose tolerance in pregnancy requires code from subcategory O99.81, not O24.4.
