# Insulin Pump Failure Coding Protocol
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Insulin Pump Failure Coding Protocol

**Key Behaviors:**
- Assign code from subcategory T85.6 as the principal or first-listed code for underdose due to insulin pump failure, followed by code T38.3X6-.
- Assign code from subcategory T85.6 as the principal or first-listed code for overdose due to insulin pump malfunction, followed by code T38.3X1-.
- Additional codes must be assigned for the type of diabetes mellitus and any associated complications resulting from the underdosing or overdosing.

**Requirements / Properties:**
- Documentation must specify the type of pump malfunction to select the correct code from subcategory T85.6.
- The encounter must be due to an insulin pump failure resulting in either an underdose or an overdose of insulin.
- Codes for diabetes mellitus and associated complications must be included to fully capture the clinical picture.
