# Insulin Pump Failure Coding Protocol
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Insulin Pump Failure Coding Protocol

**Key Behaviors:**
- Assign code from subcategory T85.6 as the principal or first-listed code for underdose due to pump failure, followed by code T38.3X6-.
- Assign code from subcategory T85.6 as the principal or first-listed code for overdose due to pump malfunction, followed by code T38.3X1-.
- Additional codes must be assigned for the type of diabetes mellitus and any associated complications resulting from the underdosing.

**Mandatory Coding Criteria:**
- The code from subcategory T85.6 must specify the type of pump malfunction.
- The provider must document the underdose or overdose resulting from the pump failure.
- Codes for diabetes mellitus and associated complications must be assigned in addition to the principal codes.
