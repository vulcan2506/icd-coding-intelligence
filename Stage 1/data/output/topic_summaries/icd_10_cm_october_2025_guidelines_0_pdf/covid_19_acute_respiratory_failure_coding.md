# COVID-19 Acute Respiratory Failure Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** COVID-19 Acute Respiratory Failure Coding

**Key Behaviors:**
- Assign code U07.1 as the principal/first-listed diagnosis when the encounter reason is a respiratory manifestation of COVID-19
- Assign code(s) for the specific respiratory manifestation(s) as additional diagnoses
- For acute respiratory failure due to COVID-19, assign code U07.1 alongside code J96.0-

**Requirements / Properties:**
- The reason for the encounter or admission must be explicitly a respiratory manifestation of COVID-19
- Code U07.1 must be assigned as the principal diagnosis for COVID-19 respiratory manifestations
- Code J96.0- must be used for acute respiratory failure in conjunction with U07.1

**New in this version:**
- Requirement to assign U07.1 as the principal diagnosis for COVID-19 respiratory manifestations
- Mandatory pairing of U07.1 with J96.0- for acute respiratory failure cases
