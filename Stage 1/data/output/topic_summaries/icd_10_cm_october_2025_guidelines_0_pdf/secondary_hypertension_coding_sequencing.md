# Secondary Hypertension Coding Sequencing
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Secondary Hypertension Coding Sequencing

**Key Behaviors:**
- Two codes are required: one for the underlying etiology and one from category I15 for hypertension
- Sequencing is determined by the reason for admission or encounter
- The code for the underlying etiology identifies the cause of the secondary hypertension
- The code from category I15 identifies the hypertension condition itself
- Documentation must specify the underlying condition to support the etiology code

**Mandatory Coding Criteria:**
- Documentation must confirm the underlying etiology of the hypertension
- Reason for admission or encounter must be reviewed to determine code order
