# Postpancreatectomy Diabetes Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Postpancreatectomy Diabetes Coding

**Key Behaviors:**
- Assign code E89.1 for postprocedural hypoinsulinemia resulting from surgical removal of all or part of the pancreas.
- Assign a code from category E13 as the principal or first-listed diagnosis.
- Assign a code from subcategory Z90.41, Acquired absence of pancreas, as an additional code.

**Requirements / Properties:**
- Clinical documentation must confirm the lack of insulin is due to the surgical removal of all or part of the pancreas.
- The diagnosis must be postpancreatectomy diabetes mellitus to utilize these specific codes.
