# Postpancreatectomy Diabetes Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Postpancreatectomy Diabetes Coding

**Key Behaviors:**
- Assign code E89.1 for postpancreatectomy diabetes mellitus
- Assign a code from category E13 as the principal or first-listed diagnosis
- Assign a code from subcategory Z90.41 as an additional code

**Mandatory Coding Criteria:**
- Diagnosis must be lack of insulin due to surgical removal of all or part of the pancreas
- Code E13 must be the principal or first-listed diagnosis
- Code Z90.41 must be assigned as an additional code
