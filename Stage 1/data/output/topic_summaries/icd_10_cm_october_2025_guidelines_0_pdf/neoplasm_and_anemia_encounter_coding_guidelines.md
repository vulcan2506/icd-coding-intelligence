# Neoplasm and Anemia Encounter Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Neoplasm and Anemia Encounter Sequencing

**Key Behaviors:**
- When admission is for management of anemia associated with malignancy and treatment is only for anemia, sequence malignancy code first followed by anemia code D63.0.
- When admission is for management of a complication associated with neoplasm (e.g., dehydration) and treatment is only for the complication, sequence complication code first followed by neoplasm code(s).
- Anemia associated with malignancy is treated as an exception to the general complication sequencing rule, requiring malignancy to be sequenced first.

**Mandatory Coding Criteria:**
- Treatment must be limited to anemia management for the malignancy-first sequencing rule to apply.
- Treatment must be limited to the complication management for the complication-first sequencing rule to apply.
- The anemia code must be D63.0 (Anemia in neoplastic disease) when associated with malignancy.
