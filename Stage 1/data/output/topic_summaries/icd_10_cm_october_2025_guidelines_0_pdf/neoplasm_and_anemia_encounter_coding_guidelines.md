# Neoplasm and Anemia Encounter Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Neoplasm and Anemia Encounter Sequencing

**Key Behaviors:**
- When admission is for management of anemia associated with malignancy and treatment is only for anemia, sequence malignancy as principal diagnosis followed by anemia code D63.0.
- When admission is for management of a complication associated with neoplasm (e.g., dehydration) and treatment is only for the complication, sequence the complication first followed by the neoplasm code.
- Anemia associated with malignancy is treated as an exception to the general complication sequencing rule, requiring the malignancy to be sequenced first.

**Requirements / Properties:**
- Treatment must be limited to anemia management for the malignancy-to-be-first rule to apply.
- Treatment must be limited to the complication management for the complication-to-be-first rule to apply.
- Clinical documentation must explicitly link the anemia to the neoplastic disease.

**Deprecated in this version:**
- Sequencing the anemia code as the principal diagnosis when the encounter is for anemia associated with malignancy.
- Sequencing the neoplasm code as the principal diagnosis when the encounter is for a complication like dehydration.

**New in this version:**
- Mandatory use of code D63.0 for anemia in neoplastic disease when sequencing malignancy first.
- Explicit exception rule for anemia overriding standard complication sequencing guidelines.
