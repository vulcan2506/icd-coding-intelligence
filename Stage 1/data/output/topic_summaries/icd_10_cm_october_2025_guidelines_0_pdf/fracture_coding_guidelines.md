# Fracture Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Fracture Coding Guidelines

**Key Behaviors:**
- Fractures of specified sites are coded individually by site in accordance with category provisions and medical record detail.
- Fractures not indicated as open or closed should be coded to closed.
- Fractures not indicated whether displaced or not displaced should be coded to displaced.
- Traumatic fractures are coded using the appropriate 7th character for initial encounter (A, B, C) for each encounter where the patient is receiving active treatment.
- Care for complications of surgical treatment for fracture repairs during the healing or recovery phase should be coded with the appropriate complication codes.
- Fractures of specified sites are coded individually by site in accordance with both the provisions within categories and the level of detail furnished by medical record content.
- Fractures are coded using the appropriate 7th character for subsequent care for encounters after the patient has completed active treatment of the fracture and is receiving routine care for the fracture during the healing or recovery phase.

**Mandatory Coding Criteria:**
- Fractures must be coded individually by site in accordance with provisions within categories S02, S12, S22, S32, S42, S49, S52, S59, S62, S72, S79, S82, S89, S92.
- A fracture not indicated as open or closed should be coded to closed.
- A fracture not indicated whether displaced or not displaced should be coded to displaced.
- Traumatic fractures must use the appropriate 7th character for initial encounter (A, B, C) for each encounter where the patient is receiving active treatment.
- Aftercare Z codes should not be used for aftercare for traumatic fractures; instead, assign the acute fracture code with the appropriate 7th character.
- A code from category M80 must be used instead of a traumatic fracture code for patients with known osteoporosis who suffer a fracture from minor trauma that would not typically break a healthy bone.
- For physeal fractures, only the code identifying the type of physeal fracture is required; a separate code for the specific bone is prohibited.
