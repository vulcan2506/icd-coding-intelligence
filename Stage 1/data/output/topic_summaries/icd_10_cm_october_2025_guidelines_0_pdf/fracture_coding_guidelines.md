# Fracture Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Fracture Coding Guidelines

**Key Behaviors:**
- Fractures of specified sites are coded individually by site in accordance with category provisions and medical record detail.
- Fractures not indicated as open or closed should be coded to closed.
- Fractures not indicated whether displaced or not displaced should be coded to displaced.
- Traumatic fractures are coded using the appropriate 7th character for initial encounter (A, B, C) for each encounter where the patient is receiving active treatment.
- Care for complications of surgical treatment for fracture repairs during the healing or recovery phase should be coded with the appropriate complication codes.

**Requirements / Properties:**
- Fractures must be coded individually by site in accordance with provisions within categories S02, S12, S22, S32, S42, S49, S52, S59, S62, S72, S79, S82, S89, S92.
- The 7th character for initial encounter must be assigned for a patient who delayed seeking treatment for the fracture or nonunion.
- For open fractures of the forearm, femur, and lower leg, the Gustilo classification type must be used to determine the 7th character.
- A code from category M80 must be used for any patient with known osteoporosis who suffers a fracture, even if the patient had a minor fall or trauma.
- Aftercare Z codes should not be used for aftercare for traumatic fractures; instead, assign the acute fracture code with the appropriate 7th character.

**Deprecated in this version:**
- Using unspecified laterality codes when documentation does not specify the side.
- Using aftercare Z codes for aftercare of traumatic fractures.
- Assigning a separate code to identify the specific bone fractured for physeal fractures.
- Coding a fracture not indicated as open or closed to open.
- Coding a fracture not indicated whether displaced or not displaced to nondisplaced.

**New in this version:**
- The appropriate 7th character for subsequent care with nonunion (K, M, N) or subsequent care with malunion (P, Q, R) should be used for care of complications of fractures.
- The 7th character for initial encounter should also be assigned for a patient who delayed seeking treatment for the fracture or nonunion.
- When the Gustilo classification type is not specified for an open fracture, the 7th character for open fracture type I or II should be assigned (B, E, H, M, Q).
- For physeal fractures, only the code identifying the type of physeal fracture should be assigned.
- A code from category M80 should be used for any patient with known osteoporosis who suffers a fracture, even if the patient had a minor fall or trauma.
