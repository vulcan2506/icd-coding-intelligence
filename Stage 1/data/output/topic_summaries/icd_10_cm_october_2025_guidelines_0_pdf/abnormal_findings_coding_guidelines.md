# Abnormal Findings Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Abnormal Findings Coding Guidelines

**Key Behaviors:**
- Abnormal findings are not coded unless the provider indicates their clinical significance.
- Findings outside the normal range require provider confirmation of clinical significance before coding.
- Providers should be queried if abnormal findings are associated with ordered tests or prescribed treatment.
- Coding practices differ between inpatient abnormal findings and outpatient diagnostic test interpretations.
- Abnormal findings must be linked to a specific clinical condition or treatment plan to be reportable.

**Requirements / Properties:**
- Provider must explicitly indicate the clinical significance of the abnormal finding.
- Documentation must show the finding is outside the normal range.
- Evidence of provider-ordered tests or prescribed treatment related to the finding is required.
- Clinical context must distinguish the finding from routine diagnostic test interpretations.
- The abnormal finding must be integral to the patient's care plan or diagnosis.

**Deprecated in this version:**
- Coding abnormal findings without explicit provider indication of clinical significance.
- Assigning codes to abnormal findings solely based on laboratory or x-ray results.
- Treating all abnormal findings as reportable conditions regardless of clinical context.
- Using unspecified codes for abnormal findings without provider clarification.
- Coding abnormal findings in outpatient settings using inpatient guidelines.

**New in this version:**
- Mandatory provider query process for abnormal findings lacking clinical significance documentation.
- Updated definitions for 'clinical significance' in the context of diagnostic results.
- Enhanced documentation requirements for linking abnormal findings to treatment plans.
- New guidelines distinguishing inpatient vs. outpatient coding for diagnostic interpretations.
- Additional data elements required to support the coding of abnormal findings.
