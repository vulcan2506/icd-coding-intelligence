# Y38 Terrorism Secondary Effects Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Y38 Terrorism Secondary Effects Coding

**Key Behaviors:**
- Assign first-listed external cause code from category Y38 when the Federal Government (FBI) identifies the cause of injury as terrorism.
- Use additional code for place of occurrence (Y92.-) when applicable.
- Assign more than one Y38 code if the injury results from more than one mechanism of terrorism.
- Assign code Y38.9 for conditions occurring subsequent to the terrorist event.
- Do not assign Y38.9 for conditions that are due to the initial terrorist act.

**Requirements / Properties:**
- The cause of injury must be identified by the Federal Government (FBI) as terrorism.
- The definition of terrorism must align with the inclusion note at the beginning of category Y38.
- Y38.9 must only be used for secondary effects occurring after the initial event.

**New in this version:**
- It is acceptable to assign code Y38.9 with another code from Y38 if there is an injury due to the initial terrorist event and an injury that is a subsequent result of the terrorist event.
