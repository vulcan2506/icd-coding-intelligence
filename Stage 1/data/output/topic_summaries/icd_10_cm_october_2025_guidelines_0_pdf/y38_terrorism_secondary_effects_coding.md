# Y38 Terrorism Secondary Effects Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Y38 Terrorism Secondary Effects Coding

**Key Behaviors:**
- Assign first-listed external cause code from category Y38 when the Federal Government (FBI) identifies the cause of injury as terrorism.
- Use additional code Y92.- for place of occurrence.
- Assign multiple Y38 codes if the injury results from more than one mechanism of terrorism.
- Assign code Y38.9 for conditions occurring subsequent to the terrorist event.
- Do not assign Y38.9 for conditions due to the initial terrorist act.

**Mandatory Coding Criteria:**
- Cause of injury must be identified by the Federal Government (FBI) as terrorism.
- Code Y38.9 must not be used for conditions resulting from the initial terrorist act.
- Code Y38.9 may be assigned with another Y38 code if there is an injury from the initial event and a subsequent result.
