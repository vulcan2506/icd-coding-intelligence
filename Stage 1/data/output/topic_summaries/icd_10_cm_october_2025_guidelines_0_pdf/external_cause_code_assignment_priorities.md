# External Cause Code Assignment Priorities
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** External Cause Code Assignment Priorities

**Key Behaviors:**
- Assign separate external cause codes for each distinct event causing separate injuries
- Prioritize child and adult abuse codes over all other external cause codes
- Prioritize terrorism event codes over all codes except child and adult abuse
- Prioritize cataclysmic event codes over all codes except abuse, terrorism, and transport accidents
- Prioritize transport accident codes over all codes except abuse, terrorism, and cataclysmic events

**Requirements / Properties:**
- Activity and external cause status codes must be assigned after all causal (intent) external cause codes
- The first-listed code must correspond to the cause of the most serious diagnosis (assault, accident, or self-harm)
