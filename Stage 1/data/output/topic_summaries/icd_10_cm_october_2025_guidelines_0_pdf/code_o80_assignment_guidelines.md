# Code O80 Assignment Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Code O80 Assignment Guidelines

**Key Behaviors:**
- Assign Code O80 for full-term uncomplicated delivery with a single healthy infant and no complications during the episode.
- Always assign Code O80 as the principal diagnosis for uncomplicated deliveries.
- Do not use Code O80 if any other Chapter 15 code is needed to describe a current complication of the antenatal, delivery, or postnatal period.
- Use additional codes from other chapters with Code O80 only if they are not related to or complicating the pregnancy.
- Assign Code O80 if a prior antepartum complication has resolved before the admission for delivery.

**Requirements / Properties:**
- Patient must be admitted for a full-term normal delivery.
- Delivery must result in a single, healthy infant.
- No complications must be present antepartum, during delivery, or postpartum during the delivery episode.
- No current complications requiring Chapter 15 codes must exist.
- If a prior complication existed, it must be resolved at the time of admission for delivery.
