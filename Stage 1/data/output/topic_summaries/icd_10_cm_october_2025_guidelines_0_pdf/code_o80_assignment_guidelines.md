# Code O80 Assignment Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Code O80 Assignment Guidelines

**Key Behaviors:**
- Assign Code O80 for full-term uncomplicated delivery with a single healthy infant and no complications.
- Code O80 is always a principal diagnosis.
- Do not use Code O80 if any other Chapter 15 code is needed to describe a current complication.
- Additional codes from other chapters may be used with Code O80 if not related to or complicating the pregnancy.
- Code O80 may be used if a prior antepartum complication is resolved before delivery admission.

**Mandatory Coding Criteria:**
- Patient must be admitted for a full-term normal delivery.
- Infant must be single and healthy.
- No complications must exist antepartum, during delivery, or postpartum during the delivery episode.
- No current complications requiring Chapter 15 codes.
- Any additional codes must not be related to or complicating the pregnancy.
