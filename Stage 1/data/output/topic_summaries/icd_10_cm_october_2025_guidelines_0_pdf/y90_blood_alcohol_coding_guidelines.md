# Y90 Blood Alcohol Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Y90 Blood Alcohol Coding Guidelines

**Key Behaviors:**
- Assign Y90 code when blood alcohol level is documented and provider documents a condition classifiable to category F10
- Y90 code may be assigned even if the blood alcohol level is not documented by the patient's provider
- Refer to Section I.B.14 for documentation by clinicians other than the patient's provider

**Mandatory Coding Criteria:**
- Information must be documented regarding blood alcohol level
- Patient's provider must document a condition classifiable to category F10 (Alcohol related disorders)
