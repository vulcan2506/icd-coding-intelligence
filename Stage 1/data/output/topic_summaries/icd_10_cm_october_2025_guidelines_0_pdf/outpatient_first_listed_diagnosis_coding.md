# Outpatient First-Listed Diagnosis Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Outpatient First-Listed Diagnosis Coding

**Key Behaviors:**
- Use 'first-listed diagnosis' terminology instead of 'principal diagnosis' in outpatient settings.
- ICD-10-CM coding conventions and disease-specific guidelines take precedence over outpatient guidelines.
- Begin code assignment search through the Alphabetic Index, never starting in the Tabular List.
- Sequence the diagnosis chiefly responsible for services first; code other diagnoses (e.g., chronic conditions) as additional diagnoses.
- Code confirmed or definitive diagnoses from physician-interpreted reports available at coding time; do not code related signs and symptoms as additional diagnoses.
- Recognize that diagnoses may not be established at the initial encounter and may require two or more visits for confirmation.
- Differentiate coding practice for outpatient encounters from the hospital inpatient setting regarding abnormal findings on test results.

**Mandatory Coding Criteria:**
- Diagnosis must be documented as chiefly responsible for the outpatient services provided during the encounter.
- Final report for diagnostic tests must be available at the time of coding to assign confirmed diagnoses.
- Routine laboratory/radiology testing without associated signs, symptoms, or diagnosis requires assignment of Z01.89.
- If routine testing occurs with a non-routine test evaluating a sign/symptom, assign both the Z code and the code for the non-routine test reason.
- Do not code related signs and symptoms as additional diagnoses when a confirmed diagnosis is documented in an interpreted report.
