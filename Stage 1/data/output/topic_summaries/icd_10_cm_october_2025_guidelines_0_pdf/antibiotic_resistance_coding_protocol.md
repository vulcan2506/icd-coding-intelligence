# Antibiotic Resistance Coding Protocol
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Antibiotic Resistance Coding Protocol

**Key Behaviors:**
- Identify all infections documented as antibiotic resistant
- Assign a code from category Z16, Resistance to antimicrobial drugs, following the infection code only if the infection code does not identify drug resistance

**Requirements / Properties:**
- Infection must be documented as antibiotic resistant
- Infection code must not already identify drug resistance

**New in this version:**
- Category Z16 codes for resistance to antimicrobial drugs
