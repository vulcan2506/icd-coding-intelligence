# Multiple Code Sequencing Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Multiple Code Sequencing Rules

**Key Behaviors:**
- 'Use additional code' notes indicate that a secondary code should be added if known to fully describe a condition.
- When a 'code first' note is present and an underlying condition exists, the underlying condition must be sequenced first.
- 'Code, if applicable, any causal condition first' notes allow the current code to be principal only if the causal condition is unknown or not applicable.
- For bacterial infections not in Chapter 1, secondary codes from categories B95 or B96 may be required to identify the organism.
- Multiple codes are required for sequela, complication codes, and obstetric codes to fully describe the condition.
- The etiology/manifestation convention requires two codes to fully describe a single condition affecting multiple body systems.
- 'Use additional code' notes appear at codes not part of an etiology/manifestation pair where a secondary code is useful to fully describe a condition.

**Requirements / Properties:**
- A secondary code must be added if known when a 'use additional code' note is present.
- The underlying condition must be sequenced first if known when a 'code first' note is present.
- The code for a known causal condition must be sequenced as the principal diagnosis if applicable.
- Secondary codes from categories B95 or B96 are required for bacterial infections not included in Chapter 1.
- Specific guidelines must be consulted for sequencing sequela, complication, and obstetric codes.
