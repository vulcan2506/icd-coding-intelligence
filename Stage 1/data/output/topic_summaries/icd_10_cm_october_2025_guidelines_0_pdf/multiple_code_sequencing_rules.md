# Multiple Code Sequencing Rules
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Multiple Code Sequencing Rules

**Key Behaviors:**
- 'Use additional code' notes indicate a secondary code should be added if known to fully describe a condition.
- 'Code first' notes require the underlying condition to be sequenced first if present.
- 'Code, if applicable, any causal condition first' notes allow the current code as principal diagnosis only if the causal condition is unknown or not applicable.
- Secondary codes from categories B95 or B96 may be required to identify bacterial organisms causing infections classified elsewhere.
- Multiple codes are required for sequela, complication codes, and obstetric codes to fully describe a condition.

**Mandatory Coding Criteria:**
- The underlying condition must be known and present for 'code first' notes to apply.
- The causal condition must be known to sequence the specific condition code as principal; otherwise, the causal condition code is sequenced first.
- The secondary code must be known to be added when a 'use additional code' note is present.
- The infectious disease code must be present to identify the need for an organism code from categories B95 or B96.
- The condition must affect multiple body systems or require more than one code to be fully described.
