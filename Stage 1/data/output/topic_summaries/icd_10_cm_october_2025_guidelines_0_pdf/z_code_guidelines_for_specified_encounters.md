# Z Code Guidelines for Specified Encounters
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Z Code Guidelines for Specified Encounters

**Key Behaviors:**
- Z codes are used to identify the reason for encounters that are not related to a disease or injury
- Z codes should be assigned when the encounter is for specified circumstances such as health checkups or screenings
- Z codes may be used as the principal diagnosis when the encounter is solely for the specified circumstance
- Z codes should not be used to code a disease or injury that is the reason for the encounter
- Z codes must be selected based on the specific purpose of the encounter as documented in the medical record

**Requirements / Properties:**
- The encounter must be solely for the specified circumstance indicated by the Z code
- Clinical documentation must clearly support the reason for the encounter without underlying disease
- The Z code must accurately reflect the primary purpose of the healthcare visit

**Deprecated in this version:**
- Using Z codes to mask a primary disease diagnosis
- Assigning Z codes when a more specific disease code exists for the encounter

**New in this version:**
- Enhanced specificity in Z code selection for preventive care encounters
- Mandatory documentation of the specific reason for the encounter to justify Z code assignment
