# G89 Codes as Principal Diagnosis
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** G89 Codes as Principal Diagnosis

**Key Behaviors:**
- G89 codes are acceptable as principal diagnosis when pain control is the reason for admission.
- Underlying cause of pain must be reported as an additional diagnosis if known.
- Assign pain code as principal when admission is for neurostimulator insertion for pain control.
- Assign underlying condition code as principal when procedure treats underlying condition and neurostimulator is inserted for pain control during same encounter.

**Mandatory Coding Criteria:**
- Pain control or pain management must be the reason for the admission/encounter.
- Underlying cause of pain must be documented and known to be reported as an additional diagnosis.
- Procedure must be aimed at treating the underlying condition for the underlying code to be principal.
