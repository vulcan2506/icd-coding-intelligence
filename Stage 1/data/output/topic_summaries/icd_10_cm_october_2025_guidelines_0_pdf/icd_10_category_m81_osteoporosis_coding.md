# ICD-10 Category M81 Osteoporosis Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Category M81 Osteoporosis Coding

**Key Behaviors:**
- Site is not a component of codes under category M81 for osteoporosis without current pathological fracture.
- Site codes under category M80 identify the fracture site, not the osteoporosis itself.
- Category M81 is used for patients with osteoporosis who do not currently have a pathologic fracture due to the osteoporosis, even if they have had a fracture in the past.
- Status code Z87.310 (Personal history of osteoporosis fracture) should follow the M81 code for patients with a history of osteoporosis fractures.

**Mandatory Coding Criteria:**
- Condition must be osteoporosis without current pathological fracture due to the osteoporosis.
- If a history of osteoporosis fracture exists, status code Z87.310 must follow the primary M81 code.
- Codes under M81 do not include site specificity.
