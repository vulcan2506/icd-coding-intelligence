# Intraoperative Complication Code Sequencing
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Intraoperative Complication Code Sequencing

**Key Behaviors:**
- Intraoperative and postprocedural complication codes should be sequenced first, followed by a code for the specific complication if applicable.
- Complication codes from body system chapters must be assigned for intraoperative and postprocedural complications unless the complication is specifically indexed to a T code in chapter 19.

**Requirements / Properties:**
- The complication must be intraoperative or postprocedural in nature.
- The complication must be specific to the organs and structures of the relevant body system chapter.
- If the complication is indexed to a T code in chapter 19, that T code takes precedence over body system chapter codes.
