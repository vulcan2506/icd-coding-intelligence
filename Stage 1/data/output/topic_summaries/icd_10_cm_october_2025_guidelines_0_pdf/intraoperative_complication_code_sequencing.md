# Intraoperative Complication Code Sequencing
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Intraoperative Complication Code Sequencing

**Key Behaviors:**
- Intraoperative and postprocedural complication codes are sequenced first, followed by codes for the specific complication.
- Complication codes from body system chapters are assigned for intraoperative and postprocedural complications unless specifically indexed to a T code in chapter 19.

**Mandatory Coding Criteria:**
- Codes must be specific to the organs and structures of the relevant body system chapter.
- Assignment is required unless the complication is specifically indexed to a T code in chapter 19.
