# Post-COVID-19 Follow-Up Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Post-COVID-19 Follow-Up Coding Guidelines

**Key Behaviors:**
- Assign code Z09 for encounters regarding follow-up after completed treatment for non-malignant conditions
- Assign code Z86.16 for personal history of COVID-19 in asymptomatic follow-up visits
- Exclude COVID-19 test results from the primary diagnosis if results are negative
- Direct coding of symptomatic or condition-related follow-up visits to guideline I.C.1.g.1.m
- Reference Section I.C.21.c.8 for factors influencing health states and contact with health services

**Requirements / Properties:**
- Patient must have a previous history of COVID-19 infection
- Patient must currently be asymptomatic (no residual symptoms or conditions)
- COVID-19 test results must be negative
- Encounter purpose must be a follow-up evaluation after completed treatment

**New in this version:**
- Code Z09 for encounter for follow-up examination after completed treatment
- Code Z86.16 for personal history of COVID-19
