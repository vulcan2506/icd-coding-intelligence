# Complications of Care Code Structure
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Complications of Care Code Structure

**Key Behaviors:**
- Some complications of care codes have the external cause included within the code itself.
- The code includes the nature of the complication as well as the type of procedure that caused it.
- No separate external cause code indicating the type of procedure is necessary for these codes.

**Requirements / Properties:**
- The code must explicitly include the nature of the complication.
- The code must explicitly include the type of procedure that caused the complication.

**Deprecated in this version:**
- Assigning a separate external cause code indicating the type of procedure when the complication code already includes it.
