# COVID-19 Diagnosis Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** COVID-19 Diagnosis Coding Guidelines

**Key Behaviors:**
- Assign code U07.1 for confirmed COVID-19 based on provider documentation without requiring a positive test result.
- Do not assign code U07.1 for suspected, possible, probable, or inconclusive cases; instead, code signs and symptoms.
- Assign specific symptom codes (e.g., R05.1, R06.02, R50.9) for patients with signs/symptoms but no definitive diagnosis.
- Assign additional code Z20.822 for patients with signs/symptoms who have actual or suspected contact with or exposure to COVID-19.
- Apply U07.1 as an exception to hospital inpatient guideline Section II, H.

**Requirements / Properties:**
- Provider must document a confirmed diagnosis of COVID-19 to assign code U07.1.
- Documentation of 'suspected', 'possible', 'probable', or 'inconclusive' precludes the use of code U07.1.
- Code U07.1 is sufficient for confirmation; a positive test result is not mandatory if the provider confirms the diagnosis.
- Symptom codes must be assigned for each presenting sign or symptom when a definitive diagnosis is not established.
- Contact or exposure to COVID-19 must be documented to assign code Z20.822 alongside symptom codes.

**Deprecated in this version:**
- Assigning code U07.1 for suspected, possible, probable, or inconclusive COVID-19 cases.
- Requiring a positive test result to assign code U07.1 when the provider has documented a confirmed diagnosis.
- Coding only the positive test result without assigning the specific COVID-19 diagnosis code U07.1 for confirmed cases.

**New in this version:**
- Code U07.1 for confirmed COVID-19 without requiring a positive test result.
- Code Z20.822 for contact with and (suspected) exposure to COVID-19.
- Exception to hospital inpatient guideline Section II, H for assigning U07.1.
