# COVID-19 Diagnosis Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** COVID-19 Diagnosis Coding Guidelines

**Key Behaviors:**
- Assign code U07.1 only for confirmed diagnoses documented by the provider, without requiring a positive test result.
- Do not assign code U07.1 for suspected, possible, probable, or inconclusive diagnoses; instead, code signs and symptoms.
- Code specific signs and symptoms (e.g., acute cough, shortness of breath, fever) for patients without a definitive diagnosis.
- Assign additional code Z20.822 for patients with signs/symptoms who have actual or suspected contact with or exposure to COVID-19.

**Mandatory Coding Criteria:**
- Provider must document a confirmed diagnosis of COVID-19 to assign code U07.1.
- Documentation of uncertainty (suspected, possible, probable, inconclusive) precludes assignment of U07.1.
- Code U07.1 is an exception to hospital inpatient guideline Section II, H.
