# ICD-10 Category O10 Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Category O10 and P36 Coding Guidelines

**Key Behaviors:**
- Category O10 includes codes for hypertensive heart and hypertensive chronic kidney disease.
- When assigning an O10 code with heart or kidney disease, a secondary code from the hypertension category must be added to specify the type of failure or disease.
- Category P36 includes congenital sepsis.
- If sepsis is documented without specifying congenital or community-acquired, the default is congenital and a P36 code should be assigned.
- If a P36 code includes the causal organism, additional codes from categories B95 or B96 should not be assigned.

**Mandatory Coding Criteria:**
- Secondary code from the appropriate hypertension category is required when O10 code includes hypertensive heart disease or hypertensive chronic kidney disease.
- Additional code from category B96 is required if the P36 code does not include the causal organism.
- Additional codes for severe sepsis (R65.2-) and acute organ dysfunction may be assigned if applicable.
