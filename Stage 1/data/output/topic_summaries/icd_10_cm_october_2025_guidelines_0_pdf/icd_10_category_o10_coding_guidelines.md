# ICD-10 Category O10 Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Category O10 and P36 Coding Guidelines

**Key Behaviors:**
- Category O10 includes codes for hypertensive heart and hypertensive chronic kidney disease.
- When assigning an O10 code with heart or kidney disease, a secondary code from the appropriate hypertension category must be added to specify the type of failure or disease.
- For Category P36, if sepsis is documented without specifying congenital or community-acquired, the default is congenital.
- If a P36 code includes the causal organism, additional codes from categories B95 or B96 should not be assigned.
- If a P36 code does not include the causal organism, an additional code from category B96 must be assigned.
- If applicable, use additional codes to identify severe sepsis (R65.2-) and any associated acute organ dysfunction for Category P36.

**Requirements / Properties:**
- Documentation must specify the type of heart failure or chronic kidney disease when using O10 codes that include these conditions.
- Documentation must clarify whether sepsis is congenital or community-acquired; if unspecified, congenital is assumed.
- Causal organism documentation determines whether additional codes from categories B95 or B96 are required or prohibited.

**Deprecated in this version:**
- Assigning additional codes from categories B95 or B96 when the P36 code already includes the causal organism.
- Coding sepsis without defaulting to congenital when laterality or acquisition type is unspecified.

**New in this version:**
- Mandatory addition of secondary hypertension codes to O10 codes specifying heart or kidney involvement.
- Explicit default rule assigning congenital sepsis when acquisition type is undocumented in perinates.
