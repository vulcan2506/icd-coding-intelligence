# Asymptomatic COVID-19 Diagnosis Verification
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Asymptomatic COVID-19 Diagnosis Verification

**Key Behaviors:**
- Query the provider when an asymptomatic individual tests positive for COVID-19 but lacks provider documentation of the diagnosis
- Recognize that false positive laboratory tests are possible for asymptomatic individuals
- Assign the diagnosis code only after the provider confirms the diagnosis and documents it accordingly
- Do not assign a COVID-19 diagnosis code based solely on a positive laboratory test without clinical confirmation
- Ensure the provider assumes responsibility for confirming the diagnosis before coding

**Mandatory Coding Criteria:**
- Provider documentation must explicitly confirm the diagnosis of COVID-19 before assigning the code
- A query to the provider is required if documentation is absent despite a positive test result
