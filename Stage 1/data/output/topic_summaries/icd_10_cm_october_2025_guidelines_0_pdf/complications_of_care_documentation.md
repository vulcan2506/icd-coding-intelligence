# Complications of Care Documentation
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Complications of Care Documentation Standards

**Key Behaviors:**
- Documentation must explicitly identify complications of care to support ICD-10-CM assignment
- Refers to Section I.B.16 for specific documentation requirements regarding complications
- Ensures accurate coding by linking clinical documentation to the complications of care category

**Mandatory Coding Criteria:**
- Clinicians must document the specific complication of care in the medical record
- Documentation must align with Section I.B.16 guidelines for valid code assignment
