# Complications of Care Documentation
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Complications of Care Documentation

**Key Behaviors:**
- Documentation of complications of care is required for healthcare claims
- Guidelines reference Section I.B.16 for specific documentation standards
- Proper documentation ensures accurate assignment of ICD-10-CM diagnosis codes
- External cause codes must be linked to the complication event
- Claims lacking complication documentation may be rejected or flagged for review

**Requirements / Properties:**
- Healthcare providers must document the specific complication event in the clinical record
- Documentation must align with Section I.B.16 guidelines for code assignment
- Diagnosis codes must reflect the complication rather than the original condition
- External cause codes must specify the nature and context of the complication
- Claims processing systems must validate complication documentation against official guidelines
