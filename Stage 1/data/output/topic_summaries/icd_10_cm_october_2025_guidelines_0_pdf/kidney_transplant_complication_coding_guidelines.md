# Kidney Transplant Complication Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Kidney Transplant Complication Coding

**Key Behaviors:**
- Assign code T86.1 for documented complications of a kidney transplant such as transplant failure or rejection.
- Do not assign T86.1- for post-transplant patients with chronic kidney disease (CKD) unless a specific transplant complication like failure or rejection is documented.
- Query the provider if documentation is unclear regarding whether the patient has a transplant complication.
- Assign codes from subcategory T86.1 for conditions affecting transplanted kidney function other than CKD, paired with a secondary code identifying the complication.
- Refer to section I.C.14 for coding chronic kidney disease and kidney transplant status when no transplant complication is present.
- Refer to section I.C.1.d.4 for specific sequencing instructions regarding sepsis due to infection in a transplanted organ.

**Mandatory Coding Criteria:**
- Documentation must explicitly state a transplant complication (e.g., failure, rejection) to assign T86.1.
- A secondary code must identify the specific complication when assigning codes from subcategory T86.1 for non-CKD conditions.
- Provider query is required if the documentation does not clarify the presence of a transplant complication.
