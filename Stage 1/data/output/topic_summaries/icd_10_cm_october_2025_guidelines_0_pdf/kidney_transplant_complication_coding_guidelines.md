# Kidney Transplant Complication Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Kidney Transplant Complication Coding

**Key Behaviors:**
- Code T86.1 should be assigned for documented complications of a kidney transplant, such as transplant failure or rejection.
- Code T86.1- should not be assigned for post kidney transplant patients who have chronic kidney disease (CKD) unless a transplant complication is documented.
- If documentation is unclear regarding a transplant complication, the provider must be queried for clarification.
- Conditions affecting transplanted kidney function other than CKD require a code from subcategory T86.1 and a secondary code identifying the complication.
- Sepsis due to infection in a transplanted organ requires specific sequencing per section I.C.1.d.4.

**Requirements / Properties:**
- Documentation must explicitly state a transplant complication (e.g., failure, rejection) to justify assigning code T86.1.
- Chronic kidney disease (CKD) alone in a post-transplant patient is insufficient to assign code T86.1 without a documented complication.
- A secondary code must be assigned to identify the specific complication when using subcategory T86.1.
- Provider queries are mandatory when clinical documentation is ambiguous regarding the presence of a transplant complication.
- Sepsis coding in transplanted organs must follow the specific sequencing rules outlined in section I.C.1.d.4.

**Deprecated in this version:**
- Assigning code T86.1- solely based on the presence of chronic kidney disease (CKD) in a post-transplant patient without documented failure or rejection.
- Coding transplant complications without verifying that the condition is not an integral part of the underlying disease process.
- Failing to query the provider when the distinction between CKD and a transplant complication is unclear in the documentation.

**New in this version:**
- Mandatory assignment of a secondary code to identify the specific complication when using subcategory T86.1 for conditions other than CKD.
- Explicit reference to section I.C.1.d.4 for the sequencing of sepsis due to infection in transplanted organs.
- Clarification that CKD following a transplant without complications should be coded according to section I.C.14 rather than T86.1.
- Reference to section I.C.14 for coding CKD following a kidney transplant when no complication like failure or rejection is present.
