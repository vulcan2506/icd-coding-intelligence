# ICD-10 Pain Code Differentiation
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10 Pain Code Differentiation

**Key Behaviors:**
- Codes G89.0 and G89.4 must only be used when the provider specifically documents 'central pain syndrome' or 'chronic pain syndrome' respectively, distinguishing them from general 'chronic pain' documentation.
- Pain associated with devices, implants, or grafts is assigned to appropriate Chapter 19 codes (Injury, poisoning, and certain other consequences of external causes) rather than general pain codes.
- Additional codes from category G89 (specifically G89.18 or G89.28) are used to identify acute or chronic pain due to the presence of a device, implant, or graft.

**Mandatory Coding Criteria:**
- Provider must specifically document 'central pain syndrome' to assign code G89.0.
- Provider must specifically document 'chronic pain syndrome' to assign code G89.4.
- Pain due to devices/implants/grafts requires assignment of a Chapter 19 code first, followed by an additional code from G89.18 or G89.28.
