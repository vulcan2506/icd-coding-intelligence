# ICD-10-CM Code Selection Workflow
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10-CM Code Selection Workflow

**Key Behaviors:**
- Locate the diagnosis term in the Alphabetic Index first, then verify the code in the Tabular List.
- Read and follow instructional notations appearing in both the Alphabetic Index and the Tabular List.
- Use both the Alphabetic Index and Tabular List to locate and assign codes, as the Index does not always provide the full code.
- Select the full code including laterality and applicable 7th characters only from the Tabular List.
- Interpret a dash (-) at the end of an Alphabetic Index entry as an indicator that additional characters are required.

**Requirements / Properties:**
- Verification of the code in the Tabular List is mandatory before assignment.
- Instructional notations in both the Alphabetic Index and Tabular List must be read and followed.
- Full code selection including laterality and 7th characters must be derived from the Tabular List.
- A dash (-) in the Alphabetic Index entry requires the coder to look for additional characters in the Tabular List.
- Even without a dash in the Alphabetic Index, the Tabular List must be consulted to verify if a 7th character is required.
