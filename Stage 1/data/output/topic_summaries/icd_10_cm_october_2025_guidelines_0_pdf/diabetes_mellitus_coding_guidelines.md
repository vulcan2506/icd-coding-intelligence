# Diabetes Mellitus Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Diabetes Mellitus Coding Guidelines

**Key Behaviors:**
- Diabetes mellitus codes are combination codes including type, body system affected, and complications.
- Codes should be sequenced based on the reason for the particular encounter.
- As many codes as necessary should be assigned to describe all complications of the disease.
- If the type of diabetes is not documented, the default is E11.-, Type 2 diabetes mellitus.
- Additional codes from category Z79 must be assigned to identify long-term use of insulin, oral hypoglycemic drugs, or injectable non-insulin antidiabetic drugs.
- Codes from categories E08 - E13 must be assigned to identify all associated conditions the patient has.
- If the patient is treated with both oral hypoglycemic drugs and an injectable non-insulin antidiabetic drug, codes Z79.84 and Z79.85 must be assigned.

**Mandatory Coding Criteria:**
- Documentation must indicate the type of diabetes; if absent, default to E11.-.
- Documentation must specify the use of insulin, oral hypoglycemics, or injectable non-insulin drugs to assign Z79 codes.
- Both Z79.4 and Z79.84 must be assigned if the patient uses both insulin and oral hypoglycemic drugs.
- Z79.4 should not be assigned if insulin is given temporarily to control blood sugar during an encounter.
- All associated conditions must be identified using codes from categories E08 - E13.
- Code Z79.4 should not be assigned if insulin is given temporarily to bring a type 2 patient's blood sugar under control during an encounter.
