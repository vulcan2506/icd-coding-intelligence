# Diabetes Mellitus Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Diabetes Mellitus Coding Guidelines

**Key Behaviors:**
- Diabetes mellitus codes are combination codes including type, body system affected, and complications, sequenced based on the reason for the encounter.
- If the type of diabetes is not documented, the default code E11.- (Type 2 diabetes mellitus) must be assigned.
- Additional codes from category Z79 must be assigned to identify long-term use of insulin, oral hypoglycemic drugs, or injectable non-insulin antidiabetic drugs.
- Code Z79.4 should not be assigned if insulin is given temporarily to bring a type 2 patient's blood sugar under control during an encounter.
- As many codes from categories E08 - E13 as necessary should be assigned to identify all associated conditions the patient has.

**Requirements / Properties:**
- Documentation must specify the type of diabetes mellitus; if absent, E11.- is mandatory.
- Documentation must indicate the specific medications used (insulin, oral hypoglycemics, injectable non-insulin) to trigger Z79 codes.
- All associated conditions must be identified using codes from categories E08 - E13.
- Coding for temporary insulin use in type 2 diabetes is prohibited under Z79.4.
- Encounter reason must determine the sequencing of the primary diabetes code.
- If a patient is treated with both oral hypoglycemic drugs and insulin, both Z79.4 and Z79.84 must be assigned.
- If a patient is treated with both insulin and an injectable non-insulin antidiabetic drug, both Z79.4 and Z79.85 must be assigned.

**Deprecated in this version:**
- Assigning unspecified diabetes codes when the type is not documented.
- Omitting Z79 codes when long-term medication use is documented.
- Assigning Z79.4 for temporary insulin administration during an encounter.
- Failing to sequence codes based on the reason for the encounter.
- Using fewer than necessary codes from categories E08 - E13 to describe all complications.

**New in this version:**
- Explicit requirement to assign Z79.84 for long-term use of oral hypoglycemic drugs.
- Explicit requirement to assign Z79.85 for long-term use of injectable non-insulin antidiabetic drugs.
- Clarification that Z79.4 is excluded for temporary insulin use in type 2 diabetes.
- Mandate to use combination codes that integrate type, system, and complications.
- Guidance to query providers if diabetes type is missing rather than defaulting without documentation.
