# I13 Hypertensive Heart and Kidney Disease Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** I13 Hypertensive Heart and Kidney Disease Coding

**Key Behaviors:**
- Assign codes from category I12 when both hypertension and a condition classifiable to category N18 are present.
- Use the appropriate code from category N18 as a secondary code with a code from category I12 to identify the stage of chronic kidney disease.
- Code acute renal failure in addition to hypertensive chronic kidney disease if present, sequencing according to the circumstances of the admission or encounter.
- Do not code CKD as hypertensive if the provider indicates the CKD is not related to the hypertension.
- Refer to Section I.C.14 for specific chronic kidney disease coding guidelines.

**Requirements / Properties:**
- Both hypertension and a condition classifiable to category N18 must be present to assign codes from category I12.
- Provider documentation must explicitly indicate the relationship between CKD and hypertension to avoid misclassification.
- Acute renal failure must be documented to be coded alongside hypertensive chronic kidney disease.
- Sequencing must reflect the primary reason for the admission or encounter.
- The stage of chronic kidney disease must be identified using the appropriate N18 code.

**Deprecated in this version:**
- Coding CKD as hypertensive when the provider indicates the CKD is not related to hypertension.
- Omitting the coding of acute renal failure when it is present alongside hypertensive chronic kidney disease.
- Failing to sequence acute renal failure according to the circumstances of the admission or encounter.
- Using unspecified codes for the stage of chronic kidney disease when specific N18 codes are available.
- Ignoring Section I.C.14 guidelines when coding chronic kidney disease.

**New in this version:**
- Mandatory use of category I12 codes when both hypertension and CKD are present.
- Requirement to sequence acute renal failure based on admission or encounter circumstances.
- Explicit instruction to query providers if the relationship between CKD and hypertension is unclear.
- Integration of Section I.C.14 guidelines for accurate CKD coding.
- Emphasis on using specific N18 codes to identify the stage of chronic kidney disease.
