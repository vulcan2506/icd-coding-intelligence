# I13 Hypertensive Heart and Kidney Disease Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** I13 Hypertensive Heart and Kidney Disease Coding

**Key Behaviors:**
- Assign codes from category I12 when both hypertension and a condition classifiable to category N18 are present.
- Do not code CKD as hypertensive if the provider indicates the CKD is not related to the hypertension.
- Use the appropriate code from category N18 as a secondary code with a code from category I12 to identify the stage of chronic kidney disease.
- If a patient has hypertensive chronic kidney disease and acute renal failure, the acute renal failure should also be coded.
- Sequence codes according to the circumstances of the admission or encounter.

**Mandatory Coding Criteria:**
- Both hypertension and a condition classifiable to category N18 must be present.
- Provider must indicate the CKD is related to hypertension to use I12 codes.
- Category N18 code must be used as a secondary code with an I12 code.
- Acute renal failure must be coded if present alongside hypertensive chronic kidney disease.
- Coding sequence must reflect the circumstances of the admission or encounter.
