# Borderline Diagnosis Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Borderline Diagnosis Coding Standards

**Key Behaviors:**
- Code borderline diagnoses as confirmed unless a specific index entry exists
- Use specific index entries (e.g., borderline diabetes) if available in ICD-10-CM
- No distinction is made between inpatient and outpatient care settings for borderline conditions
- Query the documenting clinician whenever documentation regarding a borderline condition is unclear

**Mandatory Coding Criteria:**
- Coder must verify if a specific ICD-10-CM index entry exists for the borderline condition
- Clarification query required when documentation is unclear about the borderline condition
