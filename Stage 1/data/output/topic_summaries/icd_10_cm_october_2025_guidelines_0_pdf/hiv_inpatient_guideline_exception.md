# HIV Inpatient Guideline Exception
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** HIV Inpatient Guideline Exception

**Key Behaviors:**
- Code only confirmed cases of HIV infection/illness as an exception to the hospital inpatient guideline Section II, H.
- Confirmation does not require documentation of positive serology or culture for HIV.

**Mandatory Coding Criteria:**
- Provider must make a diagnostic statement that the patient is HIV positive or has an HIV-related illness.
