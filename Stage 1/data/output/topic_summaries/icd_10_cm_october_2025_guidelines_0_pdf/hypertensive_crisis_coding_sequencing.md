# Hypertensive Crisis Coding Sequencing
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Hypertensive Crisis Coding Sequencing

**Key Behaviors:**
- Assign a code from category I16 for documented hypertensive urgency, hypertensive emergency, or unspecified hypertensive crisis.
- Code any identified hypertensive disease (I10, I15) in addition to the crisis code.
- Determine sequencing based on the reason for the encounter.

**Requirements / Properties:**
- Clinical documentation must explicitly identify the condition as hypertensive urgency, hypertensive emergency, or unspecified hypertensive crisis.
- The reason for the encounter must be established to determine the primary code position.
