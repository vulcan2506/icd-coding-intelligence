# Sequela and Late Effect Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Sequela and Late Effect Coding Guidelines

**Key Behaviors:**
- A sequela is defined as the residual effect after the acute phase of an illness or injury has terminated.
- There is no time limit on when a sequela code can be used.
- Coding generally requires two codes sequenced in the following order: the condition or nature of the sequela is sequenced first, and the sequela code is sequenced second.
- The code for the acute phase of an illness or injury that led to the sequela is never used with a code for the late effect.
- Exceptions exist where the sequela code is followed by a manifestation code identified in the Tabular List or where the sequela code is expanded to include the manifestation.
- Examples of sequela include scar formation from burns, deviated septum from nasal fractures, and infertility from old tuberculosis.
- Exceptions exist where the sequela code is followed by a manifestation code identified in the Tabular List and title.

**Mandatory Coding Criteria:**
- Documentation must indicate the condition is a residual effect after the acute phase has terminated.
- Two codes must be assigned: one for the nature of the sequela and one for the sequela itself.
- The nature of the sequela must be sequenced first in the claim.
- The sequela code must be sequenced second in the claim.
- The acute phase code must not be used in conjunction with the late effect code.
