# Sequela and Late Effect Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Sequela and Late Effect Coding Guidelines

**Key Behaviors:**
- A sequela is defined as the residual effect produced after the acute phase of an illness or injury has terminated.
- There is no time limit on when a sequela code can be used.
- Coding generally requires two codes sequenced with the condition or nature of the sequela first, followed by the sequela code second.
- The code for the acute phase of an illness or injury that led to the sequela is never used with a code for the late effect.
- Exceptions exist where the sequela code is followed by a manifestation code identified in the Tabular List or title.

**Requirements / Properties:**
- Clinical documentation must clearly identify the residual effect as a condition produced after the acute phase has terminated.
- Documentation must support the specific nature of the sequela (e.g., scar formation, deviated septum, infertility).
- Two codes must be assigned in the correct sequence: condition/nature first, sequela second.

**Deprecated in this version:**
- Using a code for the acute phase of an illness or injury in conjunction with a code for the late effect.
- Assigning a sequela code without first coding the condition or nature of the sequela.

**New in this version:**
- Recognition that specific sections (I.C.9, I.C.15, I.C.19) contain expanded guidelines for cerebrovascular disease, pregnancy complications, and Chapter 19 applications.
- Reference to Section I.C.9 for sequelae of cerebrovascular disease.
- Reference to Section I.C.15 for sequelae of complications of pregnancy, childbirth, and the puerperium.
- Reference to Section I.C.19 regarding the application of 7th characters for Chapter 19.
