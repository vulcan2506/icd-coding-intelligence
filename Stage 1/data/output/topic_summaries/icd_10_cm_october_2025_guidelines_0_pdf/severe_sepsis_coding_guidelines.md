# Severe Sepsis Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Severe Sepsis Coding Requirements

**Key Behaviors:**
- Assign a minimum of 2 codes: one for the underlying systemic infection and one from subcategory R65.2 for severe sepsis.
- Use code A41.9 for the infection if the causal organism is not documented.
- Include additional codes for any associated acute organ dysfunction.
- Query the provider if the complex nature of the case requires clarification before code assignment.

**Mandatory Coding Criteria:**
- Documentation of an underlying systemic infection is mandatory.
- Documentation of acute organ dysfunction is required for additional coding.
- If the causal organism is unspecified, code A41.9 must be used for the infection component.
