# Unspecified Medical Coding Guidelines
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Unspecified Medical Coding Guidelines

**Key Behaviors:**
- Codes titled 'unspecified' are for use when the information in the medical record is insufficient to assign a more specific code.
- For categories without a specific unspecified code, the 'other specified' code may represent both other and unspecified conditions.

**Requirements / Properties:**
- Information in the medical record must be insufficient to assign a more specific code before using an unspecified code.
- The 'other specified' code may be used to represent unspecified conditions only when no specific unspecified code exists for that category.
