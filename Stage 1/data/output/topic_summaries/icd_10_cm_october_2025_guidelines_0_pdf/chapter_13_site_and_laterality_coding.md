# Chapter 13 Site and Laterality Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** Chapter 13 Site and Laterality Coding

**Key Behaviors:**
- Codes within Chapter 13 include site and laterality designations
- Site designations identify the specific bone, joint, or muscle involved

**Mandatory Coding Criteria:**
- Code must specify the anatomical site (bone, joint, or muscle)
- Laterality must be assigned if designated in the code set
