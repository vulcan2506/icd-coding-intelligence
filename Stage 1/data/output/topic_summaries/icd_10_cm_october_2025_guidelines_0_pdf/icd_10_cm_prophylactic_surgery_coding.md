# ICD-10-CM Prophylactic Surgery Coding
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10-CM Prophylactic Surgery Coding

**Key Behaviors:**
- Assign principal code from subcategory Z40.0 or Z40.8 for encounters specifically for prophylactic organ removal due to risk factors.
- Assign additional codes to identify associated risk factors such as genetic susceptibility or family history.
- Assign a code for the existing malignancy in addition to a Z40.0 code when prophylactic removal is performed at a different site to prevent new primary or metastatic disease.
- Do not assign a Z40.0 code if the organ removal is for the treatment of an existing malignancy (e.g., testis removal for prostate cancer).

**Mandatory Coding Criteria:**
- Encounter must be specifically for prophylactic removal of an organ.
- Removal must be due to risk factors related to malignant neoplasms (e.g., genetic susceptibility, family history).
- If applicable, associated risk factors must be documented to justify additional coding.
