# ICD-10-CM CKD Classification Logic
*Source: icd_10_cm_october_2025_guidelines_0.pdf*

**Feature:** ICD-10-CM CKD Classification Logic

**Key Behaviors:**
- The word 'and' in titles is interpreted to mean either 'and' or 'or' for classification purposes.
- CKD is classified based on severity stages 1-5, with specific codes for mild (Stage 2), moderate (Stage 3), and severe (Stage 4) CKD.
- Code N18.6 (End stage renal disease) is assigned when ESRD is documented.
- If both a stage of CKD and ESRD are documented, assign code N18.6 only.

**Mandatory Coding Criteria:**
- Provider must document the specific stage of CKD (1-5) to assign the corresponding N18.2, N18.3x, or N18.4 code.
- Provider must explicitly document end-stage renal disease (ESRD) to assign code N18.6.
- When both CKD stage and ESRD are present, N18.6 takes precedence over the stage-specific code.
